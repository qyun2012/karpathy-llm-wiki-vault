## 定义

术后克罗恩病复发预防（Post-operative Recurrence Prevention）是指通过医学治疗和监测策略来延缓或防止CD患者肠道切除术后临床和/或内镜复发的系统方法。

## 克罗恩病术后复发的流行病学

### 复发时间进程

| 时间点 | 内镜复发 | 临床复发 | 需再次手术 |
|------|--------|--------|----------|
| **3个月** | 90% | - | - |
| **12个月** | 几乎普遍 | ~25-30% | - |
| **5年** | - | ~50% | 25% |
| **10年** | - | - | 35% |

**关键发现**：
- 术后复发是CD的自然历程，不代表"失败"
- 生物制剂时代未显著改变复发率（尽管已改善医学控制）
- 需要主动预防和监测策略

## 风险分层框架

### 高危患者
需要**积极的早期预防性治疗**：

1. **穿透性/渗透性表现型史**（瘘管、脓肿）
2. **肛周病变史**
3. **≥2次既往手术**
4. **年龄<30岁**（诊断时或手术时）

### 中危患者
表现为单个或部分高危特征：
- 病变肠段长（>10 cm）
- 初诊至手术间隔短（<10年）

### 低危患者
- 无上述高危因素
- 首次手术

### 唯一可修改的危险因素
**吸烟**是唯一可修改的关键风险因素，围手术期戒烟咨询至关重要

## 术后监测策略

### 内镜监测（关键）

#### 内镜复发的定义
使用**Rutgeerts评分**（术后3-6个月）：

| 评分 | 病理 | 临床意义 |
|-----|-----|---------|
| **i0** | 无病变 | 完全愈合 |
| **i1** | <5个溃疡 | 轻度复发 |
| **i2** | >5个溃疡或融合 | 中度复发 |
| **i2a** | 局限于吻合口 | - |
| **i2b** | 吻合口外回肠 | - |
| **i3** | 节段性溃疡/狭窄 | 严重 |
| **i4** | 弥漫性溃疡坏死 | 极严重 |

**现代观点**：i2a和i2b之间差异可能被高估

#### POCER试验的关键发现
- **6个月结肠镜**：积极评估（而非保守监测）
- **治疗强化**：根据内镜和/或临床复发情况升级或修改治疗
- **结果**：内镜复发减少18%
- **意义**：即使**无症状的内镜复发**也需要治疗干预

#### 复发的其他标志物（新兴）

**粪便钙蛋白（fecal calprotectin）**：
- 可能预测术后内镜复发
- 但不足以取代内镜检查
- 可用于增强的监测策略

**非侵入性成像**：
- **小肠超声**：高敏感性/特异性用于术后复发检测
- **磁共振肠道成像（MRE）**：与Rutgeerts评分高度一致
- **视频胶囊内镜（VCE）**：高敏感性和特异性
- **地位**：新兴工具，不应取代初始内镜评估

### 临床监测
- 患者症状评估（腹泻、腹痛、体重减轻）
- 生物标志物（CRP、fecal calprotectin）
- 定期临床就诊

## 预防性治疗策略

### 低风险患者
**策略**：观察等待

- **无预防性治疗**
- **6-12个月后进行**Rutgeerts评分内镜检查
- 如果**内镜复发出现**（≥i2），**启动治疗**

### 中高危患者
**策略**：主动预防治疗（强烈推荐）

#### 一线治疗：抗TNF单药

**证据**：
- 网络荟萃分析（Burr et al., J Crohns Colitis 2019）：**抗TNF单药显示最低内镜复发率**
- PREVENT研究（infliximab）：
  - 76周内镜复发：抗TNF 22.4% vs 安慰剂 51.3%（P<0.001）
  - 临床复发：抗TNF 12.9% vs 安慰剂 20.0%（P=0.097）
  - 结论：**内镜复发显著改善，临床复发改善不显著**

**时间-效果依赖性**（网络荟萃分析）：

| 时间点 | 最优药物 | SUCRA | 注释 |
|------|--------|-------|------|
| **6个月** | Adalimumab | 84.5% | 起始时效果最好 |
| **12-18个月** | Infliximab | 93.0% | 长期更佳 |

**临床启动**：
- 术后2-4周启动（允许初始恢复）
- 标准诱导 + 维持方案
- 不应延迟至等待症状出现

#### 二线选择：硫唑嘌呤（AZA）

**证据**：
- **弱于抗TNF**：临床和内镜复发预防效果有限
- **角色**：与抗TNF联合（优于单药）
- **现代实践**：单药使用减少，主要用于组合疗法

#### 三线：甲硝唑单药
- **证据有限**：对临床复发预防效果有限
- **使用**：短期术后预防（>12个月效果不持续）
- **耐受性问题**：周围神经病变风险

#### 四线：非药物干预（证据有限）
- **维生素D**：与维生素D缺乏的关联尚存在，补充证据有限
- **益生菌**：无效
- **姜黄素**：无效
- **5-ASA/氨基水杨酸**：无效

### 升级策略（基于6个月内镜复发）

**如果出现内镜复发（≥i2）**，即使无症状：

1. **已在单药抗TNF患者**：
   - 升级至**抗TNF + AZA联合**
   - 或转换至不同的抗TNF
   - 考虑治疗药物监测（TDM）优化剂量

2. **未在治疗患者**：
   - 启动**抗TNF单药**
   - 或**抗TNF + AZA联合**（高危患者）

3. **抗TNF无应答/失效**：
   - 考虑**维多利珠单抗**或**乌司替尼**
   - TDM引导优化

## 术后病例驱动管理（建议方法）

### 低风险患者案例
```
术后0-6个月：
- 标准恢复，无其他治疗
- 营养支持
- 戒烟咨询

术后6个月：
- 内镜检查（Rutgeerts评分）
- 如果≤i1：继续观察
- 如果≥i2：启动抗TNF治疗

术后1年：
- 临床评估
- 如需要，考虑内镜复查
```

### 高危患者案例
```
术后0-6周：
- 标准恢复
- 营养优化

术后2-4周：
- 启动抗TNF单药（或联合AZA高危）
- 标准诱导方案

术后6个月：
- 内镜检查（Rutgeerts评分）
- 评估抗TNF应答
- 如≥i2：升级至联合或转换药物

术后12-24个月：
- 持续维持治疗
- 临床和生物标志物监测
```

## 治疗药物监测（TDM）在术后管理中的角色

### 抗TNF TDM
- **黎凡特期**：抗TNF药物浓度与复发风险的关联
- **门槛浓度**：
  - Infliximab：8-10 μg/mL与更好的结果相关
  - Adalimumab：4-7 μg/mL
- **优化**：基于浓度调整剂量可改善复发预防

## 新型生物制剂在术后预防中的地位

### 维多利珠单抗（Vedolizumab）
- 目前证据**有限**
- 仍在研究中
- 可作为抗TNF无应答者的替代

### 乌司替尼（Ustekinumab）
- IL-12/23拮抗
- 越来越多地探索于术后预防
- 前景光明，但数据仍在积累

## 长期随访策略

**术后2-5年**：
- 继续维持治疗（高危患者）
- 定期临床评估
- 生物标志物监测
- 重复内镜检查如无症状（个体化，3-5年间隔）
- TDM优化

**术后>5年**：
- 个体化降低或停止治疗考虑（仅低风险）
- 高危患者继续维持
- 长期随访

## 临床要点总结

1. **风险分层是基础**：不同患者需要不同强度的预防
2. **抗TNF是一线预防性治疗**：尤其在高危患者
3. **内镜监测是关键**：POCER证实6个月评估与升级的价值
4. **内镜复发需要治疗**：即使无症状
5. **时间依赖性**：不同生物制剂的最优时间点不同
6. **个体化与长期**：术后随访需要5-10年的承诺
