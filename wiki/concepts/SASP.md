## 定义

**SASP（Senescence-associated secretory phenotype，衰老相关分泌表型）**是指衰老细胞积极分泌促炎细胞因子、趋化因子、生长因子、细胞外基质金属蛋白酶等生物活性分子，形成的慢性促炎微环境的生物学现象。

核心特征：
- **衰老细胞不再分裂**（细胞周期停滞），但代谢异常活跃
- **持续高分泌活性**：释放IL-6、TNF-α、IL-1β、IL-8等促炎细胞因子
- **旁分泌效应**：分泌的因子作用于邻近的健康细胞与免疫细胞，诱导全身性炎症
- **年龄依赖性积累**：衰老细胞数量与SASP强度随年龄增加而累积

## 主要组分

### 细胞因子（Cytokines）
- **IL-6**：强效促炎，驱动Th17分化、STAT3激活、血清淀粉样蛋白A产生
- **TNF-α**：肿瘤坏死因子，激活NF-κB通路、增强血管炎症
- **IL-1β**：炎症体激活产物，诱导更多IL-6产生（正反馈），增加AD/RA风险
- **IL-8/CXCL8**：中性粒细胞趋化，促进NET形成与肺炎症
- **GM-CSF**：粒细胞-巨噬细胞集落刺激因子，驱动全身细胞因子风暴（如COVID-19重症）

### 趋化因子（Chemokines）
- **CXCL1**：招募中性粒细胞，促进NET形成与急性肺损伤
- **CCL2/MCP-1**：单核细胞招募与活化
- **SDF-1/CXCL12**：干细胞迁移因子，在衰老中失调

### 生长因子
- **FGF10（纤维母细胞生长因子10）**：衰老间充质细胞分泌 → 前列腺癌多灶化
- **FGF19**：骨骼肌细胞来源 → 肝细胞癌诱导
- **VEGF（血管内皮生长因子）**：促血管生成 → 肿瘤新血管形成

### 细胞外基质酶（ECM-degrading enzymes）
- **MMP3（基质金属蛋白酶3）**：衰老成纤维细胞分泌 → 上皮-间质转变(EMT) → 肿瘤侵袭
- **MMP-9**：增强血管通透性 → BBB破坏（AD中Aβ进入大脑）

### 脂质与代谢产物
- **促血栓物质**：PAI-1（纤溶酶原激活物抑制剂-1）
- **ROS（活性氧）**：直接与标志物积累，驱动脂质氧化与血小板激活

## 生物学函数

### 病理效应（Pathological）

#### 1. 细胞衰老的正反馈强化
- 衰老细胞释放IL-6/TNF-α → 邻近健康细胞衰老加速 → SASP↑↑ → 衰老细胞积累
- 形成"衰老细胞驱动的衰老"恶性循环

#### 2. 肿瘤促进（Pro-tumoral Effects）
- **肿瘤生长加速**：FGF10/FGF19直接刺激肿瘤细胞增殖
- **转移促进**：MMP3介导的EMT → 肿瘤细胞可塑性增强 → 扩散
- **新生血管化**：VEGF/FGF等促血管生成因子 → 供血增加
- **免疫逃逸**：促炎SASP改变肿瘤微环境 → MDSCs/Tregs招募 → 抗肿瘤T细胞抑制

#### 3. 神经退行性加速（AD例）
- SASP IL-6/TNF-α进入脑脊液 → 小胶质细胞激活 → 微神经炎症
- Aβ→β-淀粉样蛋白激活炎症体 → 更多IL-1β → 神经元凋亡加速
- 血脑屏障完整性↓ → Th1/Th17细胞渗入 → 神经炎症恶化

#### 4. 代谢异常与衰弱
- IL-6↑↑ → 骨骼肌蛋白分解↑ → 肌肉萎缩(Sarcopenia)
- TNF-α→胰岛素抵抗↑ → 2型糖尿病加重
- SASP诱导脂肪细胞衰老 → 脂肪组织功能障碍

### 生理适应性（Paradoxically Protective）
- **急性损伤修复**：SASP吸引免疫细胞与成纤维细胞，清除受损组织
- **肿瘤抑制**（某些情况）：衰老细胞诱导的p16/p21激活可抑制细胞转化

## SASP的调控

### 核心驱动信号
- **NF-κB通路**：ROS积累 → IκBα磷酸化 → NF-κB p65入核 → IL-6/TNF-α转录
- **p38 MAPK**：DNA损伤 → p38激活 → MAPK-activated kinase 2/3 → MK2/3依赖的mRNA稳定性↑ → SASP因子mRNA半衰期延长

### 调控策略

#### 抑制SASP（Senomorph策略）
- **NF-κB抑制**：IKK抑制剂、Bortezomib、Curcumin、Resveratrol(SIRT1激活p65脱乙酰化)
- **p38抑制**：p38 MAPK抑制剂(预临床阶段)
- **JAK-STAT阻断**：STAT3（SASP强有力驱动）过度激活的抑制
- **IL-6受体拮抗**：Tocilizumab(RA中使用)可部分减少老龄相关炎症

#### 清除SASP（Senolytic策略）
- **Dasatinib**：酪氨酸激酶↓ → SnC特异的生存信号中断 → 凋亡
- **Quercetin**：BCL-2家族蛋白和HIF-1α靶向 → SnC凋亡诱导
- **Navitoclax(ABT-263)**：Bcl-2/Bcl-xL/Bcl-w拮抗 → 通过Caspase-3/7激活SnC凋亡

## SASP与免疫衰老的相互作用

### 双向促进机制
1. **衰老细胞驱动的T细胞衰老**：
   - 衰老细胞SASP IL-6↑ → Th17分化↑
   - 持续促炎环境 → T细胞PD-1/Tim-3↑ → 耗竭表型
   - 结果：T细胞衰老加速 → 抗感染能力↓、癌免疫编码逆转

2. **衰老免疫细胞产生的SASP**：
   - 衰老T细胞、单核细胞、巨噬细胞本身产生SASP → 全身促炎
   - 与组织衰老细胞SASP叠加 → 炎症衰老(Inflammaging)爆发

### 代谢共同驱动
- **NAD⁺↓**：Sirtuins(SIRT1/6)活性↓ → NF-κB去乙酰化失效 → SASP↑
- **线粒体ROS↑**：衰老细胞与免疫细胞的线粒体功能障碍 → ROS→NF-κB激活 → SASP
- **AMPK↓**：能量代谢失控 → mTOR过激活 → 自噬↓ → ROS积累 → SASP

## 临床应用与未来方向

### 生物标志物应用
- **SASP因子监测**：IL-6、TNF-α、IL-8血清水平可作为免疫衰老程度与治疗效果的指标
- **组织SASP指数**：脂肪活检中的MMP3、FGF表达可预测肥胖相关代谢并发症风险

### 治疗干预点
- **精准清除衰老细胞**：针对高SASP亚群的选择性衰老细胞清除(Senolytics)
- **靶向SASP信号**：抗IL-6单克隆抗体、NF-κB抑制剂的组合方案
- **代谢恢复**：NAD⁺前体(NMN/NR)恢复Sirtuin活性 → SASP↓

### 尚未解决的问题
1. **SASP异质性**：不同衰老细胞的SASP组成与强度高度异质，是否存在更精准的亚分型？
2. **炎症与修复的平衡**：急性阶段的SASP有益(清除损伤)，慢性则有害(促病)，临界点如何界定？
3. **个体响应差异**：相同年龄、相同衰老负荷的个体SASP水平差异大，遗传因素vs环境因素的比例？
