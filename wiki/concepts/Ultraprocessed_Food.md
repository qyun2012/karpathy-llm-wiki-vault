## 定义与分类

**超加工食品（UPF）定义**（基于NOVA分类，Monteiro 2019）：
工业配方，含有最少的整体食物，通常具有多种添加剂，如乳化剂、稳定剂、人工颜色、香料和其他工业化学物质。这些产品的目的是使其高度耐久、高利润、可口且成瘾性强。

**NOVA食品分类体系**（四层）：
1. **天然/最少加工食品**：种子、果实、叶菜、块根、坚果、肉类、鱼类、牛奶、豆类、谷物等
2. **加工食材**：通过干燥、压榨、磨碎、低温杀菌等工艺从第1类食物中提取和精制
3. **加工食品**：相对简单的产品，由第1类食物 + 第2类食材 + 少量添加剂（如盐、糖、油）制成（如罐装蔬菜、自制面包）
4. **超加工食品**：工业配方产品，通常包含：
   - 高含量的糖、钠、饱和脂肪
   - 低纤维、维生素、矿物质含量
   - 乳化剂、稳定剂、乳化剂、增稠剂、人工甜味剂、防腐剂、着色剂、香料
   - 例如：软饮料、包装零食、快速食品、超市现成饭菜、某些早餐谷物

## 公共卫生全球趋势

**全球消费增加**：
- 高收入国家：UPF占总膳食能量摄入的30-60%
- 中等收入国家：快速上升（10-30%）
- 低收入国家：仍相对较低，但增速最快

**西化食物模式**：
特别在北美、欧洲及越来越多的低中收入国家，UPF摄入替代了传统的最少加工食品，驱动因素包括：
- 经济可及性（UPF通常比新鲜食物便宜）
- 便利性和营销
- 社会经济差异（低收入人群UPF摄入更高）

## IBD与UPF的机制关联

### 1. 肠道微生物群失调（Dysbiosis）

**添加剂直接毒性**：
- **乳化剂**（如羧甲基纤维素CMC、聚山梨酸酯80）：
  - 破坏黏液层与上皮屏障
  - 改变菌群组成，减少产丁酸盐细菌（Faecalibacterium prausnitzii）
  - 促进促炎型细菌（如肠杆菌属）过度生长
  
- **防腐剂**（如山梨酸、苯甲酸）：
  - 直接杀死有益菌
  - 选择性促进病原菌生长

- **人工甜味剂**（如糖精、三氯蔗糖）：
  - 改变肠道菌群
  - 降低微生物多样性
  - 促进葡萄糖耐受不良

### 2. 肠道屏障功能破坏

**紧密连接降解**：
- UPF添加剂诱发上皮细胞损伤
- 紧密连接蛋白（claudins, occludin, ZO-1）表达下降
- 导致"漏肠"（intestinal permeability ↑）

**黏液层损伤**：
- 乳化剂破坏MUC2黏蛋白屏障
- 细菌易位增加（LPS translocation）
- 诱发全身LPS血症和代谢内毒血症

### 3. 促炎免疫反应

**系统性炎症激活**：
- LPS刺激TLR4 → NF-κB激活
- 产生促炎细胞因子（TNF-α, IL-6, IL-8）
- Th17细胞分化增加（IL-17产生）
- Th1/Treg平衡失衡

**遗传易感者反应**：
- IBD相关遗传变异（NOD2, IL-23, JAK2）的个体对UPF更易感
- 微生物群-免疫失调的恶性循环

### 4. 代谢失调

**短链脂肪酸（SCFA）缺乏**：
- UPF低纤维含量，菌群发酵差
- 丁酸盐产生↓ → 肠上皮能量供应↓
- 调节性T细胞（Tregs）分化↓（丁酸盐是HDAC抑制剂）

**代谢产物异常**：
- 次级胆酸增加（某些菌株β-glucuronidase过表达）
- 胆酸受体（FXR, TGR5）信号减弱
- 肠上皮分化与屏障功能受损

## IBD中UPF的流行病学证据

### PURE研究发现（2025）

**西方人群（北美/欧洲）**：
- n = 13,970倾向评分匹配参与者
- UPF摄入中位数：7.5份/天（仅6.3%达到<1份/天）
- IBD发病率：218例（中位随访12.9年）

**剂量-反应关系**：
| UPF摄入 | 20年IBD风险 | aHR | 95% CI |
|--------|-----------|-----|---------|
| <0.5份/天 | 0.53% | 0.43 | 0.33-0.57 |
| <1份/天 | 0.64% | 0.56 | 0.54-0.58 |
| <1份/周 | 0.61% | 0.53 | 0.50-0.56 |
| 常规饮食（参考） | 1.21% | 1.00 | — |
| 1.5-3份/天 | 0.95% | — | — |
| ≥3份/天 | 1.24% | — | — |

**具体UPF亚类**：
- 超加工谷物<1份/周 → aHR 0.20 (20%相对风险)
- 炸食品<1份/周 → aHR 0.24
- 加工肉制品<1份/周 → aHR 0.45
- 软饮料<1份/周 → aHR 0.23
- 甜食<1份/周 → aHR 0.19

**时间效应**：
保护性关联随随访时间增强（aHR: 前5年0.62 → 15-20年0.30）

### 其他队列证据

**观察性数据支持**：
- Nurses' Health Study：高纤维（低UPF）饮食HR 0.75
- 多项队列研究显示UPF消费与IBD风险正相关

**因果关系推断**：
- 双向交叉设计研究显示添加剂直接诱发肠道炎症
- 动物模型证实乳化剂+特定菌群→自发性结肠炎

## 公共卫生干预策略

### 个体层面

**改变建议**（基于证据）：
- **目标**：UPF摄入<1份/天
- **实际难度**：西方人群中仅6.3%达到此目标
- **渐进式方法**：即使部分减少也有益（暴露-反应曲线）

**具体食物替换**：
| 替换 | 减少UPF | 增加营养价值 |
|------|--------|------------|
| 软饮 → 清水/茶 | 液体糖 + 人工甜味剂 | 零热量 + 多酚 |
| 超加工谷物 → 全谷物 | 乳化剂 + 精制淀粉 | 纤维 + B族维生素 |
| 包装零食 → 坚果/水果 | 反式脂肪 + 添加剂 | 单不饱和脂肪 + 抗氧化剂 |
| 快餐 → 家庭烹饪 | 盐 + 饱和脂肪 + 添加剂 | 控制成分 + 新鲜 |

### 政策与系统层面

**关键障碍**：
- 社会经济因素：UPF更便宜、更方便、更易获得
- 营销与行为强化：工业品多于家庭食品的营销预算
- 食品环境不公平：低收入社区食品沙漠vs 快餐密集

**循证政策建议**：
1. **膳食指南**：明确鼓励减少UPF的国家指南
2. **经济杠杆**：
   - 补贴新鲜食物（水果、蔬菜、全谷物）
   - 对UPF征税（特别是含添加剂/高糖产品）
3. **食品标签与教育**：
   - 前端营养标签（FOPL）清晰显示超加工等级
   - 学校营养教育项目
4. **食品环境改造**：
   - 超市食物布局改进（新鲜食品可见性↑）
   - 学校/工作场所食堂菜单优化
5. **规制层面**：
   - 限制向儿童营销UPF
   - 建立食品添加剂黑名单（乳化剂等）

## 临床实践建议

### IBD患者饮食建议

**缓解期**：
- 逐步增加整体食物比例
- 减少特别是液体UPF（软饮、超加工饮料）
- 优先选择最少加工食品

**活跃期**：
- 短期可能需要液态营养（但选择最少加工型，如纯果汁、清汤）
- 避免高纤维的超加工制品（炸薯条等）
- 尽快向整体食物过渡

### 监测与指导

**可计量指标**：
- 基线饮食问卷（FFQ或食物日记）
- 逐月UPF摄入减少目标
- 相关生物标志物：粪钙卫蛋白、CRP、菌群多样性
