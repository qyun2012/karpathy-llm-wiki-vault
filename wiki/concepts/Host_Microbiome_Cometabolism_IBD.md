## 定义与核心概念

**宿主-微生物组共代谢脱调**是指在 IBD 炎症期，**宿主组织代谢与肠道微生物群落代谢的协调失衡**。具体表现为：

1. **微生物群落内部代谢交互破裂** → 群落功能碎片化
2. **菌群向宿主的代谢物供应链中断** → 关键产物（SCFA、NAD、维生素）缺乏
3. **宿主代谢的全局衰退** → 能量代谢、氨基酸代谢、脂肪代谢崩塌
4. **宿主异常代谢物反馈** → 进一步抑制菌群功能

这一脱调是 **IBD 病理的核心驱动力之一**，而非仅仅的继发结果。

## 代谢交互的三个层级

### **Tier 1：菌群内部代谢交叉代谢（Within-Microbiota Cross-Feeding）**

**定义**：某菌株A的代谢产物作为另一菌株B的营养源。

**健康状态**：高度整合的代谢互补性
```
Clostridia（丁酸盐生产者）←← 葡萄糖、丙酸盐
  ↑
  ↑ （通过交叉代谢获得）
  ↓
Bacteroidia（糖酵解）→→ 丙酮酸、乳酸
```

**IBD 炎症期变化**：
- **减少的交叉代谢对象**（预测 10 种代谢物）：
  - 葡萄糖 ↓（发酵基质供应减少）
  - 丙酸盐 ↓（同时产生和交叉代谢减少）
  - 草酸戊二酸 ↓（TCA 中间体枯竭）
  - 丙氨酸/天冬氨酸 ↓（转氨反应受阻）

- **异常增加的交叉代谢**：
  - 乳酸 ↑（过度发酵，产生过量乳酸积累的菌株繁殖）

**后果**：
```
群落合作菌株（Clostridia、Bacteroidia）↓
代谢孤立菌株（机会致病菌）↑
群落功能：SCFA 产生 ↓↓ 能量获取 ↓ 屏障支持 ↓
```

### **Tier 2：菌群向宿主的代谢物交换（Host-Microbiota Metabolite Exchange）**

**关键产物与缺陷**：

| 代谢产物 | 正常菌群产生 | IBD炎症期 | 宿主功能影响 |
|---------|----------|---------|-----------|
| **丁酸盐** | Clostridia, Faecalibacterium | ↓↓ | 肠道屏障能量↓，免疫调节↓ |
| **烟酸（NAD前体）** | Prevotella, Bacillus | ↓↓ | 宿主NAD合成↓↓ → 能量危机 |
| **次级胆酸** | Clostridia (胆盐水解酶) | ↓ | 脂肪吸收↓，肝肠轴破坏 |
| **谷氨酸衍生物** | 各种菌株 | ↓ | 氮代谢与转氨化↓ |
| **维生素B群** | 各种菌株 | ↓ | 辅酶不足 |

**深层机制**：
1. **群落组成变化** → 益生菌减少，机会菌增加
   - Clostridia: 9 种代谢物交叉代谢 → 功能菌，需优先保护
   - Bacteroidia: 同样高贡献 → 需要联合策略

2. **代谢能力丧失** → 即使菌株存在，也无法产生关键产物
   - 例：某些 Faecalibacterium 菌株在低 pH/高炎症环境下无法激活丁酸盐通路

### **Tier 3：宿主代谢全局脱调（Host-Level Metabolic Collapse）**

#### **3.1 能量代谢危机**

**核心观察**：NAD 和 FAD 浓度升高（相对缺乏感知）

```
原因链条：
  1. 微生物烟酸产生↓
  2. 宿主色氨酸浓度↓ （色氨酸酶↑，但利用率更快）
  3. 宿主NAD合成两个来源均断：
     - 烟酸途径：缺乏微生物烟酸
     - 色氨酸途径：缺乏色氨酸基质
  4. 线粒体能量生产↓
  5. ATP 不足 → 应激代谢更新（catabolism）↑
```

**临床后果**：
- 肠道上皮细胞能量耗尽 → 屏障功能瘫痪
- 免疫细胞能量不足 → 无法有效清除病原体
- 恶性循环：能量不足 → 屏障破坏 → 更严重的炎症激活

#### **3.2 氨基酸转氨化轴脱调**

**关键发现**：90% 的转氨反应与疾病活动度相关

```
焦点：谷氨酰-精氨酸轴
  谷氨酸 ⇌ α-酮戊二酸 + 谷氨酰胺
  精氨酸 ⇌ 肌酸 + 其他氮衍生物

在IBD炎症期：
  ✓ 转氨酶基因表达↓↓
  ✓ 酶活性↓↓
  ✓ 底物竞争（谷氨酸被用于能量）

导致后果：
  → 氮平衡紊乱（ammonia accumulation）
  → 多胺合成↓ （细胞增殖受损）
  → 谷胱甘肽合成↓ （抗氧化防御崩塌）
  → 磷酸肌酸代谢↓ （肌肉能量储备耗尽）
```

**验证**：血清中 Gln、Arg、His、Lys 均显著下降（p < 0.001）

#### **3.3 一碳（C1）代谢轴与膜脂代谢**

**关键通路**：
```
同型半胱氨酸（来自微生物或宿主）
  ↓ （甲基转移酶）
甲基四氢叶酸 (mTHF)
  ↓ （甲基转移）
5-甲基四氢叶酸 (5-mTHF)
  ↓ （胆碱合酶）
磷脂酰胆碱（细胞膜主要成分）
```

**IBD 炎症期脱调**：
1. **同型半胱氨酸产生↓**
   - 微生物携带同型半胱氨酸合酶 → 产生减少
   - 宿主甲硫氨酸代谢受阻

2. **C1 循环速率↓**
   - 转甲基反应（methylation reactions）↓
   - 胆碱可利用性↓

3. **磷脂重组受阻**
   - 血清磷脂酰胆碱↓（lyso-phosphatidylcholine ↓）
   - 磷脂酰乙醇胺转酯化反应中断
   - 结果：膜结构脆弱 → **肠道屏障完整性进一步破坏**

**临床关联**：磷脂酰胆碱水平与肠道屏障标志物（zonula occludens, claudins）正相关

|--------|------------------|-----------------|------------------|
| **NAD代谢** | 菌群烟酸产生↓ + 宿主合成↓ | NAD合酶基因↓, 色氨酸酶↑ | 色氨酸↓, NAD/NADH比↓ |

| **SCFA生成** | Clostridia代谢↓↓ | 丁酸激酶↓ | 丁酸血浓度↓ |

**最短路径网络分析**：
```
微生物产物 ←→ 宿主代谢中心 ←→ 血清代谢物
(model predict)  (network topology)  (measure)
```

例如：八酰基肉碱（β-氧化中间体） ← FAD/NAD（能量代谢）← 宿主线粒体瘫痪
- **网络解释**：脂肪酸进入但无法氧化（能量危机），积累为血清脂肪酸↑

## 宿主-菌群"代谢双重打击"机制

### **Example 1：色氨酸-NAD 轴的双重打击**

```
             宿主侧                      菌群侧
  
  Tryptopha ──→ Kynurenine ──→ Quinolinate ──→ NAD
  生产↑        分解↑                  合成↓
  
  + 微生物产生的烟酸 ↓↓
  
  结果：NAD合成通道两端均被中断
        能量危机级联 → 肠屏障崩塌 → 免疫激活恶化
```

### **Example 2：SCFA 的多因素衰退**

```
          菌群侧              宿主侧              结果
  
  膳食纤维    ↓          肠道pH↓         低发酵
  ├─ 发酵能力↓    ← 炎症状态
  ├─ Clostridia↓  ← 竞争↑
  └─ 丁酸盐↓↓         屏障破坏↑
                           ↓
                     SCFA吸收↓
                           ↓
                     小肠上皮能量更差↑
```

|---------|---------|----------|

## 膳食干预的代谢学基础

### **为什么膳食很关键？**

膳食不只是营养补充，更是**代谢恢复的桥梁**：

1. **选择性增强特定菌群**
   - 特定膳食纤维 → 优先促进 Clostridia/Bacteroidia
   - 限制其他菌群生存（如乳酸菌）

2. **直接补充缺失前体**
   - 烟酸（Nicotinic Acid）直接供应 NAD 前体
   - 色氨酸补充 → 宿主色氨酸途径恢复
   - 胆碱补充 → C1 代谢通路支持

3. **修复代谢缺陷**
   - 低残基脂肪酸 → 减少不可消化脂肪（减少促炎菌群竞争源）
   - 易消化蛋白（部分水解） → 减少氮负荷，改善转氨化

### **模型预测的膳食策略**

基于代谢模型的逆向工程：
```
观察：Clostridia 代谢缺陷 (butyrate ↓, NAD production ↓)
   ↓
预测：需要增加哪些底物 + 去除哪些竞争物质
   ↓
膳食建议：
  ✓ 增加：低聚果糖（inulin）、阿拉伯木聚糖
  ✓ 减少：高脂肪、高纤维（可能导致pH过低）
  ✓ 补充：烟酸、色氨酸强化食物
```

## 临床应用与前景

### **从理论到实践的转化**

1. **个体化代谢建模**
   - 根据患者 16S 菌群 + 转录组 → 重建其特定的菌群-宿主共代谢模型
   - 预测该患者的代谢缺陷 → 定制膳食方案

2. **代谢标志物驱动治疗（Metabolic-Guided Therapy）**
   - 不只监测炎症（CRP、FCP）
   - 同时监测代谢恢复：血清 NAD、色氨酸、磷脂
   - 代谢恢复滞后 → 提前强化膳食/菌群干预

3. **预防性膳食干预**
   - 在症状缓解前启动代谢导向膳食
   - 加速代谢恢复 → 降低复发风险
