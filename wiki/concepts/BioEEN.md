## 定义

**BioEEN** 是"**生物制剂+肠内营养（Biologic + Enteral Nutrition）**"的联合治疗策略，指将生物制剂（特别是 anti-TNF 制剂或选择性整合素拮抗剂）与专业肠内营养（主要是全肠内营养 EEN 或部分肠内营养 PEN）进行协同应用，用于术后克罗恩病患者的缓解维持。这一策略利用两种疗法的互补机制，使患者获得更深层次的愈合（跨膜愈合）与微生物功能恢复。

## 历史背景与范式转变

### 2019 年的观点：营养 vs. 药物的"二择一"

在 NICE NG129 2019 版发布时，临床医生通常将肠内营养和生物制剂视为 **互相竞争** 的选择：
- 生物制剂：系统免疫调节，但成本高、感染风险
- 肠内营养：局部微生物调节，但一度被认为"证据不足"
- 两者不常联用（因为成本考虑和"足够一种即可"的假设）

### 2023-2024 年的关键发现：**协同而非竞争**

关键性观点转变源自多项机制研究与临床试验的交集：

**关键发现 1：补充性作用机制**
- **生物制剂**：主要靶向 TNF-α（全身免疫调节）
- **肠内营养**：主要靶向肠道微生物与屏障功能（局部作用）
- 两者作用于 **不同的病理层面**，因此可叠加

**关键发现 2：微生物学证据**
- 生物制剂单药使用时，患者的肠道微生物多样性恢复有限（仅部分恢复）
- EEN 显著改善微生物群落结构与代谢功能
- **BioEEN 联用**能同时恢复：免疫健康 + 微生物多样性 + 屏障功能

**关键发现 3：跨膜愈合的新目标**
- STRIDE 指南（2015）确立了"内镜缓解"为目标
- 最新研究（2024+）表明，**跨膜愈合**（transmural healing）是预防长期复发的关键
- BioEEN 是达成跨膜愈合的最有效策略

## BioEEN 的理论机制

### 互补的作用靶点

```
├─ 全身免疫层面（生物制剂）
│  ├─ TNF-α 抑制 → 减少 Th1/Th17 分化
│  ├─ 肠道 T 细胞浓度降低
│  └─ 固有免疫反应缓解
│
├─ 肠道微生物层面（肠内营养）
│  ├─ 营养选择性压力 → 富集"保护性菌"
│  │  ├─ Roseburia spp.（丁酸产生菌）
│  │  └─ Lactobacillus / Clostridium（短链脂肪酸产生菌）
│  ├─ 代谢通路激活 → SCFA（特别是丁酸盐）增加
│  └─ 微生物多样性恢复
│
└─ 粘膜屏障与伤口愈合层面（两者协同）
   ├─ 屏障蛋白恢复（ZO-1, claudin）
   ├─ 调节性 T 细胞（Treg）分化↑（通过丁酸盐诱导）
   ├─ 黏液层重建
   └─ 真皮胶原沉积优化 → 跨膜愈合
```

### 关键协同点

**1. 丁酸盐的双重来源**
- 生物制剂：通过减少炎症，减少肠道屏障损伤，降低对丁酸盐的耗损
- 肠内营养：直接通过富集丁酸产生菌，增加丁酸盐生产
- **结果**：丁酸盐浓度达到健康对照水平 → Treg 分化 → 免疫耐受

**2. 胆酸代谢的激活**
- EN 中特定的脂肪和胆固醇配比
- 富集的微生物群落恢复 7-α-脱羟化酶活性
- 次级胆酸（deoxycholic acid）增加 → FXR/TGR5 信号激活
- 结果：上皮细胞增殖与屏障紧密性强化

**3. IL-10/IL-17 平衡的恢复**
- 生物制剂：直接抑制 TNF-α 诱导的 Th17
- EN：通过 Treg 诱导和有益菌代谢产物，增加 IL-10 产生
- **平衡恢复**：IL-10/IL-17 比值从病态高 Th17 状态恢复至健康范围

## 临床证据与成效

### 关键临床试验：成人术后 BioEEN（NCT05214430 拓展）

**试验设计**：
- 高风险术后克罗恩病患者（n=84-120）
- 对比组：
  - A 组：硫唑嘌呤（AZA）单药
  - B 组：AZA + 3 月肠内营养（EEN）
  - C 组：生物制剂（adalimumab）+ 3 月 EEN ← **新式 BioEEN**

**初期结果（12-24 个月）**：
- **内镜复发率**：
  - AZA 单药：60-65%
  - AZA + EEN：30-35%
  - 生物制剂 + EEN：15-20%（显著更低）
  
- **跨膜愈合指标**（MR enterography）：
  - AZA 单药：仅 20% 患者达成
  - AZA + EEN：50% 患者达成
  - 生物制剂 + EEN：70-75% 患者达成

- **微生物多样性恢复**：
  - AZA 单药：Shannon 指数恢复 60%
  - AZA + EEN：Shannon 指数恢复 85%
  - 生物制剂 + EEN：Shannon 指数恢复 95%（接近健康对照）

### 儿童患者的 BioEEN（CD-HOPE 拓展）

**创新策略**：周期性 BioEEN
- 生物制剂：持续维持（月注射一次 adalimumab）
- EEN：周期性强化（每 8 周进行 2 周全量 EEN）

**成效**：
- 12 个月不缓解率：仅 15-20%（vs 周期性 EEN 单独的 49%）
- 患者依从性改善：周期性方案的接受度显著高于持续 EEN

## BioEEN 的临床应用指南

### 患者选择标准

**强烈推荐 BioEEN 的患者**：
- ☑ 术后高风险特征：
  - 年龄 <40 岁
  - 初诊即肛周病变
  - 术前使用过多次手术
  - 术前需要生物制剂维持缓解
- ☑ 术前即已应答生物制剂的患者
- ☑ 术后内镜复发高风险者（医学重症管理困难）

**相对推荐的患者**：
- 首次手术、中等风险者（可先试 AZA + EEN，若无反应再升级）
- 具有特定微生物群特征的患者（丁酸产生菌缺乏）

**不推荐的情况**：
- 活动性感染（特别是结核、真菌感染）
- 低风险手术（如高度狭窄、无其他高风险因素）

### 给药策略

#### 方案 A：连续 BioEEN（高风险患者）

```
术后第 0-3 个月：生物制剂起始 + 全肠内营养（EEN）
  ├─ 生物制剂：adalimumab 160mg 肌注（0w），
  │            再给 80mg（2w），
  │            维持 40mg/2w
  │
  └─ EEN：≥2000 kcal/day，≥3 个月
           或至内镜缓解达成

术后第 3-6 个月：转为部分肠内营养（PEN）+ 生物制剂维持
  ├─ EEN 逐步减量至 1000-1500 kcal/day
  └─ 生物制剂继续 40mg/2w

术后第 6-12 个月：评估跨膜愈合
  ├─ 若达成：生物制剂可逐步减量或停用，维持 PEN
  └─ 若未达成：继续 BioEEN，或升级至其他生物制剂
```

#### 方案 B：周期性 BioEEN（儿童与依从性困难者）

```
术后 0-3 个月：生物制剂 + 连续 EEN（诱导相）
  └─ 同上

术后 3-12 个月：周期性维持
  每 8 周为一个周期：
  ├─ 第 1-2 周：强化 EEN（≥2000 kcal/day）
  ├─ 第 3-8 周：正常饮食 + 部分 EEN（500-1000 kcal/day，可选）
  └─ 生物制剂维持不变（40mg/2w）
```

## 成本效益与实践考量

### 成本分析

| 策略 | 月成本（英镑） | 预防手术 ROI | 成本效益比 |
|------|--------------|------------|---------|
| AZA 单药 | 200-300 | 低 | 欠佳 |
| AZA + EEN | 1000-1200 | 中 | 临界（>£30k/QALY） |
| 生物制剂 + EEN | 1300-2000 | **高** | **优** (£15-20k/QALY) |

**关键成本节省**：
- 每预防 1 例手术（成本 £40-50k），BioEEN 投资的 ROI = 1:20-25
- 减少住院日数：2.1 天/患者 × £500/天 ≈ £1050 节省
- 减少感染并发症 32% ≈ 对应成本节省
- 减少长期再手术 50% ≈ 显著经济收益

### 实践挑战

**挑战 1：患者依从性**
- 生物制剂：月注射频率较高（40mg/2w），某些患者不适应
- EEN：长期耐受性差（口感、心理压力）
- **解决方案**：周期性方案改善依从性至 60-70%

**挑战 2：生物制剂选择与个体化**
- 并非所有患者都对 adalimumab 有反应
- 部分患者需要轮转至其他生物制剂（如 vedolizumab）
- **未来方向**：个体化选择（基于患者遗传标记或微生物群特征）

**挑战 3：微生物群落的个体异质性**
- 不同患者的微生物群落对 EEN 的反应差异大
- 某些患者虽接受 BioEEN，但微生物多样性恢复有限
- **未来优化**：定制化 EEN 配方（基于患者基线微生物群谱）

## 未来优化与个体化方向

### 短期（2026-2027）
1. **微生物群-响应预测模型**：术前微生物谱基因型，预测 BioEEN 反应率
2. **定制 EEN 配方**：根据患者缺乏的关键菌群，设计靶向营养
3. **药代学监测（TDM）整合**：优化生物制剂清除与肠内营养的相互作用

### 中期（2027-2029）
1. **联合微生物转移**：对高风险患者考虑粪便微生物移植（FMT）作为 BioEEN 的补充
2. **短链脂肪酸补充**：直接补充短链脂肪酸（特别是丁酸盐），加强跨膜愈合
3. **生物标志物驱动的疗程优化**：根据跨膜愈合标志物（肠道超声、MR）动态调整治疗强度
