## 定义

**跨膜愈合（Transmural Healing）** 是指炎症性肠病（特别是克罗恩病）中，肠壁 **全层（从黏膜层到浆膜层）** 的完全愈合与组织学恢复，不仅表现为黏膜炎症的消失，更涉及黏膜下层（submucosa）和肌层（muscularis）的纤维化消退、水肿消除与正常组织结构重建。这是相对于传统的"黏膜愈合"（mucosal healing）的更深层、更具预后意义的治疗目标。

## 历史背景：从"症状缓解"到"跨膜愈合"的演进

### 第一代目标：临床缓解（<1990s）
- 定义：患者自觉症状消失（CDAI < 150）
- 问题：临床缓解者仍有 50% 内镜可见炎症（组织学仍活跃）
- 局限：无法预测长期结局，二次手术率仍然很高

### 第二代目标：黏膜愈合（1990s-2010s）
- 定义：内镜下无溃疡、无活跃炎症（Rutgeerts i0-i1）
- 优点：黏膜愈合与长期预后关联改善
- 遗留问题：仍有 30-40% 患者黏膜缓解但黏膜下仍有纤维化（"隐性复发"）

### 第三代目标：跨膜愈合（2015-2026）
- 定义：全层愈合，无纤维化改变、无血管充血、无脂肪浸润
- 技术支撑：**MR 小肠成像（MR enterography）**、**肠道超声**、**组织学评估**
- 临床意义：**最强有力的长期预后预测因素**，可预防二次手术 60-70%

## 病理学分层理解

### 克罗恩病的组织学层次病变

```
黏膜层（Mucosa）
  ├─ 上皮细胞
  ├─ 固有层（lamina propria）
  │  ├─ 毛细血管充血 → 炎症指标
  │  ├─ 淋巴细胞浸润 → 活跃度指标
  │  └─ 肉芽肿 → CD 特异性病理
  └─ 黏膜肌层（muscularis mucosae）

黏膜下层（Submucosa）
  ├─ 胶原纤维沉积 ← 纤维化的主要部位
  ├─ 脂肪浸润 → 脂肪化（fatty infiltration）
  ├─ 水肿 → 炎症指标
  └─ 淋巴管扩张

肌层（Muscularis Propria）
  ├─ 环肌层
  │  └─ 平滑肌纤维肥厚（术前狭窄的形成）
  └─ 纵肌层

浆膜下层与浆膜（Serosa）
  ├─ 纤维脂肪性肥厚 → MR "肠系膜脂肪绞扰"（mesenteric fat stranding）
  └─ 黏膜粘连与囊肿
```

### 跨膜愈合 vs. 黏膜愈合的对比

| 特征 | 黏膜愈合 | 跨膜愈合 |
|------|--------|--------|
| **表观特征** | 无溃疡、无充血 | + 无纤维化、无脂肪浸润 |
| **检查方法** | 内镜（直视） | MR/超声（全层评估）+ 组织学 |
| **患者比例** | 50-60% 在治疗中达成 | 20-30%（仅在优化治疗后） |
| **1 年复发率** | 仍有 20-30% | 仅 5-10% |
| **5 年无手术率** | 60-70% | **85-90%** |
| **治疗要求** | 中等强度 | 高强度（BioEEN、延长维持） |

## 跨膜愈合的影像学评估

### MR 小肠成像（MR Enterography, MRE）

**评分系统：综合炎症指数（CI）**

```
黏膜层评分（0-3）
  0 = 正常
  1 = 轻度充血、轻度增厚
  2 = 中度充血、中度增厚
  3 = 重度充血、重度增厚

黏膜下层评分（0-2）
  0 = 无水肿、无脂肪浸润
  1 = 轻-中度水肿或脂肪浸润
  2 = 重度水肿或脂肪浸润

肌层与浆膜评分（0-2）
  0 = 正常
  1 = 轻度厚化或信号改变
  2 = 重度厚化或明显纤维化

综合炎症指数（CI）= Σ（各层得分）
  CI 0-2 = 跨膜愈合达成
  CI 3-5 = 部分愈合（黏膜愈合但亚临床炎症）
  CI ≥6 = 活跃炎症
```

### 肠道超声（Transabdominal Ultrasound）

**关键参数**：
- **肠壁厚度**：健康 < 3mm；活跃炎症 > 5mm；跨膜愈合恢复至 3-3.5mm
- **血流信号（Doppler）**：活跃炎症 ≥ 2（Limberg 分级）；缓解 0-1
- **肠系膜脂肪绞扰**：纤维化指标（CI 中的"肌层与浆膜"评分对应）
- **肠系膜扩大**：淋巴水肿与炎症相关

### 组织学评估（活检）

**跨膜愈合的组织学特征**：
- ✓ 黏膜完整，无溃疡
- ✓ 固有层无或极少淋巴细胞浸润
- ✓ 无肉芽肿（或仅见病理前期的稀疏肉芽肿）
- ✓ 肌层无平滑肌纤维肥厚
- ✓ 胶原沉积不超过健康对照 30%

## 跨膜愈合的生物学机制

### 关键愈合通路

#### 1. 血管成生与血管重构
- **VEGF 信号**：血管内皮生长因子刺激新血管形成
- **新血管成熟化**：从"渗漏的"（permeable）新生血管 → 正常成熟的毛细血管
- **黏膜下血管正常化**：减少血管充血与炎症细胞募集

#### 2. 上皮屏障修复与紧密连接
- **Claudin 与 ZO-1 恢复**：上皮细胞间的紧密连接蛋白
- **黏液层重建**：Goblet 细胞功能恢复，MUC2 表达正常化
- **肠道微生物群落恢复**：支持屏障功能的有益菌（丁酸产生菌）增加

#### 3. 纤维化消退机制
- **TGF-β 信号下调**：减少成纤维细胞激活（从而减少胶原产生）
- **基质金属蛋白酶（MMP）激活**：分解过量的沉积胶原
- **肌动蛋白表达下调**：减少肌成纤维细胞（myofibroblasts）分化

#### 4. 免疫耐受机制
- **调节性 T 细胞（Treg）扩增**：IL-10 和 TGF-β 诱导
- **Th17 细胞减少**：破坏性 T 细胞的削减
- **固有淋巴细胞（ILC）平衡**：ILC3（产生 IL-22，支持屏障）恢复

### 关键的分子驱动因子

| 分子 | 来源 | 作用机制 | 相关治疗 |
|------|------|--------|--------|
| **丁酸盐** | 肠道微生物 | Histone 去乙酰化酶抑制 → Treg 诱导，屏障强化 | EEN + 有益菌 |
| **IL-22** | ILC3、Th22 | 上皮细胞增殖与愈合 | 生物制剂诱导 |
| **FGF-7** | 亚上皮成纤维细胞 | 上皮细胞再生与分化 | 营养因子（谷氨酰胺） |
| **VEGF** | 内皮细胞、肌成纤维细胞 | 新血管成生 | 控制炎症（TNF 抑制） |
| **MMP-9** | 中性粒细胞、巨噬细胞 | 基质破坏与重塑 | 控制炎症 |

## 临床实现：跨膜愈合的分阶段策略

### 第一阶段：黏膜愈合的达成（0-3 个月）

**目标**：Rutgeerts i0-i1（内镜下无活跃炎症）

**策略**：
- 强力诱导治疗（生物制剂或高剂量免疫抑制剂）
- 肠内营养支持
- 避免诱发因素（NSAIDs、吸烟）

**成功标志**：
- ✓ 临床症状消失（CDAI < 150）
- ✓ C-反应蛋白正常化
- ✓ 粪钙卫蛋白 < 100 μg/g
- ✓ 内镜缓解（Rutgeerts i0-i1）

### 第二阶段：黏膜下层愈合（3-6 个月）

**目标**：MRE CI 降至 ≤ 3（黏膜下水肿与脂肪浸润消退）

**策略**：
- 继续生物制剂维持
- 肠内营养延续（特别是 EEN，强调丁酸菌产生的诱导）
- 微生物组调节（益生元、益生菌）
- 氧化应激控制（抗氧化剂考虑）

**成功标志**：
- ✓ 肠壁厚度恢复 < 4mm（超声）
- ✓ MRE 黏膜下层评分 ≤ 1
- ✓ 微生物多样性 Shannon ≥ 3.5

### 第三阶段：肌层与浆膜愈合（6-12 个月）

**目标**：MRE 肌层评分 = 0，CI < 2，纤维化消退

**策略**：
- 生物制剂维持（可逐步减量，但通常持续至 12-24 个月）
- 肠内营养转为周期性维持（每 8 周强化 2 周）
- 长期微生物管理（避免抗生素，维护有益菌）
- 心理与生活方式优化

**成功标志**：
- ✓ MRE 肌层与浆膜评分 = 0
- ✓ 肠系膜脂肪绞扰完全消失
- ✓ **"真正的"跨膜愈合达成**

## 跨膜愈合在术后克罗恩病中的特殊意义

### 术后 16 周 EEN 作为跨膜愈合的"唯一保护性因素"

最新研究（2024）通过回归分析发现：

**单因素分析中**，预测跨膜愈合达成的因素包括：
- 生物制剂使用
- EEN 持续时间
- 基线微生物多样性
- 术前 TNF-α 水平
- 患者年龄

**多因素调整后**，仅有 **一个因素保持独立显著性**：
> **"术后 16 周或更长的 EEN 课程"** → OR = 12.5 (95% CI: 4.2-37.3)

**临床启示**：
- EEN 不仅是辅助手段，而是跨膜愈合的 **关键驱动因子**
- 生物制剂强力（adalimumab vs AZA）虽然有帮助，但不如 EEN 持续时间重要
- 最优策略：**BioEEN（生物制剂 + 16周+ EEN）**能达成 70-75% 的跨膜愈合率
