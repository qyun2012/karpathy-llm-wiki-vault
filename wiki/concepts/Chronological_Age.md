## 定义

**时间年龄（Chronological Age）** 是指自出生至今的日历年数，是最简单、最常用的衰老测量方法。虽然时间年龄易于计算和沟通，但在临床医学中，它是一个**极其贫乏的患者状态预测因子**，常常掩蔽真实的生理和功能风险，导致医源性伤害和医学决策的不公正。

## 临床滥用的历史

### 年龄作为"拒绝治疗"的借口

#### 肿瘤学中的传统做法
- **历史实践**："患者年龄太大，不适合化疗"
- **后果**：老年患者被系统性地拒绝有效治疗，尽管他们在生理上可能能够耐受
- **伦理问题**：基于时间年龄的治疗限制违反了医学中的公正原则

#### IBD 管理中的同样问题
- **传统思路**："65-70 岁患者应避免生物制剂"
- **证据现状**：[[Vedolizumab]] 和 [[Ustekinumab]] 在老年 IBD 患者中的安全性与年轻患者相似
- **真实风险**：[[comorbidity-assessment-in-ibd]] 表明，**共病症而非年龄是安全性的真正预测因子**

### 现代医学对年龄偏见的认识

最近 20 年，医学界逐步认识到：

> **时间年龄本身不是患者能否接受治疗的有效指标**

这种认识在老年肿瘤学、老年 IBD、老年心内科等多个领域都得到了印证。

## 时间年龄的核心局限性

### 1. 个体差异性极大

#### 同龄患者的生理差异
同两个 70 岁的患者为例：

**患者 A（生理年龄 ~55 岁）**
- 无高血压、糖尿病、心脏病
- ADL/IADL 完全独立
- 肾脏功能正常（eGFR >70）
- 无认知障碍
- 活跃的社会生活和支持系统
- **结论**：可耐受标准强度化疗或生物制剂治疗

**患者 B（生理年龄 ~85 岁）**
- 有高血压、糖尿病、心衰、肾功能衰退
- IADL 多项缺陷，跌倒频繁
- eGFR 30-45（肾功能中度衰退）
- 认知轻度障碍
- 社会隔离，无主要照顾者
- **结论**：需要保守治疗，强化支持性护理

**关键洞察**：两个患者的**时间年龄相同（70 岁），但生理年龄相差 30 岁，治疗策略应完全不同**

### 2. 掩蔽隐蔽的功能衰退

#### ECOG 评分的无力

**案例**：老年患者 ECOG 0-1（医生评为"可进行化疗"）

但当进行详细功能评估时发现：
- **ADL**：行走、洗浴独立
- **IADL**：但无法独立管理财务、用药、购物
- **身体活动**：Timed Up and Go >15 秒（异常），表示跌倒和虚弱风险高
- **认知**：Mini-Cog 显示轻度认知障碍
- **营养**：体重 6 个月下降 8%

**传统评价**："ECOG 0，可行化疗"
**基于 GA 的评价**："脆弱患者，需要剂量调整和强化监测"

### 3. 不具备可干预性

#### 年龄是固定的
- 无法改变患者的时间年龄
- 医生和患者对此无能为力
- 决策变为被动的拒绝，而非主动的优化

#### 功能年龄是可改善的
- [[Functional_Age]] 中的某些缺陷可通过干预改善
- 营养不良 → 营养支持 → 肌肉质量恢复
- 步态缓慢 → 物理治疗 → TUG 时间改善
- 社会隔离 → 社区资源连接 → 社会支持改善

## 生物学论证：为什么时间年龄是贫乏的

### 生物学多样性

#### 同一时间年龄，不同的衰老轨迹

**分子层次**
- 衰老生物标志物（IL-6、P16、DNA甲美化年龄）个体间相关性弱
- 一个人的"表观年龄"可能远超或远低于其"时间年龄"
- 甚至同一个体的不同组织衰老速度也不同

**细胞层次**
- 细胞衰老（cellular senescence）的速率因人而异
- 干细胞功能衰退程度不同
- 修复机制完整性差异大

**器官层次**
- 有人的肾脏在 70 岁已有 50% 功能衰退，有人仅 20%
- 有人的认知在 70 岁已有显著衰退，有人依然敏锐
- 有人的免疫衰老显著，有人依然能良好应答疫苗

### 衰老的"多路径"模型

衰老不是线性的，也不是同步的。

```
时间轴（年）
    ↓
个体 A：脑 → 心脏 → 肾脏 → 免疫
        年龄 50  60    70    80

个体 B：肾脏 → 免疫 → 脑 → 心脏
        年龄 50  60    70    80
```

同一时间年龄，不同器官的衰老程度完全不同，因此需要器官特异性的评估（如肾功能测试、认知筛查）而非年龄阈值。

## 医学伦理维度

### 年龄歧视（Ageism）在医学中的表现

#### 定义
年龄歧视是指基于年龄的系统性偏见和不公平对待，在医学中表现为：
- 假设所有"老年患者"都虚弱且无法耐受治疗
- 基于年龄而非医学证据做出治疗决策
- 限制老年患者获得有效治疗的机会

#### 证据
- WHO 将年龄歧视列为全球健康问题
- 医学文献记录了因年龄而被限制治疗导致的不良健康结果
- 老年患者常因年龄而被错误分类为"太脆弱"，失去改善生存的机会

### 公正原则（Justice）

医学伦理要求：
- **不伤害（Non-maleficence）**：不因年龄而限制治疗，这本身是伤害
- **行善（Beneficence）**：基于个体的实际状态提供最优治疗
- **公正（Justice）**：患者应基于能力（functional status）而非年龄获得治疗机会

**时间年龄驱动的决策违反了这些原则**

## 临床案例对比

### 案例 1：乳腺癌患者决策

#### 基于时间年龄的传统做法
```
患者：76 岁，诊断激素受体+ HER2- 乳腺癌
决定：仅推荐激素治疗（如芳香酶抑制剂），拒绝化疗
理由："太老了，化疗风险太大"
```

#### 基于 GA 的个体化做法
**情景 A**：患者健康（Fit）
```
GA 评估：无 ADL/IADL 缺陷，CCI=0，认知正常，eGFR=75
决定：考虑化疗+激素治疗，预期耐受良好
```

**情景 B**：患者脆弱（Vulnerable）
```
GA 评估：轻度 IADL 缺陷，CCI=3，营养欠佳，eGFR=45
决定：激素+低剂量化疗+强化营养支持和监测
```

**情景 C**：患者衰弱（Frail）
```
GA 评估：多项 ADL/IADL 缺陷，CCI=5，认知障碍，eGFR=35
决定：仅激素治疗+最优支持性护理
```

### 案例 2：IBD 患者生物制剂选择

#### 基于时间年龄的传统做法
```
患者：68 岁中-重度克罗恩病
决定："年龄太大，避免生物制剂，先试5-ASA或硫唑嘌呤"
结果：疾病进展，最终需要紧急手术，结果不佳
```

#### 基于 GA 和 CCI 的个体化做法
**情景 A**：老年患者但无合并症
```
GA/CCI 评估：CCI=0（无重要合并症）
决定：根据疾病活动性选择生物制剂（Vedolizumab 优先，感染风险最低）
预期：良好的临床反应和耐受性
```

**情景 B**：老年患者伴中度合并症
```
GA/CCI 评估：CCI=2-3（如糖尿病、心脏病）
决定：Vedolizumab（肠道选择性，感染风险较低）+ 强化感染监测
预期：可耐受，需要每 4-6 周临床评估
```

**情景 C**：老年患者伴严重多重合并症
```
GA/CCI 评估：CCI≥4
决定：权衡：
  - 若疾病严重威胁生命：考虑 Vedolizumab + 极强化监测
  - 若疾病相对稳定：保守治疗 + 密切观察
预期：需要多学科管理（GI + 老年医学 + 感染科）
```

## 时间年龄 vs 功能年龄 vs 生理年龄

### 三层年龄的对比

```
衡量维度          时间年龄           生理年龄             功能年龄
─────────────────────────────────────────────────────
定义             日历年数           生物标志物推断       生活活动能力
测量方法         出生日期计算       血液检查、成像        ADL/IADL 评估
客观性           100% 客观         相对客观             相对主观（但可标准化）
个体间差异       大（可达30岁）      大（可达20岁）        大（可达10-15岁功能差异）
临床可用性       高但无用           中（成本、技术）      高（快速、经济）
可干预性         无                 部分（炎症控制）      高（营养、康复、社会支持）
预测力           弱                 中-强（表观年龄）     强
决策价值         低（掩蔽风险）      中-高                高（可操作）
```

## 为什么医学仍然依赖时间年龄

### 历史惯性
- 医学培训中根深蒂固的年龄阈值思维
- 指南中常见的"≥70 岁应避免..."这样的措辞
- 临床医生的认知惯性和工作流的便利性

### 系统性障碍
- GA 工具需要培训和时间（虽然只需 15-20 分钟）
- 医疗报销系统常基于诊断，不鼓励详细的功能评估
- 指南的更新滞后于科学进展

### 对年龄的哲学误解
- 将"年长"与"虚弱"等同
- 忽视个体差异的巨大性
- 对衰老生物学的理解不足

## 从年龄驱动到功能驱动的转变

### 所需的范式转变

#### 医学教育
- 强调衰老生物学和老年医学评估
- 反复教导：**时间年龄不是患者状态的有效预测因子**
- 培训标准化的 GA 工具

#### 临床实践指南
从：
```
"70 岁及以上患者应避免化疗"
```

到：
```
"基于老年医学评估（ADL/IADL、认知、营养、共病症）进行风险分层：
- Fit 患者：可进行标准治疗
- Vulnerable 患者：需要剂量调整和强化监测
- Frail 患者：保守治疗"
```

#### 医疗系统
- 鼓励老年患者的全面评估，而非仅基于年龄的决策
- 提供 MDT（多学科团队）管理的时间报酬
- 更新报销政策，支持 GA 的实施

## 患者倡导与自我赋权

### 患者应了解
- **时间年龄不决定你的医疗**
- 如果医生仅因年龄而拒绝治疗，要求详细的功能评估
- 寻求支持 GA-driven 决策的医疗团队

### 关键问题患者可提出
1. "除了我的年龄，你还评估了什么？"
2. "我的具体风险是什么？不是'太老'，而是具体的医学风险"
3. "有没有可能进行全面的老年医学评估？"
4. "如果我没有这些风险因素，为什么年龄是障碍？"
5. "能否获得第二意见？"
