## 定义

**术后缓解维持（Post-Surgical Remission Maintenance）** 是指通过药物、营养与生活方式干预，在克罗恩病患者术后（通常为肠道切除术后）维持和加强已获得的缓解状态，防止早期复发（endoscopic/clinical recurrence）与长期二次手术的过程。这是克罗恩病外科与医学管理的交界点，涉及多学科协作（外科医生、消化科医生、营养师、微生物学家）。

## 临床背景：术后复发的严峻挑战

### 术后复发的流行病学

术后克罗恩病的复发率在全球数据中极其一致：

| 复发类型 | 时间框架 | 发生率 | 临床意义 |
|---------|---------|-------|--------|
| **内镜复发** | 术后 1 年 | 70-90% | 无症状微观炎症反应 |
| **临床复发** | 术后 2 年 | 20-50% | 患者出现症状 |
| **需要二次手术** | 术后 5-10 年 | ~50% | 进一步的组织损伤与手术风险 |
| **严重内镜复发** | 术后 1 年 | 30-40% | Rutgeerts i3-i4（显著瘢痕/溃疡） |

### 复发的病理生物学

复发的发生遵循可预测的时间序列：

```
术后第 0 个月：手术创伤、吻合口暴露于肠腔内容物
          ↓
术后 1-2 周：固有免疫激活、中性粒细胞浸润
          ↓
术后 1-3 个月：微生物失调（dysbiosis）、炎症细胞因子升高
          ↓
术后 3-6 个月：内膜炎症表现（内镜可见溃疡、炎症）← 内镜复发
          ↓
术后 6-12 个月：瘢痕形成、肉芽肿等肉眼病变
          ↓
术后 12+ 个月：如未控制 → 临床症状出现 → 二次手术
```

**关键转折点**：术后 **3 个月**（医学干预的最佳窗口）

## 维持治疗的演进（2015-2026）

### 第一代标准（≤2018）：单一药物方案

**当时的推荐**：
- **首选**：硫唑嘌呤（AZA）± 甲硝唑 3 个月
- **备选**：5-ASA（已淘汰）、英夫利昔单抗（成本过高）
- **成效**：内镜复发率 40-60%（效果有限）

### 第二代进展（2019-2022）：NICE NG129/1 引发的变革

**关键问题的提出**：
- "肠内营养在术后维持中的角色是什么？"

**实证答案**：
- **EEN 单独**：相对降低复发 30-40%
- **EEN + 免疫抑制剂**：相对降低复发 45-55%（相对优于单药）

### 第三代范式（2023-2026）：多层次协同维持

**现代观点**（基于多组学与机制研究）：
- 不仅要"控制症状"，更要"恢复生物学"
- 靶点从"全身免疫"→ **"免疫+微生物+屏障+营养"的多维度恢复**

**三层策略框架**：

#### 第 1 层：免疫调节（生物制剂或免疫抑制剂）
- 目的：全身炎症控制
- 首选：AZA（经济）或 adalimumab（强效）
- 时程：至少术后 12 个月

#### 第 2 层：微生物与营养调节（肠内营养）
- 目的：恢复肠道微生物多样性、代谢产物、屏障功能
- 方案：全肠内营养（EEN）3 个月 + 周期性 PEN
- 关键：持续到内镜缓解确认（通常 6-12 个月）

#### 第 3 层：患者自我管理与生活方式
- 目的：避免诱发因素，优化肠道环境
- 策略：
  - 饮食优化（低 FODMAP、避免工业油脂）
  - 心理管理（减少应激）
  - 微生物健康维护（避免不必要抗生素）

## 术后缓解维持的关键医学端点

### 从 STRIDE 框架的进展

**STRIDE（Selecting Therapeutic Targets in IBD）2015**：
- 临床缓解：CDAI < 150 或 Harvey-Bradshaw < 4
- 生物标志物缓解：C-反应蛋白 < 3 mg/L 或粪钙卫蛋白 < 100 μg/g
- **内镜缓解**：Rutgeerts i0-i1（无或轻微病变）

**2026 年更新（围术期扩展）**：
- 以上基础 +
- **跨膜愈合**：MR 或超声声学放射学证据显示无活跃炎症纹理、无纤维脂肪浸润
- **微生物多样性恢复**：Shannon 指数接近健康对照
- **长期无症状与无手术**：≥5 年无复发

### 新生物标志物的应用

#### 早期预测复发的标志物

| 标志物 | 基线测量时间 | 预测意义 | 临床用途 |
|-------|-----------|--------|--------|
| 粪钙卫蛋白（fCal） | 术后 4-6w | fCal > 250 μg/g → 高复发风险 | 指导生物制剂强化 |
| 肠道超声声学纹理 | 术后 3 个月 | 纤维脂肪增生 → 预示瘢痕形成 | 决定维持强度与时程 |
| 微生物多样性（Shannon） | 术后 1 个月 | Shannon < 3.5 → 持续失调 | 需强化 EEN 或调整营养 |
| 丁酸盐代谢产物 | 术后 2 个月 | 血浆丁酸盐 < 200 μM → 屏障风险 | 考虑丁酸盐补充 |

## 临床实践指南：术后缓解维持的分层方案

### 患者风险分层

**高风险患者**（复发概率 60-70%）：
- ✓ 初诊时年龄 < 40 岁
- ✓ 肛周病变现在或既往存在
- ✓ 术前需要生物制剂维持
- ✓ 第一次手术即为复杂情况（弥漫性病变、多灶性）
- ✓ 术中发现瘢痕型狭窄（预示高纤维化倾向）

**中等风险患者**（复发概率 30-50%）：
- 初诊年龄 40-60 岁
- 无肛周病变
- 术前用药相对简单

**低风险患者**（复发概率 <30%）：
- 初诊年龄 > 60 岁
- 局限型病变，无并发症

### 分层的维持治疗方案

#### 方案 A：高风险患者（推荐 BioEEN）

```
术后第 0-3 个月（诱导强化期）
│
├─ 免疫调节：生物制剂起始
│   └─ Adalimumab 160mg (0w) → 80mg (2w) → 40mg/2w 维持
│       或 Azathioprine 2.5 mg/kg/day
│
├─ 营养支持：全肠内营养（EEN）
│   └─ ≥2000 kcal/day，持续 ≥3 个月
│       或至 Rutgeerts i ≤ 1（内镜缓解）
│
└─ 微生物监测：术后 4-6w 检测粪钙卫蛋白与微生物多样性
    └─ fCal > 250 → 考虑生物制剂升级（增加频率或剂量）
    └─ Shannon < 3.5 → 加强 EEN 或补充益生元/益生菌

术后第 3-6 个月（巩固过渡期）
│
├─ 免疫调节：继续生物制剂 40mg/2w
│
├─ 营养支持：逐步转为部分肠内营养（PEN）
│   ├─ EEN 1000-1500 kcal/day（减量 25-50%）
│   └─ 逐步引入常规食物，监测耐受性
│
└─ 内镜评估（术后 3 个月）
    ├─ Rutgeerts i0-i1 → 继续现有方案
    ├─ Rutgeerts i2-i4 → 升级治疗（强化 EEN / 更换/加用生物制剂）
    └─ 微生物监测：评估 Shannon 进展

术后第 6-12 个月（维持与评估期）
│
├─ 免疫调节：
│   ├─ 若达内镜缓解 + 无症状 → 考虑逐步减量或停用 AZA
│   └─ 生物制剂继续至术后 12-24 个月
│
├─ 营养：
│   ├─ 若跨膜愈合达成 → 逐步停用 EEN/PEN，过渡至常规饮食
│   └─ 否则 → 继续周期性 PEN（每 8w 强化 2w）
│
└─ 二次内镜评估（术后 12 个月）
    ├─ 完全缓解 + 跨膜愈合 → 可考虑停药
    ├─ 部分缓解 → 延续维持治疗
    └─ 复发迹象 → 升级或更换生物制剂

术后 12+ 个月（长期维持期）
│
└─ 个体化维持：基于生物标志物与症状
    ├─ 若持续缓解（>2 年）→ 逐步递减维持治疗
    ├─ 若症状/生物标志物波动 → 长期低强度维持
    └─ 若再次复发迹象 → 升级干预或更换生物制剂
```

#### 方案 B：中等风险患者（推荐 AZA + EEN）

```
术后第 0-3 个月
├─ 免疫：AZA 2.5 mg/kg/day + 甲硝唑（可选）
├─ 营养：EEN 2000 kcal/day，≥3 个月
└─ 监测：4-6w 粪钙卫蛋白

术后第 3-6 个月
├─ 继续 AZA
├─ 转为 PEN（1000-1500 kcal/day）
└─ 内镜评估（3 个月）
    ├─ 若 Rutgeerts i0-i1 → 继续
    └─ 若 Rutgeerts i2-i4 → 升级至生物制剂

术后 6-12 个月
└─ 继续 AZA，评估是否停用
└─ 过渡至常规饮食（如耐受）
```

#### 方案 C：低风险患者（推荐 AZA 单药 ± 早期营养）

```
术后第 0-3 个月
├─ 免疫：AZA 2.5 mg/kg/day
├─ 营养：早期开始常规进食，可选轻度 PEN（500-1000 kcal/day）
└─ 监测：8-12w 粪钙卫蛋白

术后 3+ 个月
├─ 继续 AZA（至 12-24 个月，根据内镜结果决定停用）
└─ 常规饮食（无需强化营养支持）
```

## 长期结局与成功的定义

### 术后 5 年的关键指标

**成功的术后缓解维持**（定义为）：
- ✓ 无需二次手术
- ✓ 无或极少症状复发（CDAI 多数时间 < 150）
- ✓ 粪钙卫蛋白持续 < 150 μg/g（或完全正常化）
- ✓ 最后一次内镜检查（2-3 年）显示 Rutgeerts i0-i1
- ✓ 微生物多样性在正常范围内（Shannon > 3.5）

### 维持治疗的成本效益

| 方案 | 5 年总成本 | 防止手术次数 | ICER | 成本效益 |
|------|----------|-----------|------|--------|
| 观察（无干预） | £2000 | 0 | — | 高复发率 |
| AZA 单药 | £5000 | 0.3 | £100k/QALY | 欠佳 |
| AZA + EEN | £25000 | 1.2 | £20k/QALY | **优** |
| BioEEN | £45000 | 2.0 | £15k/QALY | **优**（高风险患者） |
