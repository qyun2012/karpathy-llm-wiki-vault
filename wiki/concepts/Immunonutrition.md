## 定义

**免疫营养** (Immunonutrition, INN) 是指含有特定免疫调节营养物质的营养补充，旨在增强患者免疫功能、减少感染并发症和缩短住院时间。最常见的活性成分包括：

- **L-精氨酸 (L-Arginine)**：条件必需氨基酸，参与一氧化氮(NO)和多胺合成，增强T细胞增殖和免疫功能
- **Omega-3多不饱和脂肪酸 (EPA/DHA)**：调节炎症反应，降低促炎细胞因子(TNF-α, IL-6)产生
- **RNA核苷酸 (Nucleotides)**：增强核苷酸依赖的免疫细胞(特别是T细胞)的增殖分化
- **可溶性纤维 (Soluble Fiber)**：特别是部分水解瓜尔胶(PHGG)，促进肠道菌群发酵和SCFA产生

## 核心机制

### 多层次免疫调节

1. **固有免疫激活**
   - 精氨酸→增强粒细胞和巨噬细胞的吞噬作用
   - RNA核苷酸→增强树突状细胞抗原呈递
   - Omega-3→调节补体系统和自然杀伤细胞活性

2. **适应性免疫增强**
   - 精氨酸与核苷酸→T细胞增殖（CD4+和CD8+）
   - Omega-3→降低Th1/Th17与Treg的失衡，抑制促炎细胞因子
   - 纤维→SCFA（丁酸）促进Treg分化，增强IL-10产生

3. **肠道屏障保护**
   - 精氨酸→促进肠上皮细胞增殖和紧密连接蛋白(claudins/occludin)表达
   - 纤维→SCFA提供能量(70%结肠上皮来自丁酸)，增强肠屏障完整性
   - 核苷酸→增强肠道固有免疫细胞功能

## 临床证据（基于2026系统综述）

### 感染预防效果

| 结局 | 干预 | 效果 (OR, 95% CI) | 确定性 | 样本 |
|------|------|-----------------|--------|------|
| **感染并发症** | 免疫营养 | 0.36 (0.21-0.62) | **中等** | 440人, 5项RCT |
| **手术部位感染(SSI)** | 免疫营养 | 0.35 (0.17-0.71) | **中等** | 475人, 3项RCT |
| **血流感染(BSI)** | 免疫营养 | - | - | 仅1项研究 |
| **住院时间(LOS)** | 免疫营养 | -2.64天 | **低** | 5项RCT, I²=85.74% |

### 亚组分析关键发现

1. **给药途径**：
   - 口服：**最优**（OR=0.24, 95%CI 0.07-0.78）
   - 口服+肠内/肠外混合：OR=0.41
   - 纯肠内或肠外：效果较弱

2. **给药时间**：
   - 术前给药：**优越**（OR=0.25, 95%CI 0.06-0.98）
   - 围手术期给药(术前+术中/术后)：OR=0.39

3. **最优成分组合**：
   - **L-精氨酸 + Omega-3 + RNA核苷酸 + 可溶性纤维(PHGG)**：OR=0.26，效果**最强**
   - 不完整组合(缺少部分成分)：效果减弱

4. **年龄差异**（观察性研究）：
   - ≤65岁患者：OR=0.20（更强保护）
   - >65岁患者：OR=0.38（保护作用减弱但仍显著）

### RCT vs 观察性研究的重要发现

关键发现：**分析型研究显示较弱效果**
- RCT研究：SSI的OR=0.35（中等确定性）
- 观察性研究：SSI的OR=0.58（无统计学意义）
- **解释**：观察性研究中高危患者被优先选择免疫营养（逆向选择偏倚），部分掩盖真实效果

## 临床应用指南

### 实施建议

**1. 患者选择**
- ✅ **推荐**：高危手术患者
  - 肿瘤手术（最强证据基础）
  - 胃肠道重建手术
  - 年龄>70或合并症
  - 营养不良或肌肉减少症

- ⚠️ **待定**：低危手术患者（成本-效益需评估）

**2. 推荐制剂与剂量**
```
标准免疫营养配方（每天）：
- L-精氨酸: 10-14 g
- EPA+DHA: 1.5-2.0 g  
- RNA核苷酸: 1-2 g
- 可溶性纤维(PHGG): 3-5 g
- 给药周期: 术前5-7天至术后(根据耐受)
```

**3. 给药方式**
- **第一选择**：口服营养补充(ONS)，最简便有效
- **肠内营养**(EN)：患者无法口服时使用
- **肠外营养**(PN)：仅在胃肠道完全不可用时(罕见)

**4. 时间优化**
```
最优时间点：
术前5-7天 → 术前1天 → 术中(如可行) → 术后早期(48-72h)
关键：术前给药期不应<5天
```

## 经济学证据

### 成本-效益分析（基于系统综述的假设情景）

**假设场景**：500床医院，年度2000例重大肿瘤手术，基线SSI率20%

| 指标 | 数值 |
|------|------|
| 基线SSI患者 | 400例/年 |
| 免疫营养成本 | $100-150/疗程 |
| 总成本 | $200,000-300,000 |
| **预防SSI例数**(基于OR=0.35) | ~260例 |
| **每预防1例SSI成本** | $769-1,154 |
| **SSI相关成本节省** | $780,000-2,600,000 |
| **床日解放** | 1,800-2,600天 |
| **净收益** | $480,000-2,400,000 |

**结论**：即使保守估计，成本-效益比>1，医疗机构应积极实施

| 干预 | 特点 | 证据强度 | 临床应用 | 成本 |
|------|------|---------|---------|------|
| **免疫营养(INN)** | 多成分，靶向免疫 | **中等RCT** | ✅ 推荐 | 中 |

## 需要解决的关键问题

1. **制剂标准化**：全球缺乏统一的免疫营养配方标准，导致成分和剂量差异
2. **长期监测体系**：现有研究多为短期(≤30天)，长期抗生素耐药性和生活质量数据缺乏
3. **低中等收入国家证据**：92%研究来自高收入国家，真实世界效果不清楚
4. **个体化预测模型**：缺乏基于患者特征(遗传、微生物群、代谢)的应答预测
5. **与生物制剂的协同**：IBD患者常合并使用生物制剂，免疫营养与生物制剂的PK/PD协同机制未明
