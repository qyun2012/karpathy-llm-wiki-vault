## 定义

老年IBD诊断的复杂性在于老年患者肠道症状的非特异性（腹泻、便血、体重减轻）极易被误诊为常见的年龄相关疾病。正确诊断前必须穷尽所有可能的鉴别诊断，排除"模拟者"疾病。

## 诊断困境

### 为什么老年IBD诊断困难？

1. **症状非特异性**：腹泻、便血、体重减轻等症状常见于多种老年疾病
2. **多病共存**：许多老年患者同时存在心血管病、神经系统疾病等
3. **多重用药影响**：某些药物（如NSAIDs）可诱发肠炎，混淆诊断
4. **内镜检查风险**：老年患者进行结肠镜检查的风险更高

## 关键"模拟者"疾病

### 1. 缺血性结肠炎（Ischemic Colitis）

**典型临床特征**：
- 突发的左侧腹痛伴随血性腹泻
- 常见于具有心血管风险因素的患者（高血压、心律失常）
- 发作后往往能自行愈合（与IBD的慢性反复相反）

**内镜特征**：
- 病变通常局限于脾曲或降结肠的节段性受损
- 粘膜水肿苍白
- 愈合时间相对较短

**区分要点**：
- 病变的节段性和位置（脾曲）是关键鉴别特征
- 病程通常自限性

### 2. 憩室相关结肠炎（SCAD）

**定义**：
结肠憩室周围的节段性结肠炎，可模拟UC的表现。

**典型特征**：
- 左下腹痛
- 炎症通常局限于受憩室累及的肠段
- **直肠往往不受累**（这是与UC的关键区分点）

**诊断线索**：
- 憩室病背景
- 炎症分布与憩室分布一致

### 3. NSAID诱发肠炎（NSAID-Induced Enteritis）

**背景**：
老年人普遍存在多重用药现象，长期使用非甾体抗炎药（NSAIDs）可诱发肠炎。

**临床与内镜表现**：
- 多发性溃疡和膈样狭窄
- 易与[[Crohn_Disease]]混淆
- 需要药物史详尽询问

**区分方法**：
- 停止NSAIDs后症状改善
- 病变模式与剂量和用药时长相关

### 4. 显微镜下结肠炎（Microscopic Colitis）

**亚型**：
- 胶原性结肠炎（Collagenous Colitis）
- 淋巴细胞性结肠炎（Lymphocytic Colitis）

**流行特征**：
- 在老年女性中多见
- 主要表现为慢性水样腹泻

**诊断困境**：
- **内镜大体检查往往正常**（无可见炎症）
- 必须依靠**分段活检**的组织学诊断
- 胶原带增厚（>10μm）或淋巴细胞浸润

**关键点**：
> 显微镜下结肠炎提示：即使内镜外观正常，也需要活检以排除该病

### 5. 感染性肠炎（Infectious Colitis）

**特别关注：艰难梭菌（C. difficile）感染**

**重要性**：
- 在老年患者中具有极高的致残率和病死率
- 应作为腹泻患者的常规筛查项目
- 常见触发因素：抗生素使用、住院期间

**诊断**：
- 粪便毒素检测或核酸扩增
- 内镜可见假膜性肠炎

**其他感染原因**：
- Salmonella、Shigella、Campylobacter
- 诺如病毒、轮状病毒
- 结核分枝杆菌（在免疫抑制患者中）

### 6. 结直肠癌与淋巴瘤

**临床特征**：
- 贫血
- 大便习惯改变
- 全身症状

**诊断**：
- 肿块或浸润性病变
- 活检明确诊断

## 鉴别诊断的系统方法

### 第一步：病史采集

**关键信息**：
1. **发病方式**：急性突发 vs. 慢性逐渐
2. **症状特点**：腹泻性质（水样、粘液、血便）
3. **药物史**：NSAIDs、抗生素、其他用药
4. **心血管风险**：高血压、心律失常、血管病史
5. **既往憩室病**：家族史或个人史

### 第二步：初步检查

**实验室**：
- 全血细胞计数（检测贫血、感染）
- 肝肾功能
- C反应蛋白、红细胞沉降率
- 粪便钙卫蛋白（FCP）— **无创IBD筛查工具**
  - FCP <50 μg/g：IBD可能性低，考虑其他诊断
  - FCP ≥50 μg/g：提示肠道炎症，需进一步评估

**微生物学**：
- C. difficile toxin/PCR（必须筛查）
- 血培养（若发热）

### 第三步：影像学评估

**腹部CT/MRI**：
- 评估病变分布和特征
- 识别并发症（穿孔、狭窄）
- 区分节段性（ischemic）vs. 弥漫性病变

### 第四步：结肠镜检查

**关键观察点**：
1. **病变分布**：
   - 节段性 → 考虑缺血性、SCAD、NSAID损伤
   - 弥漫性 → 更符合IBD
   
2. **直肠累及**：
   - UC：几乎始终累及直肠
   - SCAD：直肠往往保留（关键区分点）
   
3. **具体内镜特征**：
   - 脾曲病变 → 缺血性结肠炎典型
   - 膈样狭窄 + 多发性小环形溃疡 → NSAID诱发
   - 假膜性肠炎 → C. difficile

4. **活检**：
   - 显微镜下结肠炎诊断必须依靠活检
   - 排除感染（PAS+, AFB等特殊染色）
   - 评估肉芽肿（提示CD）

## 特殊情景应对

### 情景1：患者不愿意结肠镜检查

**替代策略**：
- 粪便钙卫蛋白（FCP）检测（高灵敏度、非侵入）
- CT enterography（评估小肠病变，适合CD）
- 胶囊内镜（若怀疑小肠病变）
- MR enterography（对CD的肠外病变敏感）

### 情景2：患者处于急性发作期，内镜困难

**处理**：
1. 先进行非侵入检查：粪便钙卫蛋白、C. difficile筛查、CT
2. 稳定后再进行结肠镜
3. 临床表现 + FCP + 影像通常已足够初步诊断

### 情景3：多个诊断可能并存

**例如**：患者既往有IBD，现在出现C. difficile感染
- 需要同时治疗两个条件
- FCP仍可能升高（反映IBD的背景炎症）
- C. difficile toxin阳性确认感染

## 诊断决策树

```
老年患者：腹泻、便血或腹痛

↓ 采集详细病史和药物史

┌─→ 发病急性，心血管病史 → 考虑缺血性结肠炎 (FCP可能<50)
│
├─→ 长期NSAIDs使用 → NSAID诱发肠炎 (停药后改善)
│
├─→ 最近抗生素使用 → C. difficile筛查（优先！）
│
├─→ 慢性水样腹泻，内镜正常 → 显微镜下结肠炎（活检）
│
├─→ 憩室病背景，炎症局限 → SCAD（直肠通常保留）
│
└─→ 弥漫性炎症，直肠累及 → IBD（需进一步分型）

↓ FCP检测

FCP <50 μg/g → 器质性病变低可能，重新评估上述诊断
FCP ≥50 μg/g → 进行结肠镜 + 活检确认

```
