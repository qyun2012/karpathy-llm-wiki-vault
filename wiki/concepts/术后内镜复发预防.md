## 定义

**术后内镜复发预防** 是指在克罗恩病（CD）肠道手术后，通过优化营养支持、免疫抑制剂应用、生物标志物监测和生活方式干预等多维度策略，预防术后 1-2 年内的内镜复发（内镜下可见的炎症）的综合医学管理体系。

---

## 临床背景与重大挑战

### 术后复发的流行病学

#### 复发率现状
```
术后 1 年：
  - 内镜复发率：70-90%
  - 临床复发率：20-50%

术后 2 年：
  - 内镜复发率：进一步升高至 80-95%
  - 临床复发率：约 50-60%

术后 5 年：
  - 手术二次复发率：约 40-50%（需再次手术）
```

#### 内镜复发 vs 临床复发的差异

| 特性 | **内镜复发** | **临床复发** |
|------|----------|-----------|
| **定义** | 内镜可见的糜烂或溃疡 | 出现腹痛、腹泻等症状 |
| **发生时间** | 术后 3-6 个月即可出现 | 通常滞后于内镜复发 |
| **患者感知** | 无症状，易被忽视 | 患者自觉不适 |
| **预测价值** | **高**——约 80% 无症状内镜复发最终进展为临床复发 |
| **治疗意义** | 早期干预的机会窗口 | 已经错过防控最佳时期 |

**关键启示**：无症状内镜复发是临床复发的前兆，积极预防内镜复发可显著降低临床恶化风险。

### 术后复发风险的危害

1. **吻合口瘘**：营养不良患者风险增加 5-10 倍
2. **感染并发症**：增加住院时间和医疗成本
3. **肠道功能恶化**：频繁复发导致肠道纤维化，增加再次手术需求
4. **生活质量下降**：反复症状和治疗加重患者负担
5. **医疗成本激增**：每次复发需要新的药物调整和监测

## 术后内镜复发预防的分层策略

### 第一步：术后复发风险分层

#### 低风险患者
**定义标准**：
- 首次手术
- 术前稳定缓解状态
- 无明显营养不良
- 无遗传学高风险标志物

**预防策略**：
- 标准 5-ASA（柳氮磺吡啶或美沙拉嗪）
- 或：AZA/MP 单药
- 营养状态监测（每 3 个月）
- 无症状患者：一般不需要内镜随访（除非接受治疗决策支持体系）

#### 中等风险患者
**定义标准**：
- 术前活跃炎症（需手术的原因）
- 中度营养不良
- 复杂性 CD（瘘管、脓肿）

**预防策略**：
- AZA/MP 或硫唑嘌呤
- 或：一线生物制剂（如 Infliximab）
- 术后早期 EEN（2-3 个月）或周期性 EEN
- 术后 3 个月内镜评估（基线）
- 粪便钙卫蛋白每 2-3 个月监测

#### 高风险患者
**定义标准**：
- 多次手术史
- 术前严重营养不良（BMI < 19，血清白蛋白 < 3.5 g/dL）
- 术前活跃炎症迹象
- 高炎症标志物（CRP > 10 mg/dL，FCP > 500 μg/g）
- 遗传学高风险（CARD15 突变）

**预防策略**：
- **AZA/MP + 术后 3 个月 EEN**（金陵医院 RCT 推荐）
  - 3 个月内镜复发率：8.6% vs 单药 28.1%
  - 12 个月内镜复发率：33.3% vs 单药 63.2%
- 或：早期生物制剂联合
- 术后 1 个月启动密集监测：CRP、粪便钙卫蛋白、血清营养指标
- 术后 3 个月进行内镜评估
- 考虑术后心理支持和营养咨询

## 核心预防策略详解

### 策略 A：营养支持与微生态优化

#### 术后早期营养支持（术后 24 小时内启动）

**模式选择**：

| 模式 | **适用人群** | **给予方式** | **关键参数** |
|------|-----------|---------|---------|
| **口服进食** | 可耐受 | 逐步扩展 | 从清汤→粥→蒸蔬菜 |
| **鼻饲 EEN** | 口服耐受差 | 24h 恒速或间断 | 目标：全营养需求 |
| **经胃造口** | 长期需求 | 间断或连续 | >4 周 EEN 时考虑 |

**营养配方选择**：
- **聚合型（标准）EEN**：适合大多数患者
- **高 ω-3 PUFA 配方**：可能额外抗炎
- **含益生元配方**：支持有益菌群
- **高谷氨酰胺配方**：支持肠上皮修复

#### 周期性 EEN 方案（术后维持缓解）

**最佳方案（CD-HOPE 试验证据）**：
```
周期：8 周一循环
  ├─ 第 1-2 周：100% EEN（禁止常规食物）
  └─ 第 3-8 周：恢复常规饮食 + 生活质量恢复

疗效：12 个月内镜复发率 49%（vs PEN 76%，P = 0.0051）
持续：12-24 个月或长期
```

**给予方式**：
- 第 1-2 周：鼻饲或胃管，连续 18-24 小时/天
- 第 3-8 周：管道移除，恢复常规饮食（带饮食日记）

**生活质量权衡**：
- 优势：6 周"自由期"可参加社交、工作、学习
- 劣势：2 周禁食期的心理和实际负担

#### 饮食管理与微生态重建

**恢复期饮食策略（第 3-8 周）**：
```
第 3-4 周：清汤、清粥、蒸白鱼、鸡肉（低刺激）
第 4-5 周：加入蒸蔬菜、低纤维谷物
第 5-6 周：逐步加入全谷物、豆类、坚果
第 6-8 周：接近常规饮食或地中海饮食
```

**禁止食物**（EEN 周期外，基于耐受性）：
- 高脂肪食物（炸制品、高脂肪肉类）
- 高纤维食物（生菜、坚硬蔬菜、全谷物——初期）
- 乳糖（如果不耐受）
- 刺激性食物（辛辣、浓缩咖啡）

### 策略 B：药物维持与免疫抑制

#### 一线药物选择

| 药物类别 | **推荐等级** | **剂量与方案** | **主要优势** | **主要局限** |
|---------|-----------|-----------|---------|---------|
| **5-ASA** | 可选（证据弱） | 中等剂量 | 安全性好 | 疗效有限 |
| **AZA/MP** | **推荐** | 1.5-2.5 mg/kg/day | 成本低，疗效中等 | 起效慢（2-4周），需监测 |
| **Infliximab** | **推荐**（特别是高风险） | 5 mg/kg at 0,2,6周，后维持 | 快速起效，疗效强 | 成本高，感染风险 |
| **Adalimumab** | **推荐** | 160/80 at 0,2周，后维持 | 皮下注射便捷 | 成本高 |

#### 推荐方案（基于风险分层）

**低风险患者**：
- 5-ASA 或 AZA 单药
- 疗程：至少 2-3 年

**中等风险患者**：
- AZA（推荐）
- 或：一线生物制剂
- 疗程：至少 2-3 年

**高风险患者（推荐）**：
- **AZA + 术后 3 个月 EEN**（金陵医院 RCT）
  - 结果：12 月复发率 33.3%（vs AZA 单药 63.2%）
- 或：**早期生物制剂**（Infliximab ± AZA）
  - 更快的起效，可能更强的早期保护
- 起效时间覆盖：AZA（2-4周）+ EEN（立即）= 最优窗口覆盖

#### 药物监测与安全性

**定期监测指标**：

| 指标 | **基线** | **每月** | **每 3 月** | **目标/阈值** |
|------|---------|---------|---------|-----------|
| **全血细胞计数** | ✓ | ✓ | - | WBC > 3.5K/μL, Plt > 100K/μL |
| **肝功能（ALT, AST）** | ✓ | - | ✓ | <3× ULN |
| **TPMT** | ✓（开始前） | - | - | 通常型或中等活性 |
| **CRP** | ✓ | - | ✓ | <5 mg/L（缓解） |
| **粪便钙卫蛋白** | ✓ | - | ✓ | <250 μg/g（缓解） |
| **血清白蛋白** | ✓ | - | ✓ | >3.5 g/dL |

### 策略 C：生物标志物监测与早期干预

#### 关键生物标志物的临床应用

##### 1. 粪便钙卫蛋白（FCP）— 黏膜炎症指标

**临床价值**：
- 术后 3-6 个月 FCP 升高 → 高复发风险预警
- 可预测 80-90% 的后续内镜复发
- 无症状患者首选的复发风测工具

**监测策略**：
```
术后 1 个月：基线 FCP
术后 3 个月：重要评估点（决定是否需要治疗强化）
术后 6 个月：评估干预效果
术后 9-12 个月：最终评估
```

**基于 FCP 的决策树**：
```
FCP < 250 μg/g（低风险）
  → 继续当前方案，3 个月后重新评估

250-500 μg/g（中等风险）
  → 评估临床症状
  → 如无症状：考虑治疗强化或内镜评估
  → 如有症状：启动治疗调整

> 500 μg/g（高风险）
  → 立即内镜评估
  → 强化治疗（增加生物制剂、优化 EEN）
  → 密集监测（每 2 周 FCP）
```

##### 2. C 反应蛋白（CRP）— 系统炎症指标

**临床应用**：
- 反映全身炎症状态
- 与吻合口瘘、感染风险相关
- 早期发现全身感染

**目标值**：
- 术后 2 周：应逐渐下降（趋势评估）
- 术后 4 周：< 10 mg/L
- 术后 8 周：< 5 mg/L

**异常提示**：
- CRP 升高或不降 → 考虑吻合口漏、腹腔感染
- 需要 CT、腹部超声评估

##### 3. 血清营养指标 — 营养状态评估

| 指标 | **术后基线** | **目标值** | **异常预警** |
|------|-----------|---------|---------|
| **血清白蛋白** | - | > 3.5 g/dL | < 3.0 g/dL 提示严重营养不良 |
| **血清前白蛋白** | - | > 20 mg/dL | 短期营养恶化指标 |
| **总淋巴细胞数** | - | > 1200/μL | < 800/μL 免疫功能严重受损 |

#### 监测频率与策略

**强化监测期（术后 0-3 个月）**：
- 每周：CRP、血清白蛋白
- 每 2 周：FCP
- 每月：完整代谢组（电解质、肝肾功能）

**标准监测期（术后 3-12 个月）**：
- 每月：FCP、CRP
- 每 3 个月：营养指标、肝肾功能

### 策略 D：内镜评估与策略

#### 术后内镜评估的时间点

| 时间点 | **目的** | **关键问题** | **基础或干预决策** |
|------|---------|-----------|-----------------|
| **术后 1 个月** | 评估吻合口愈合 | 吻合口是否光滑完整？ | 如有泄漏迹象，加强营养与抗菌 |
| **术后 3 个月** | 基线评估（重要） | 是否已出现内镜复发？ | 决定后续治疗强化策略 |
| **术后 6 个月** | 监测进展 | 复发程度加重或缓解？ | 评估当前方案的效果 |
| **术后 12 个月** | 定期评估 | 是否达到内镜缓解？ | 确定维持治疗策略 |

#### 内镜复发的 Rutgeerts 评分

```
i0：无病变
i1：≤5 个小的（<2cm）口腔溃疡（轻度复发）
i2：>5 个小口腔溃疡、或较大溃疡、或合并节段性炎症（中度复发）
i3：弥漫性、覆盖面 >1/4 的溃疡、或合并狭窄（重度复发）
i4：弥漫性覆盖面 >3/4 的溃疡、或合并狭窄（极重度复发）

临床转归关联：
  i0-i1：良好预后（10-20% 进展为临床复发）
  i2：中等预后（50% 进展为临床复发）
  i3-i4：不良预后（80-90% 进展为临床复发）
```

#### 基于内镜评分的干预

**无复发或轻度复发（i0-i1）**：
- ✓ 继续当前方案
- 考虑是否可以逐步减少干预（如停止 EEN，改为 PEN）
- 3-6 个月后重新评估

**中等复发（i2）**：
- ⚠ 强化当前方案
- 启动周期性 EEN（如未进行）
- 或：增加免疫抑制剂剂量
- 评估是否需要生物制剂

**重度复发（i3-i4）**：
- 🔴 立即治疗调整
- 启动或升级生物制剂
- 强制启动/延长 EEN 方案（≥3 个月）
- 密集营养支持
- 评估是否需要再次手术

### 策略 E：生活方式干预

#### 饮食管理

**缓解期推荐的长期饮食**：
- [[地中海饮食]]：证据最充分，长期坚持有益
- [[CDED]]（修正地中海饮食）：特别针对 CD
- 个体化"可耐受食物清单"：基于症状日记调整

**关键原则**：
- 高纤维只适合缓解期，活跃期应避免
- 高脂肪食物可诱发症状，应限制
- 乳糖不耐受的患者需替代

#### 生活方式修改

| 因素 | **干预建议** | **证据强度** |
|------|----------|---------|
| **吸烟** | 强烈戒烟（增加复发风险 3 倍） | 强 |
| **压力管理** | 心理咨询、冥想、瑜伽 | 中 |
| **运动** | 适度有氧运动，避免高强度 | 中 |
| **睡眠** | 7-9 小时/晚，规律作息 | 弱 |
| **酒精** | 限制或戒除 | 弱 |
| **NSAIDs** | 绝对避免（诱发活动） | 强 |

#### 心理支持

- 接受诊断后的心理调适支持
- 长期 EEN 带来的心理压力（特别是青少年）
- 积极鼓励参加社交活动和工作
- 必要时转诊心理医生

## 综合预防方案的实施路径

### 高风险患者的理想预防方案（循证推荐）

```
术后第 0 天
  └─ 启动营养支持：术后 24 小时内开始肠内营养
  └─ 启动药物治疗：硫唑嘌呤（或生物制剂）
  └─ 基线检查：CRP、白蛋白、全血细胞计数

术后第 1-2 周
  └─ EEN 周期启动（第 1 个循环）
  └─ 密集监测：CRP（每周）、白蛋白（每周）
  └─ 毒性监测（如用 AZA）

术后第 3-8 周
  └─ 常规饮食恢复，逐步扩展（饮食日记）
  └─ 药物监测：每月评估 AZA 耐受性与疗效
  └─ 心理支持与患者教育

术后第 12 周（关键评估点）
  ├─ 临床评估：症状评估
  ├─ 生化评估：CRP < 5，FCP < 250
  ├─ 内镜评估：Rutgeerts 评分
  └─ 后续方案制定：
      ├─ i0-i1（无/轻度复发）→ 继续当前方案
      ├─ i2（中度复发）→ 强化方案
      └─ i3-i4（重度复发）→ 生物制剂启动或升级

术后第 3-12 个月
  └─ 月度监测：FCP、CRP、临床症状
  └─ 3 个月复查：营养指标、肝肾功能
  └─ 基于 FCP 的决策与调整

术后第 12 个月
  └─ 重复内镜评估（确认缓解状态）
  └─ 评估长期维持方案
  └─ 考虑是否逐步减少干预（风险评估）
```

```
术前：营养优化（EEN 优先）

术后维持（3-12月）：周期性 EEN + AZA（或生物制剂）
长期：基于 FCP 和内镜评估的个体化维持
```

## 预防失败的识别与补救

### 预防失败的定义

**临床预防失败**：
- 术后 12 个月内出现**有症状的临床复发**（腹痛、腹泻、体重下降）

**亚临床预防失败**：
- 术后 3-6 个月已出现**内镜复发**（无症状）
- 或：**生化标志物异常升高**（FCP > 500 或 CRP > 10）

### 失败的原因分析

| 原因 | **常见场景** | **补救策略** |
|------|----------|---------|
| **患者依从性不佳** | 未坚持 EEN、自行停药 | 加强患者教育、家庭支持 |
| **药物耐受性差** | AZA 导致胃肠道症状 | 换用生物制剂、优化给药 |
| **营养不足** | EEN 给予不足或中断 | 强化 EEN、转为更密集方案 |
| **潜在感染** | 吻合口漏、腹腔脓肿 | 影像学评估（CT/超声）、可能需手术 |
| **遗传学高风险** | CARD15 多基因标志物 | 考虑生物制剂升级 |

### 补救方案

**第 1 层（优化当前方案）**：
- 强化 EEN（从周期性改为连续 4-6 周）
- 加强药物监测与依从性管理

**第 2 层（生物制剂启动）**：
- 停用 AZA，启动 Infliximab（或其他 TNF 拮抗剂）
- 或：添加 Vedolizumab（肠道特异性）

**第 3 层（深度评估与转诊）**：
- 外科转诊评估是否需要再次手术
- 消化学专家咨询（精准医学检查）

## 长期预后与维持策略

### 术后 2-5 年的维持原则

**如果术后 1 年达到内镜缓解（i0-i1）**：

**选项 A：继续强化维持**
- 周期性 EEN 继续 2-3 年
- AZA 继续或长期维持
- 优点：最低复发风险
- 缺点：患者负担高

**选项 B：逐步减少干预**
- 12-18 月时：尝试停止 EEN，改为 PEN 或常规饮食
- 评估 FCP 反应
- 如 FCP 不升高（6 个月内），继续 AZA，年度内镜评估
- 优点：改善生活质量
- 缺点：复发风险增加

**选项 C：维持最小化方案**
- 仅继续 AZA（或生物制剂）
- 每 6 个月 FCP 监测
- 无症状内镜复发出现时再启动 EEN
- 需要患者充分理解风险
