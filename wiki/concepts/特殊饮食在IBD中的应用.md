## 定义与核心理念

**特殊饮食在IBD中的应用** 是指通过排除或包含特定食物组分，利用营养作为治疗工具来**诱导或维持IBD缓解**的系统性干预策略。

### 理论基础

```
营养（食物成分选择）
    ↓
肠道菌群重塑（多样性 ↑，益生菌 ↑，致炎菌 ↓）
    ↓
代谢产物改变（SCFA ↑，LPS ↓，色氨酸代谢正常化）
    ↓
屏障修复 + 免疫调节（Tregs ↑，炎症细胞因子 ↓）
    ↓
临床缓解 + 维持缓解
```

**关键假设**：不同饮食成分通过选择性塑造菌群，进而驱动免疫调节，而非直接药物作用。

## 一、主流特殊饮食方案对比

### 1. EEN（独家肠内营养 Exclusive Enteral Nutrition）

#### 定义与组成
- **特点**：口服或肠道给药的营养完全配方，**排除所有固体食物**
- **配方类型**：
  - 整蛋白配方（标准）
  - 半元素配方（肽基）
  - 元素配方（氨基酸基）
- **给药周期**：通常 4-6 周
- **允许液体**：某些方案允许清液体（水、清汤）

#### 机制
```
整蛋白配方
    ↓
减少肠道抗原暴露（特别是蛋白质源和高脂肪）
    ↓
降低肠道通透性 & 改善屏障完整性
    ↓
抑制炎症性 T 细胞反应
    ↓
菌群重塑（虽然多样性可能↓，但致炎菌↓）
```

#### 临床证据

**儿童 CD（最强证据）**：
- **13 项 RCT 的 Cochrane 综述**：EEN 与糖皮质激素效果相当或优于激素
- **缓解率**（4-8 周）：
  - EEN：83-89%（24-27/29-29）
  - 糖皮质激素：61-73%（17/28）
  - **P = 0.031**（意向治疗分析，EEN 优）

**成人 CD**：
- 证据较少，但 Cochrane 综述支持：与激素相当
- EEN + 免疫调节剂的联合使用正在研究

**UC**：
- 证据有限，EEN 通常**不是一线治疗**
- **表 2 元分析**：EEN vs 激素无显著差异（OR 0.77，95% CI 0.58-1.03）

#### 优势
✅ **儿科 CD 中强有力的证据**  
✅ 避免糖皮质激素的系统不良反应  
✅ 菌群改善（虽然暂时）  
✅ 诱导缓解速度快（2-4 周可见效果）

#### 劣势
❌ **患者耐受性差**：
  - 鼻胃管的体积和不适感（难以接受）
  - 长期（6 周）营养单调
  - 社交生活受限，心理影响
  
❌ **菌群多样性暂时下降**：
  - EEN 期间菌群单调化
  - 停止 EEN 后需时间恢复
  - 可能增加条件致病菌定植风险

❌ **不能长期使用**（>6-8 周）

#### 临床应用
- **推荐人群**：儿童 CD 活跃期、对激素不耐受或无法接受
- **给药方案**：鼻胃管或口服（4-6 周），部分患者可逐步混入固体食物

### 2. CDED（CD排除饮食 Crohn's Disease Exclusion Diet）

#### 定义与食物排除清单
**排除的食物组**：
- 全谷物（但**白面包**允许）
- 加工肉类（培根、腊肠、午餐肉等）
- 某些动物脂肪（红肉的高脂肪部分）
- 全脂乳制品（但低脂奶酪、酸奶允许）
- 油炸和超加工食品
- 高糖甜点
- 某些蔬菜（玉米、花生等，如有症状）

**允许的食物**：
- 白面包 & 精制谷物
- 瘦肉 & 鱼
- 低脂乳制品
- 水果 & 大多数蔬菜
- 植物油

#### 机制
```
排除高脂肪、加工、难消化的成分
    ↓
减少肠道黏膜对 MAMPs 的暴露
    ↓
降低炎症性信号（TLR 激活↓）
    ↓
保留更多膳食纤维（相对）
    ↓
益生菌增加（如 Roseburia）
    ↓
SCFA 产生↑，屏障修复↑
```

#### 临床证据

**与 EEN 的比较**：
- CDED 可用于 EEN **之前或之后**，改善耐受性
- **CDED + PEN（部分肠内营养）**：可能与仅 EEN 相当有效，但耐受性更好

**独立应用的证据**（有限）：
- 大多数研究将 CDED 与 EEN 或 PEN 结合
- 孤立的 CDED RCT 数据稀少

**儿科 CD 的前瞻性应用**：
- 5 名患有活跃 CD 的儿童入院，进行 8 周 CDED 治疗：
  - **结果**：5/5 达到临床缓解，疾病活动显著减少
  - **副作用**：无严重不良事件

#### 优势
✅ **可接受性优于 EEN**（普通食物，非配方）  
✅ **社交生活不受限制**（可外出进食，相对灵活）  
✅ **菌群变化可能不如 EEN 剧烈**（保留某些益生菌源）  
✅ **可长期维持**（不仅限于 6 周）  
✅ **无需医疗设备**（鼻胃管等）

#### 劣势
❌ **患者依从性挑战**：排除常见食物，需严格教育  
❌ **证据量少于 EEN**：大多数 RCT 为小样本  
❌ **需个体化调整**（某些患者可能难以耐受）  
❌ **维持缓解数据有限**

#### 临床应用
- **推荐人群**：轻中度 CD 活跃期，寻求可接受的饮食干预或 EEN 的替代品
- **给药方案**：至少 6-8 周，评估反应，可逐步重新引入排除食物

### 3. CD-TREAT（Crohn's Disease Treated）

#### 定义与理念
**创新饮食设计**：旨在用**日常食物组合**模拟 EEN 配方的效果，同时保持口感和可接受性。

#### 食物构成与机制
```
白面包 + 精制淀粉（低纤维）
    ↓
减少肠道黏膜的机械刺激 & 微观颗粒负荷
    
+

瘦肉 + 鱼（易消化蛋白）
    ↓
高生物利用度，低炎症触发

+

低脂乳制品（益生菌源）
    ↓
益生菌定植（Lactobacillus, Bifidobacterium）
    
+

坚果 & 鱼油（微量 n-3 PUFA）
    ↓
抗炎代谢产物
    
→ 综合效果 ≈ EEN 的菌群改变 + 屏障修复
```

#### 临床证据

**小型试点研究**（25 名健康成人和 CD 患者）：
- **设计**：测试 CD-TREAT 在日常生活中的可行性和安全性
- **结果**：
  - 耐受性优于 EEN（容量小、易消化）
  - 患者满意度高（"日常食物"的心理优势）

**活跃 CD 患者的试点数据**（5 名儿童，中等活跃疾病）：
- **8 周 CD-TREAT 治疗**：
  - **5/5 达到临床缓解**（Mayo 评分↓，内镜改善）
  - **疾病活动显著减少**
  - **微生物组变化**：与 EEN 相似的致炎菌↓

#### 优势
✅ **心理接受度最高**（普通食物，无"配方感"）  
✅ **长期可持续**（可无限期使用，与一般人群饮食相近）  
✅ **成本低**（普通食材 vs 昂贵的 EEN 配方）  
✅ **社交灵活性**（可在家庭和学校进食）  
✅ **菌群改变与 EEN 相似**（基于初步数据）

#### 劣势
❌ **证据仍在积累**（RCT 数据有限，多为试点）  
❌ **个体差异大**（食物选择、耐受性的个体化）  
❌ **需精细教育**（患者必须遵循精确的食物组合）  
❌ **规范化不足**（不同中心可能有变体）

#### 临床应用
- **推荐人群**：能接受和遵守特殊饮食的轻中度 CD 活跃期患者，特别是寻求长期可持续干预的患者
- **给药方案**：至少 6-8 周，可长期使用

### 4. 其他特殊饮食方案（证据更弱）

#### 特定碳水化合物饮食（SCD Specific Carbohydrate Diet）
**定义**：排除复杂碳水化合物和多糖，允许单糖和二糖。

| 特性 | 评价 |
|------|------|
| 证据等级 | ⭐⭐（小型试点） |
| RCT 数据 | 仅 2 个对照试验 |
| 儿科 CD | **DINE-CD 研究**：SCD vs MDP，46.5% vs 43.5%（**无差异**），P > 0.05 |
| 生物标志物 | SCD 与 UC 患者微生物组显著改变相关，但临床意义不确定 |
| 劣势 | 排除全谷物和豆类 → 膳食纤维↓，SCFA 产生↓ |
| 风险 | 长期依从性差，营养不均衡风险 |

#### 低 FODMAP 饮食
**应用范围**：**仅限于功能性症状**（腹胀、腹泻在非炎症期）

| 特性 | 评价 |
|------|------|
| IBD 活跃期 | ❌ **不推荐**（减少益生菌底物，菌群多样性↓） |
| 症状管理 | ✅ 在 IBD 缓解期可能改善功能性症状 |
| 证据 | 无 RCT 支持在 IBD 中的诱导缓解作用 |

#### 低乳化剂饮食 & 无超加工食品（UPF）限制
**概念**：超加工食品中的乳化剂和食品添加剂可能破坏肠屏障完整性。

| 特性 | 评价 |
|------|------|
| 机制 | 动物模型证实乳化剂损害粘膜屏障，但人类数据有限 |
| 临床证据 | 1 项交叉试验（7 名 UC 患者）：含 guar 胶的无乳化剂饮食改善症状 |
| 推荐级别 | ⭐ 试验性，支持证据不足 |

## 二、饮食选择的临床决策树

### 患者分层与饮食推荐

```
活跃期 IBD 患者
    ↓
├─ 轻度症状（无系统症状、体重稳定）
│   ├→ 尝试 CDED 或 CD-TREAT（6-8 周）
│   └→ 如无反应 → 升级至 EEN
│
├─ 中度症状（体重↓, 腹泻, 腹痛中等）
│   ├→ **EEN（首选，特别是儿科）**或 CDED+PEN
│   └→ 并行：生物制剂诱导
│
└─ 重度症状（营养不良、频繁肠外症状）
    ├→ 静脉营养 (TPN) 或全肠外营养 (PN)
    └→ 并行：生物制剂 + 可能手术考虑
```

### 缓解维持的饮食策略

```
缓解期患者
    ↓
├─ 膳食纤维：增加至 25-30 g/d（分次引入，评估耐受性）
├─ 全谷物、豆类、水果蔬菜：逐步增加
├─ 减少：加工肉类、含糖饮料、超加工食品
├─ 地中海饮食模式：长期预防复发
└─ 补充益生菌 & 益生元：考虑个体化选择
```

## 三、特殊饮食与菌群变化

### EEN 的菌群影响

**短期（4-6 周）**：
- 菌群多样性↓（Shannon index 下降）
- 致炎菌↓（Enterobacteriaceae ↓, Adherent-invasive E. coli ↓）
- 保护性菌可能↓（Faecalibacterium, Roseburia）
- 优势菌：一些研究报告 Bacillus 和 Bifidobacterium 增加

**停止 EEN 后**：
- 多样性逐步恢复（通常 2-4 周）
- 菌群组成与基线相似性恢复
- **关键问题**：是否恢复到致炎的基线菌群？

### CDED/CD-TREAT 的菌群改变

**预期效果**（基于初步数据）：
- 益生菌（Roseburia, Lachnospiraceae）↑
- 致炎菌↓
- **多样性维持** 或轻度改善（相比 EEN）
- SCFA 产生↑（因保留某些膳食纤维源）

**长期维持**（6+ 个月）：
- 可能建立更稳定的益生菌主导的菌群
- 有利于 IBD 缓解维持

### SCD 的菌群影响

**观察数据**（限制性）：
- 全谷物排除 → 膳食纤维↓ → SCFA 产生↓ → 可能的屏障脆弱性↑
- 某些 SCD 研究报告微生物组显著改变，但临床意义不确定
- **风险**：长期 SCD 可能不利于菌群健康和屏障维护

## 四、特殊饮食的实际应用指南

### 饮食教育的 5 个关键要点

1. **预期管理**
   - 设定现实的缓解时间表（通常 2-4 周对症状改善、4-8 周对内镜愈合）
   - 强调这是**临时诱导期**，之后需逐步过渡

2. **依从性与灵活性**
   - 允许适度的饮食变异（不是绝对严格）
   - 社交场合的实用建议（外出进食、家庭聚餐）
   - 识别个人触发食物（患者可能对某些排除食物有特殊反应）

3. **营养监测**
   - 定期营养评估（每 2-4 周）
   - 微量元素（铁、锌、维生素 D、B12）的血清监测
   - 必要时补充（如缺陷被识别）

4. **菌群生物标志物指导（未来方向）**
   - 基线菌群谱型 → 预测饮食反应？（研究中）
   - 定期粪便 SCFA 或多样性监测（尚未标准化）
   - 个体化调整基于菌群轨迹（原型阶段）

5. **多学科协作**
   - 胃肠病医生 + 营养师 + 微生物学家的团队
   - 共享决策（患者参与选择饮食方案）

### 监测与评估指标

| 时间点 | 临床评估 | 实验室 | 生物标志物 |
|--------|--------|--------|-----------|
| **基线** | Mayo/CDAI，症状日记 | CRP, FCP, 血红蛋白 | 菌群测序（可选） |
| **2 周** | 症状改善？ | CRP/FCP 趋势 | 耐受性评估 |
| **4 周** | 症状 + 体重 | CRP/FCP ↓？ | 粪便微生物学（研究用） |
| **6-8 周** | 内镜评估缓解 | 生物标志物正常化？ | 菌群恢复评估（研究用） |
| **3-6 月** | 缓解维持 | 一般监测 | 长期菌群稳定性 |

## 五、特殊饮食的知识缺口与未来方向

### 关键研究问题

❓ **个体化预测**：
- 基线菌群组成是否可预测特定患者对 CDED 或 CD-TREAT 的反应？
- 宿主遗传学（如 FUT2、NOD2）是否影响饮食效果？

❓ **机制完全理解**：
- EEN 与 CDED 的治疗效果的确切分子机制是什么？
- 是菌群改变还是营养成分本身（或两者）驱动改善？

❓ **长期维持策略**：
- 诱导缓解后最优的过渡饮食是什么？
- 是否存在"缓解维持饮食"的最佳组合？

❓ **CD vs UC 的差异**：
- 为何 CD 对饮食干预的反应更一致？
- UC 中是否存在饮食反应的特异性亚群？

❓ **组合策略**：
- 饮食 + 益生菌 + 益生元 + 药物的最优联合方案？
- 分阶段应用（诱导 → 缓解维持 → 预防复发）？

### 未来方向

🔮 **菌群谱型的分类学方法**（"菌群表型学"）
- 建立"益生菌主导"vs "致炎菌主导"的基线分类
- 为每个表型定制饮食处方

🔮 **生物标志物导向的饮食决策**
- 整合菌群、代谢产物、屏障标志物的多维评估
- 实时监测与动态调整

🔮 **精准营养医学框架**
- 将饮食从"方案选择"转变为"个体优化"
- 与遗传、免疫、菌群的整合模型
