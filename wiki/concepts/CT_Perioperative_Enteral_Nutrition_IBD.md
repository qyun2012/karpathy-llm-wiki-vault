## 试验概况

### 试验题目
**腹部手术前后肠内营养强化（高MCFA配方）对克罗恩病患者术后并发症和肠菌群重建的影响：多中心随机对照试验**

### 注册编号与阶段
- **试验阶段**: Phase III（关键性试验）
- **预期注册编号**: 待申报（ClinicalTrials.gov）
- **研究类型**: 多中心、随机、双盲、安慰剂对照试验

---

## 背景与证据缺口

### 现状分析

围术期是克罗恩病（CD）患者的高风险期，术后复发率极高：
- **12个月复发率**: 40-60%（无干预）
- **5年复发率**: 60-80%（无干预）
- **术后并发症发生率**: 20-40%（吻合口漏、感染、肠梗阻）

### 指南证据缺口

| 临床问题 | 现有证据 | 缺口 |
|--------|--------|------|
| **围术期EN的菌群效应** | 部分队列数据（2019-2020） | 缺乏关键期（术前-术后2周）的RCT数据 |
| **高MCFA配方的耐受性** | 门诊诱导缓解数据（良好） | 术后患者耐受性未知，进食量变化未明 |
| **菌群恢复的最优策略** | Observational（术后肠道菌群丧失） | 无前瞻性强化EN对术后菌群重建的干预证据 |
| **成本-效益** | 部分卫生经济学研究 | 围术期EN成本与复发预防的边际收益未量化 |

**核心证据缺口**: 
> 围术期强化肠内营养（特别是MCFA配方）对CD患者术后菌群重建、术后并发症防控的因果效应未被充分证实。

## 试验设计

### 研究对象

#### 纳入标准
- **年龄**: 18-75岁
- **诊断**: 符合CD诊断标准（ECCO 2016），确诊>3个月
- **手术指征**: 
  - 计划进行腹部手术（小肠切除、结肠切除、回肠造口等）
  - 活动性炎症不需要在术前≥2周内进行，但需要医学管理稳定
- **基线粪便菌群特征**（术前2周检查）:
  - 高风险群体优先纳入（便于探索获益最大化）
  - 肠杆菌属 >30% 或 F. prausnitzii <2% 或 Shannon指数 <2.8

#### 排除标准
- 急诊手术（>24小时内必须进行）
- 全肠外营养（TPN）依赖患者
- 吞咽困难或胃肠道梗阻无法进行EN
- 活动性肠外感染（≤2周内）
- 妊娠或哺乳期女性
- 严重肝肾功能不全（CHILD评分 >B 级，eGFR <30 ml/min）
- 心血管或肺部重症疾病（ASA评分 ≥4）

#### 预期入组数量
- **总人数**: n = 180（90人/组）
- **样本量计算**:
  - 主要结局: 术后12个月临床复发率
  - 假设无干预复发率 55%，干预组降至 35%
  - α = 0.05（双侧），β = 0.2（80%功效）
  - 预留脱落率 15%

### 试验组间

#### 1. 干预组（高强度肠内营养，n=90）

**术前阶段**（手术前2-3周）
```
配方选择：强化全聚合型或半元素型（MCFA ≥40%）
- 推荐产品: Peptamen（MCFA 52%）或 Peptisorb（MCFA 50%）
- 替代产品: 国产配方 + MCT Oil 强化（自定义 MCFA 40-50%）

进食方案：
- 分次或鼻饲（根据患者意愿与消化耐受性）
- 目标摄入: 1.5-2.0 L/天（满足60-70% 热量需求）
- 补充普通饮食: 30-40% 热量（餐桌食物，患者耐受范围内）
- 历程: 2-3周，直至手术前24小时

目标指标：
- MCFA 目标结肠浓度: 15-25 mM
- 粪便菌群目标: F. prausnitzii ↑ 至 >3%，Roseburia ↑ 至 >5%
- 粪便钙卫蛋白 (FCP) ↓ 至 <100 μg/g（或基线↓>40%）
```

**术中管理**
```
术中禁食，肠内给药管（feeding tube）保留或重新置入
- 胃管或鼻肠管: 取决于手术范围与外科团队决策
- 无特殊肠内营养干预（术中麻醉下）
```

**术后阶段**（手术后第1-14天）

**第1-3天: 过渡期**
```
- 禁食: 维持肠外支持（标准TPN）
- Day 3: 胃/肠内管通畅检查，开始逐步进EN
- 初始 EN: 10-20 mL/h（同干预组配方）
```

**第4-14天: 强化恢复期**
```
配方: 继续术前同种高MCFA配方

逐步进食方案：
| Day | 浓度 | 进食量 | 频率 | 目标 |
|-----|------|--------|------|------|
| 4-5 | 25% | 20-30 mL/h | 连续 | 耐受性评估 |
| 6-7 | 50% | 40-50 mL/h | 连续 | 无腹胀/腹泻 |
| 8-10 | 75% | 60-80 mL/h | 连续或间歇 | 达到目标量 |
| 11-14 | 100% | 100-150 mL/h | 间歇或口服 | 目标: 1.0-1.5 L/天 |

EN耐受性监测:
- 每日评估: 腹胀、腹泻、残留胃液
- 腹胀/腹泻处理: 
  - 轻度（无临床影响）: 维持进食，密切观察
  - 中度（进食减少>20%）: 暂停2-4小时，重新评估
  - 严重（>50% 进食减少，有临床症状）: 暂停EN，改为TPN，咨询营养师与外科团队
```

**目标指标（术后Day 14）**:
- EN耐受量: ≥1.0 L/天（热量>50%）
- FCP变化: 相比基线↓ 20-30%（反映术后炎症改善）
- 粪便菌群: 初步恢复迹象（F. prausnitzii 和 Roseburia 未进一步丧失）

---

#### 2. 对照组（标准围术期管理，n=90）

**术前阶段**（手术前2-3周）
```
配方选择：标准全聚合型或无特殊饮食干预
- 推荐产品: Ensure Plus 或 国产康全素（MCFA 15-25%）
- 或: 患者常规饮食（无强制EN）

进食方案：
- 同时饮食（标准家庭饮食）
- 无强化肠内营养
- 基线菌群特征记录，无特殊干预

基线检查：粪便菌群、FCP等参数记录（用于对比）
```

**术中管理**
```
标准围术期管理（同干预组）
```

**术后阶段**（手术后第1-14天）

**第1-3天**
```
标准术后TPN支持（常规方案）
Day 3: 开始逐步进EN（标准配方或清流饮），不涉及高MCFA强化
```

**第4-14天**
```
标准围术期EN方案:
- 标准配方（全聚合型，MCFA 15-25%）
- 或逐步过渡至普通清流饮→软食→常规饮食
- 无特殊肠内营养干预

进食方案：以患者耐受和外科医生建议为主（无强化目标）
```

### 随机分组与盲法

**随机分配**
- 1:1 随机分配（干预组 vs 对照组）
- 分层随机因素:
  1. 性别（男/女）
  2. CD亚型（小肠型 vs 结肠型 vs 全消化道型）
  3. 基线粪便菌群风险分层（高风险 vs 低风险）
  4. 研究中心

**盲法设计**
- **患者**: 部分盲（EN配方外观/口感可能不同，完全盲法困难）
  - 对策: 告知患者"两种营养方案对比"，不透露具体成分差异
- **医护人员**: 不盲（EN配方处理需要具体知识）
  - 对策: 主要结局评估由独立第三方（不知分组）进行
- **结局评估者**: 完全盲法（数据分析人员）

## 主要结局与次要结局

### 主要结局（Primary Endpoint）

#### 1. **术后12个月临床复发率**（主要对比指标）
```
定义: 患者在术后12个月内出现以下任一项:
  • CDAI ≥150（临床复发）
  • 需要重新使用或升级生物制剂/免疫抑制剂（医学失败）
  • 需要再次手术（结构性复发）
  
评估时间: 
  - Month 6（中期）
  - Month 12（主要终点）
  
预期效应:
  - 干预组: 35-40% 的复发率
  - 对照组: 55-60% 的复发率
  - 相对风险降低: 30-35%
```

#### 2. **术后14天粪便菌群恢复的生物标志物**（机制验证）
```
指标:
  1. F. prausnitzii 相对丰度 (%)
     - 基线 vs Day 14 的变化
     - 目标: 干预组 F. prausnitzii ↑ ≥50% 相比对照组
     
  2. Shannon 多样性指数 (α多样性)
     - 基线 vs Day 14 的变化
     - 目标: 干预组维持 >2.8，对照组可能进一步下降
     
  3. Roseburia 丰度 (%)
     - 基线 vs Day 14 的变化
     - 目标: 干预组 Roseburia ≥5%，对照组 <3%
     
  4. 肠杆菌属丰度降低率
     - 基线 vs Day 14
     - 目标: 干预组肠杆菌属↓ ≥20%，对照组↓ <10%

检测方法:
  - 16S rRNA 基因测序（V3-V4区，Illumina MiSeq）
  - 或完整基因组测序（Shotgun metagenomic）
```

- **患者随访**: 12个月/人

## 安全性管理

### 预期不良事件（Adverse Events）

#### 常见不良事件（预期）
| 不良事件 | 高危人群 | 处理 | 严重度评级 |
|---------|--------|------|----------|
| **胃肠胀气** | MCFA 不耐受患者 | 减速进食、暂停4h重评 | 轻-中 |
| **腹泻** | 术后肠道菌群失衡 | 益生菌辅助、液体管理 | 轻-中 |
| **胃残留量升高** | 术后早期、胃动力差 | 胃动力促进剂、胃肠减压 | 中 |
| **肠内管阻塞/移位** | 高浓度配方、长期留置 | 及时更换/重新置入 | 轻-中 |
| **吸入性肺炎** | 吞咽困难、意识改变 | 暂停EN、提升床头角度、评估进食方式 | 中-重 |

#### 严重不良事件（Serious Adverse Events, SAEs）
- **定义**: 导致住院延长、生命威胁或死亡的事件
- **预期SAE**:
  - 肠梗阻（术后>3%）
  - 吻合口漏（术后>5%，非EN相关）
  - 败血症（术后感染相关）

#### 不良事件报告与监测
```
轻度不良事件：
  - 记录在患者日志中
  - 周一次回顾与报告（Week 1-2）
  
中度不良事件：
  - 24小时内报告至主要研究者
  - 监测委员会周一次评审
  
严重不良事件：
  - 24小时内报告至数据安全监测委员会（DSMB）
  - 可能导致暂停患者入组的评估
  
死亡事件：
  - 4小时内报告（如适用）
```

### 数据安全监测委员会（DSMB）
- **组成**: 独立临床医学、统计学、伦理学专家
- **职责**:
  - 每半年（或每20例入组后）审核安全性数据
  - 评估入组/随访的质量
  - 决策是否继续/修改/停止试验
  
- **停止规则** (Pre-specified):
  - 任一组出现 **≥3 例与EN相关的SAE**（肺炎、肠梗阻）
  - 死亡事件数量异常升高（p < 0.05）

## 统计分析计划

### 分析人群
1. **Intent-to-Treat (ITT)**: 所有随机患者
2. **Per-Protocol (PP)**: 完成>80%协议的患者
3. **Safety**: 接受≥1次EN的患者

### 主分析
```
主要结局 - 术后12个月临床复发率:

分析方法: Kaplan-Meier曲线 + Cox回归
  - KM曲线: 两组复发率轨迹对比
  - Cox模型: 调整分层因素（性别、CD亚型、基线菌群）
  - HR与95% CI报告
  - 非劣效性检验（若HR 0.65-1.0）

主要结局 - 术后14天菌群恢复:

分析方法: 广义线性模型 (GLM)
  - 因变量: F. prausnitzii 相对丰度（Day 14 vs Baseline）
  - 自变量: 分组（干预 vs 对照）+ 分层因素
  - 计算 β系数、95% CI、p值
  
多样性指数分析（Shannon):
  - Wilcoxon秩和检验（若非正态分布）
  - 或t检验（若正态分布）
  - 报告 Cohen's d 效应量
```

### 亚组分析
```
预指定的亚组分析:
  1. 按CD亚型分层
     - 小肠型 vs 结肠型 vs 全消化道型
     - 交互作用检验（p_interaction）
     
  2. 按基线菌群风险分层
     - 高风险（F. prausnitzii <2%） vs 低风险（≥2%）
     - 假设高风险患者获益更大
     
  3. 按EN耐受性分层
     - 高耐受（Day 14 EN量 ≥1.0 L/day）vs 低耐受（<1.0 L/day）
     - 探索EN进食量与复发风险的关系
     
  4. 按年龄段分层
     - 年轻组（18-50岁） vs 中老年组（>50岁）
     - 探索年龄对MCFA耐受性的影响
```

### 敏感性分析
```
1. 遗漏数据处理:
   - 主分析: MCAR (Missing Completely At Random) 假设下的完全病例分析
   - 敏感性: 多重插补法（Multiple Imputation）
   
2. 脱落患者分析:
   - 最坏情景: 脱落患者假设为复发
   - 最好情景: 脱落患者假设为无复发
   
3. 协议偏离的影响:
   - 排除严重协议偏离患者的结果
```

### 多重检验校正
```
Bonferroni校正（保守估计）:
  - 主要结局 × 2（KM曲线 + Cox模型）
  - 次要结局 × 8（如上表）
  - 总检验数 = 10
  - 调整α = 0.05 / 10 = 0.005

或采用False Discovery Rate (FDR) 控制
  - 更灵活的多重检验校正
```
