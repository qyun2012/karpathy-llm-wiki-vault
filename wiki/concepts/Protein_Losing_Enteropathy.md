## 定义

**蛋白丢失性肠病(Protein-Losing Enteropathy, PLE)**是指由于肠道黏膜完整性破坏，导致血清蛋白(特别是白蛋白和免疫球蛋白)和其他分子不可控制地漏出进入肠道，从而在粪便中丧失的病理过程。

**关键特征**：
- 血清蛋白、生物制剂等大分子物质直接从肠道泄漏
- 这个过程在活跃期IBD中是**连续且大量的**
- 是低白蛋白血症的主要原因之一
- 直接影响生物制剂的血清浓度(药物从肠道丧失)

---

## IBD中PLE的流行程度

### 临床表现

| 时期 | IBD患者中的营养不良患者比例 | PLE的贡献程度 |
|------|----------------------|------------|
| 缓解期 | 39% | 低(主要原因：进食减少、吸收不良) |
| 活跃期 | 83% | **高**(主要原因：PLE + 炎症消耗) |

### 定量评估指标

**反映PLE程度的检查**：
- **α-1抗胰蛋白酶清除率**(临床金标准但费时)
- **51Cr-白蛋白追踪**(研究用)
- **粪便α-1抗胰蛋白酶**(便利但特异性不100%)
- **血清白蛋白水平**(间接指标)
- **粪便钙卫蛋白**(反映肠道炎症程度，与PLE相关)

## PLE的病理生理机制

### 1. 粘膜屏障的物理破坏

#### 正常肠道屏障

- 单层柱状上皮细胞紧密排列
- 紧密连接蛋白(Tight Junction Proteins)：ZO-1, occludin, claudins
- 形成"选择性通透"的屏障：小分子通过，大分子阻挡
- 维持血清与肠腔之间的压力梯度平衡

#### IBD中的屏障破坏

**活跃期特征**：
- **深度溃疡**：上皮细胞完全脱落，露出下层肉芽组织
- **微血管暴露**：溃疡底部直接暴露毛细血管网络
- **紧密连接蛋白降解**：炎症蛋白酶(MMP-9等)切割ZO-1等蛋白
- **上皮细胞凋亡加速**：TNF-α、IFN-γ诱导的细胞程序性死亡

**物理后果**：
粘膜就像一个"有漏洞的筛子"，允许血液中的蛋白质直接流入肠腔

### 2. 血清与肠腔压力差的反转

#### 正常状态
```
血清胶体渗透压 (25 mmHg) 
  > 肠腔胶体渗透压 (5-10 mmHg)
  
→ 蛋白质向血液侧转移(吸收)
```

#### 活跃期IBD
```
肠道炎症 → 毛细血管压升高(充血)
         → 肠腔压升高(水肿、积液)
         
血清胶体渗透压 (25 mmHg) 
  < 肠腔胶体渗透压 (升至15-20 mmHg)
  
→ 蛋白质向肠腔侧转移(渗漏)
```

### 3. 溃疡面的"毛细血管易位"

**Crohn病中的特殊机制**：
- 深层穿透性溃疡
- 暴露的肉芽组织血管丛与肠腔直接相连
- 大量血清蛋白持续从裸露的毛细血管床漏出

**可视化**：想象粘膜的一个"创面"，渗液源源不断流向肠腔。

## PLE对患者的直接危害

### 1. 低白蛋白血症与营养危机

**白蛋白丧失的量化**：
- 正常人：肠道白蛋白周转约4-5 g/天
- 活跃期IBD伴PLE：可达20-40 g/天(增加5-10倍)
- 肝脏代偿产生新白蛋白的能力有限(~10 g/天)
- **结果**：血清白蛋白进行性下降，可跌至<2.5 g/dL

**生理后果**：
- 血清胶体渗透压↓ → 周围组织水肿(腿肿)
- 凝血因子合成↓ → 凝血功能异常
- 免疫球蛋白丧失 → 感染风险↑

### 2. 生物制剂的直接丧失

**关键现象**：注射到患者体内的生物制剂(IFX, ADA, VDZ等)与内源性白蛋白竞争，同样会通过破损的粘膜漏出。

**定量估算**：
在深度溃疡病变中，可能有10-30%的给药生物制剂直接进入肠道，在粪便中丧失。

**临床影响**：
- 患者本应在血清中有的生物制剂浓度会显著低于预期
- 医生看到"低谷值"，可能误认为是清除过快，而升级剂量
- 实际可能是药物漏出，不是代谢加快
- 结果：过度给药，浪费昂贵的生物制剂

## PLE与新生儿Fc受体(FcRn)的恶性循环

### FcRn的正常功能

[[FcRn_Receptor_and_IgG_Recycling]]：
- 血管内皮细胞表面的受体
- 将IgG(抗体类生物制剂)和白蛋白从溶酶体拯救出来
- 使它们回收进入血液，延长其半衰期(3-4周)

### PLE破坏FcRn的两种方式

#### 方式1：竞争性饱和

**正常状态**：
```
少量要被降解的IgG和白蛋白 
  + 足量FcRn受体
  = FcRn可高效回收两者
```

**活跃期PLE**：
```
大量IgG和白蛋白被迫周转(为补偿丧失)
  + 有限的FcRn受体数量
  
结果：FcRn超载，无法全部回收
     生物制剂(IgG)因缺乏FcRn保护而加速降解
```

#### 方式2：内皮细胞功能损害

**炎症对FcRn表达的直接抑制**：
- 活跃期的TNF-α、IL-6等细胞因子
- 直接抑制内皮细胞的FcRn表达
- FcRn蛋白数量↓ → 回收效率↓

### 数学模型的证明

**蒙特卡洛模拟(儿童CD患者标准IFX给药)**：

```
基线白蛋白3.0 g/dL (严重低下, 代表活跃期PLE)
  → 标准IFX维持剂量(5 mg/kg q8w)
  → 仅21%的概率在Week 14达到治疗谷值(>3 μg/mL)

改善白蛋白至4.0 g/dL (接近正常)
  → 相同剂量IFX
  → 概率升至41% (+95%)
```

**启示**：纠正白蛋白(从而解除FcRn饱和)可使生物制剂成功率翻倍！

## 临床识别与诊断

### 常见症状和体征

| 症状/体征 | 频率 | 临床意义 |
|--------|------|--------|
| 下肢水肿(腿肿) | 常见 | 血清胶体渗透压↓ |
| 体重减轻 | 常见 | 蛋白质大量丧失 |
| 腹胀 | 常见 | 炎症+低蛋白血症诱发的肠腔积液 |
| 反复感染 | 时有 | 免疫球蛋白丧失 |

### 检查指标

**第一步：证实低白蛋白血症**
- 血清白蛋白<3.5 g/dL (正常4.5-5.5)
- 血清总蛋白<6.0 g/dL

**第二步：排除其他原因**
- 肝脏合成功能：看肝酶、PT/INR、albumin/globulin比值
- 尿蛋白：确认不是肾病综合征

**第三步：证实PLE**
- 粪便α-1抗胰蛋白酶清除率(>5 g/天)
- 血清β-2微球蛋白↓(同时丧失)

**第四步：评估肠道炎症程度(反映PLE严重程度)**
- 粪便钙卫蛋白(FCP) — 越高PLE越重
- 肠镜下溃疡面积与深度

## PLE的纠正策略

### 1. 消除病因(根本之策)

**控制肠道炎症**，使粘膜屏障修复：
- 启动或优化生物制剂治疗
- **膳食干预**(见下)
- 必要时加用免疫抑制剂

### 2. 营养补充(支持疗法)

| 方案 | 机制 | 时长 |
|------|------|------|
| **完全肠内营养(EEN)** | 完全低刺激液体营养 | 6-8周(短期冲击) |
| **部分肠内营养(PEN)** | 供应30-50%热量 | 长期(维持) |
| **膳食高蛋白** | 增加蛋白摄入来补偿丧失 | 进行中(长期) |

**量化目标**：
- 蛋白摄入：1.5-2.0 g/kg/day(正常为0.8 g/kg/day)
- 热量：30-35 kcal/kg/day

### 3. 膳食干预(关键环节)

**CDED或EEN可通过以下机制迅速修复屏障**：
- 移除促炎食物成分 → 肠道炎症↓ → TNF-α↓
- 低刺激性营养供应 → 上皮细胞负荷↓
- 通常2-4周内可观察到：
  - 粘膜肉眼愈合开始
  - FCP显著下降
  - 粘膜屏障功能改善指标(粪便渗漏指数↓)

### 4. 补充微量营养素

活跃期PLE患者同时丧失多种微量元素：
- **铁**：血红蛋白合成↓ → 贫血
- **锌**：免疫功能↓ + 手术耐受性↓
- **镁、钾**：肠道渗漏↓ + 腹泻丧失
- **维生素B12, 叶酸**：肠道吸收↓

**补充建议**：
```
活跃期IBD伴PLE患者应定期(每3-6月)检查：
  - 铁代谢(铁蛋白、转铁蛋白饱和度)
  - 血清锌、镁
  - B12、叶酸
  - 维生素D
并根据结果补充
```

## 膳食干预对PLE的机制作用

### EEN/CDED的修复时间线

```
Day 1-3: 移除食物刺激
  ↓ (肠道炎症信号减弱)
Week 1-2: 上皮细胞凋亡率↓, 紧密连接蛋白表达↑
  ↓ (粘膜开始愈合)
Week 2-4: 肉眼可见溃疡缩小, FCP显著下降
  ↓ (屏障完整性提升)
Week 4-6: 血清白蛋白开始升高, PLE闭合
  ↓ (生物制剂不再漏出)
Week 6-8: 完全修复, 可逐步过渡到普通食物
```

### 修复的分子证据

**粘膜活检显示**：
- 紧密连接蛋白(ZO-1)重新表达
- 杯状细胞密度恢复
- 固有层淋巴细胞浸润明显减少

**这意味着**：不仅是"症状缓解"，而是真正的组织学修复！

## PLE对生物制剂TDM的影响(重要!)

### 临床误诊风险

**常见情景**：

```
患者血清IFX谷值<2 μg/mL
医生的第一反应："药物清除过快，升级剂量！"

但实际可能是：
  ✗ 真的清除过快
  ✓ 或者大量药物通过PLE漏出
    (血清浓度低，不是因为代谢，而是因为流失)
```

### 如何区别？

**关键信息**：获取患者的**FCP**和**血清白蛋白**：

| 血清IFX | FCP | 白蛋白 | 最可能诊断 | 对策 |
|---------|-----|--------|----------|------|
| 低 | 高(>200) | 低(<3.5) | **PLE主导** | 膳食干预优先(EEN/CDED) |
| 低 | 低(正常) | 正常 | 真正的高清除 | 升级剂量或换药 |
| 低 | 高 | 正常 | 进行中的炎症+其他因素 | 联合膳食+考虑升级剂量 |

## PLE对患者预后的影响

### 短期(6-12月)
- 营养不良进行性加重
- 生物制剂失效风险升高
- 手术耐受性↓, 术后并发症↑

### 长期(1-5年)
- 严重贫血、营养缺陷
- 生长发育延迟(儿童)
- 反复感染
- 如未得到控制，最终可导致需要外科干预

### 转折点：膳食干预的影响
```
有膳食干预:
  Week 1-4: 迅速修复
  Month 2-3: PLE闭合, 白蛋白恢复
  Month 6+: 生物制剂疗效显著提升
  
无膳食干预(单纯医药):
  Month 1-3: 缓慢、不稳定的改善
  Month 6+: 可能需要多次剂量调整
  Long-term: 较高的复发风险
```
