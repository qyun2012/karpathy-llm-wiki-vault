# Nature Review 论文的高质量图表设计方案

## **Figure 1：四阶段闭环诊疗框架（Infographic）**

### **图表类型**：流程环形图 + 时间线混合

### **核心设计**：
```
                 ┌─────────────────────────────────────────┐
                 │   患者生命周期：从诊断到停药管理        │
                 └─────────────────────────────────────────┘

        ┌──────────────────────────────────────────────────────────┐
        │                   PHASE 1: DIAGNOSIS                     │
        │                  (T = Week 0)                            │
        │  采集 4 层基线数据                                         │
        │  ├─ Clinical phenotype                                   │
        │  ├─ Biomarkers (CRP/Cal/SAA)                            │
        │  ├─ Shallow omics (16S/cytokines)                       │
        │  └─ Deep omics (metabolome/transcriptome)               │
        │  ↓ 建立个性化参考轨迹 f(t)                              │
        └──────────────────────────────────────────────────────────┘
                              ↓
        ┌──────────────────────────────────────────────────────────┐
        │                  PHASE 2: INDUCTION                      │
        │              (T = Week 4, 8, 12)                        │
        │  三次决策点：轨迹偏离评估                                 │
        │  ┌─ Week 4: σ < 0.5 ✓ / 0.5-1.5 ⚠ / >1.5 🔴            │
        │  ├─ Week 8: 维度特异干预（微生物/屏障/免疫）              │
        │  └─ Week 12: STRIDE-IV 三层 + 分子缓解评估              │
        │  ↓                                                       │
        │  输出：完全/部分/无分子缓解的量化分类                      │
        └──────────────────────────────────────────────────────────┘
                              ↓
        ┌──────────────────────────────────────────────────────────┐
        │                 PHASE 3: MAINTENANCE                     │
        │            (T = Month 3-12)                             │
        │  MRD 驱动的递进式监测                                    │
        │  ├─ CMR: 3 月监测 (复发率 8%)                          │
        │  ├─ PMR: 1 月监测 (复发率 16-20%) → 干预                │
        │  └─ NMR: 周度监测 (复发率 30-35%) → 升级               │
        │  ↓                                                       │
        │  转变为"韧性评分"而非"绝对值"评估                        │
        └──────────────────────────────────────────────────────────┘
                              ↓
        ┌──────────────────────────────────────────────────────────┐
        │                 PHASE 4: LONG-TERM                       │
        │          (T > 12 months)                                │
        │  停药安全门槛评估                                        │
        │  ├─ 12 个月 CMR ✓                                      │
        │  ├─ 轨迹稳定性 > 95% ✓                                 │
        │  ├─ 轨迹韧性评分高 ✓                                   │
        │  └─ → 25-30% 患者成功停药                              │
        │  ↓                                                       │
        │  长期随访：年度深层 omics 评估                            │
        └──────────────────────────────────────────────────────────┘
```

### **设计元素**：
- **配色**：绿(缓解) → 黄(警告) → 红(干预)
- **时间轴**：横向 12 个月，标注关键里程碑
- **符号**：✓(达成) / ⚠(警告) / 🔴(干预)
- **文字**：Helvetica Neue, 12pt（打印清晰）
- **分辨率**：300 DPI，单页宽（185 mm）

### **配图建议**：
- 左侧：四个圆形堆叠（代表四阶段）
- 中央：螺旋上升曲线（代表患者病程与缓解）
- 右侧：关键指标的动态变化缩略图

## **Figure 2：轨迹医学的数学模型与多维空间**

### **图表类型**：3D 交互式曲线 + 2D 投影

### **核心设计**：
```
Panel A: 单个患者的 Logistic 轨迹

    CRP / Cal / Shannon
          ↑
        90│           ╱─────────────── L (目标缓解水平)
          │         ╱
          │       ╱  
          │     ╱    f(t) = L / (1 + e^{-k(t-t₀)})
          │   ╱    
          │ ╱      k = 缓解速率（个体化）
          │└──────────────────────→ t (周数)
          0  2  4  6  8  10 12
              t₀ = 预期最快缓解点

   例1："快速缓解者"
   - L = 15 (CRP), k = 0.8, t₀ = 3
   - 周 4 CRP 可降至 < 5 mg/L
   
   例2："延迟缓解者"  
   - L = 15 (CRP), k = 0.4, t₀ = 6
   - 周 4 CRP 仅降至 10 mg/L（轨迹偏离 1.2σ）→ 干预


Panel B: 多维空间中的轨迹偏离（Mahalanobis Distance）

    在 4D 空间中：(CRP, Shannon, Butyrate, CD4/CD8)
    
    ┌─────────────────────────────────────────┐
    │  ∘ 健康对照 (n=50)                      │
    │  • 患者期望轨迹 (Week 0-12)             │
    │  ★ 实际患者轨迹                        │
    │                                         │
    │        ┌─────────┐                     │
    │        │ ∘∘∘     │                     │
    │      ┌─┤ ∘∘∘∘∘  │                     │
    │    ┌─┤ │ ∘∘∘∘∘∘ │   Week 4            │
    │  ┌─┤ │ └─┤ ∘∘∘∘ │                     │
    │  │●│ │   └─────┤ ★ (偏离 0.8σ)        │
    │  ●●│ │         │                      │
    │  ●●●             Week 8                │
    │    ★ (偏离 0.3σ)                       │
    │        Week 12                         │
    │    ●●● (期望区间)                      │
    │      ★ (偏离 < 0.2σ)                   │
    └─────────────────────────────────────────┘


Panel C: 维度特异的偏离模式（Radar Plot）

    Week 4 Assessment - 维度特异化的干预触发
    
                     Microbiota
                    /      \
                   /        \
             Barrier        Immune
             (DAO)          (IL-6)
                |            |
           Metabolism   Clinical
            (Butyrate)  (Symptoms)
    
    患者 A (偏离来自微生物):
    ─ 微生物: 0.3σ (偏离) ← 触发: 益生菌或FMT
    ─ 代谢: 0.1σ (正常)
    ─ 屏障: 0.2σ (正常)  
    ─ 免疫: 0.15σ (正常)
    
    患者 B (偏离来自免疫):
    ─ 微生物: 0.1σ (正常)
    ─ 代谢: 0.15σ (正常)
    ─ 屏障: 0.2σ (轻微)
    ─ 免疫: 0.7σ (偏离) ← 触发: TDM + 剂量升级
```

### **设计元素**：
- **Panel A**：蓝色预期轨迹 vs 红色实际轨迹
- **Panel B**：灰色"期望云"（正常值范围）vs 绿→黄→红的偏离轨迹
- **Panel C**：雷达图的 5 个维度，过样子包括过头实例

### **技术要求**：
- 用 R/Python 的 plotly 或 matplotlib 制作交互版本
- 静态版本：300 DPI PDF，支持层次叠加显示

## **Figure 3：周期性决策树与临床干预映射**

### **图表类型**：分层流程图（Flowchart）

### **核心设计**：
```
                    ┌─────────────────────┐
                    │   Week 4 Assessment  │
                    │   轨迹偏离程度评估    │
                    └──────────┬───────────┘
                               │
                ┌──────────────┼──────────────┐
                │              │              │
         σ < 0.5          0.5 < σ < 1.5   σ > 1.5
         ────────         ─────────────    ───────
       ✓ Normal        ⚠ Mild Deviation  🔴 Alert
                │
     ┌──────────┴──────────┐
     │                     │
   Clinical           Molecular
   维度分析            维度分析
     │                     │
  ┌──┴──┐           ┌──────┴──────┐
  │     │           │             │
Microbiota      Microbiota   Barrier    Immune
Depletion       Fine         Damage     Activation
                Analysis
  │                │           │         │
┌─┴─┐         ┌────┴────┐  ┌──┴──┐   ┌─┴──┐
│   │         │         │  │     │   │    │
I1  I2        I3   I4   I5  I6   I7  I8  I9
│   │         │    │    │   │    │   │   │
v   v         v    v    v   v    v   v   v

I1: 继续当前治疗
    推荐：益生菌辅助
    监测：3 个月复查
    
I2: 升级至 JAK 抑制剂
    考虑：前期试验成功 + 免疫活跃
    监测：周度细胞因子
    
I3-I5: 益生菌 / FMT / 膳食优化
    根据基线菌群特征选择
    监测：周度 16S
    
I6-I7: 肠内营养强化 (EEN/MCFA)
    修复屏障：2-4 周
    监测：周度 DAO/ZO-1
    
I8-I9: 生物制剂升级或增加频率
    根据 TDM（谷值）调整
    监测：周度 IL-6 + 流式细胞术


【关键特征】
1. 每个干预分支都有：
   - 干预措施（明确的处方）
   - 监测指标（追踪响应）
   - 响应判断（何时升级）

2. 颜色编码：
   ✓ 绿色：维持当前
   ⚠ 黄色：轻度干预
   🔴 红色：立即升级

3. 右侧注释框：
   - 证据等级（基于 Kiel 数据或文献）
   - 预期效果（多少患者 4 周内改善）
   - 安全性注意事项
```

### **设计元素**：
- **布局**：从上至下的瀑布流（Waterfall）
- **连接线**：实线(强烈建议) vs 虚线(可选)
- **图标**：微生物(🦠) / 屏障(🛡) / 免疫(🔬) / 药物(💊)

### **可视化工具**：
- Lucidchart / OmniGraffle（专业医学图表）
- 或 Figma（现代设计）
