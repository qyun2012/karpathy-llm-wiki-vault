# 待完成研究项目汇总与进度追踪

---

## I. 执行摘要

当前共有 **7 个核心研究项目** 处于不同阶段：
- **4 个大型临床试验**：处于设计定版或伦理审批阶段，预期2026-2028年逐步启动
- **3 个综合研究计划**：优先级排序完成，第1优先项目（TDM+膳食干预RCT）已准备启动

这些项目涵盖：
- **临床转化**：精准医学、生物标志物驱动治疗、TDM-膳食协同
- **特殊人群**：老年IBD、妊娠期IBD、儿童CD
- **全球对接**：基于《IBD全球研究议程》（37个优先级）的本地化实施
- **政策导向**：中国本土老年IBD指南制定、基层IBD管理赋能

## II. 4 大临床试验详表

### Trial 1: AGED-IBD-BioGuide
**老年IBD患者生物标志物驱动治疗与功能维持 RCT**

| 维度 | 内容 |
|------|------|
| **试验名称** | Biomarker-guided treatment versus standard care for older adults with IBD: a randomized controlled trial on functional preservation |
| **缩写** | AGED-IBD-BioGuide |
| **阶段** | Phase III（非劣效性） |
| **设计** | 多中心、前瞻性、开放标签、分层随机化 |
| **样本量** | n=250-300（两臂） |
| **纳入标准** | ≥65岁，确诊IBD，基于综合老年评估框架（躯体-功能-心理三域） |
| **干预组** | 基于免疫衰老评分(CD4/CD8、IL-6、端粒) + 虚弱指数(Fried、VES-13)的个体化生物制剂选择 + TDM监测驱动的调整 |
| **对照组** | 标准治疗（医生经验驱动） |
| **主要终点** | 52周CDC缓解率（非劣效性，差异-10%）+ 功能维持(Fried评分不恶化比例) |
| **次要终点** | 临床缓解、内镜缓解、黏膜愈合、免疫衰老指标改善、安全性、生活质量 |
| **预期周期** | 24-30个月（招募12-15月 + 随访12月 + 分析） |
| **资助方** | ECCO、NIH、欧盟Horizon 2025（待定） |
| **预计成本** | $3.5-5.5M |
| **核心创新** | **首个整合免疫衰老评分 + 虚弱指数**的老年IBD RCT；功能维持作为主要评估目标；破除年龄排除标准 |
| **临床价值** | 填补当前0个老年IBD试验测量免疫衰老生物标志物的空缺；提供老年患者个体化生物制剂选择的证据 |
| **相关文献** | [[摘要-ibd-aging-clinical-trials-landscape]], [[摘要-elderly-ibd-trial-exclusion]], [[Vieujean-elderly-ibd-2022]] |

### Trial 2: BioNutr-IBD
**生物制剂与营养配方协同优化的多臂分层 RCT**

| 维度 | 内容 |
|------|------|
| **试验名称** | Optimization of biologic therapeutic response through targeted nutritional interventions: a randomized controlled trial of enteral nutrition formulations with different lipid compositions |
| **缩写** | BioNutr-IBD |
| **阶段** | Phase III（平行对照） |
| **设计** | 多中心、前瞻性、患者与营养师盲法、三臂分层 |
| **样本量** | n=300-400（三臂各100-133） |
| **分层因子** | 生物制剂类别(Anti-TNF vs其他)、基线白蛋白(<3.5 vs ≥3.5 g/dL)、诊断(CD vs UC) |
| **三臂设计** | ① **MCFA配方** + TDM监测驱动的生物制剂调整 vs ② **混合脂肪酸配方** + 标准护理 vs ③ **生物制剂单药** + 常规饮食 |
| **MCFA配方标准** | 蛋白质来源肽水解物、MCFA占脂肪≥60-70%、1500-2000 kcal/日（强化阶段）、无纤维 |
| **核心机制** | PLE(蛋白丢失) → 低白蛋白 → FcRn饱和 → 生物制剂清除加速；MCFA纠正PLE→白蛋白恢复→生物制剂谷值维持 |
| **主要终点** | 12周**内镜缓解率**(无溃疡) + **生物制剂谷值维持率** |
| **次要终点** | 临床缓解、白蛋白变化、粪便α1-抗胰蛋白酶(PLE指标)、菌群恢复(F.prausnitzii、Roseburia)、安全性、生活质量 |
| **延伸随访** | 26周、52周评估缓解维持与复发率 |
| **预期周期** | 18-24个月 |
| **资助方** | ECCO、营养品制造商（中立赞助）、公共卫生基金 |
| **预计成本** | $2.9-4.2M |
| **核心创新** | **首个前瞻性RCT验证脂肪酸组成与生物制剂协同**；量化PLE-白蛋白-FcRn-生物制剂清除的因果链；MCFA的标准化使用指南 |
| **临床价值** | 改善生物制剂失效患者预后；膳食干预可提升内镜缓解率30-50%（中国人群数据缺失） |
| **相关概念** | [[Protein_Losing_Enteropathy]], [[Hypoalbuminemia_and_FcRn]], [[MCFA_Clinical_Selection]], [[TDM_Guided_Dietary_Integration]] |

### Trial 4: MOVE-IT (NCT06788340)
**UST/VDZ MIPD 精准给药 RCT**

| 维度 | 内容 |
|------|------|
| **试验名称** | MOVE-IT: Multiomics-guided personalized dosing in IBD |
| **NCT号** | NCT06788340 |
| **阶段** | Phase III |
| **时间** | 2025-2027 |
| **地点** | 丹麦（Hvidovre Hospital/Copenhagen University） |

## III. 3 大综合研究计划详表

### Plan 1: IBD临床课题设计6大切入点
**从全球研究议程到中国本地化切入点** ([[synthesis-ibd-clinical-research-entry-points]])

**背景**：基于《IBD全球研究议程》(37个优先级陈述，Solitano et al., Nature Rev Gastroenterol Hepatol 2025)与《精准医学在IBD中的实现路径》(Schreiber et al., Gut 2025)

#### 优先级矩阵

| 优先级 | 研究方向 | 目标 | 预期周期 | 样本量 | 可行性 | 学术价值 |
|--------|---------|------|---------|--------|--------|---------|
| **第1** | **TDM + 膳食干预协同 RCT** | 验证膳食干预优化生物制剂药代动力学 | 18-24月 | n=300-400 | ⭐⭐⭐⭐⭐ | 国际证据空白，操作清晰 |
| **第2** | CDC多中心实现评估 | 综合疾病控制的标准化与临床转化 | 12-18月 | n=300-400 | ⭐⭐⭐⭐ | Grade A共识，快速形成网络 |
| **第3** | 早期诊断途径优化 | 诊疗延迟缩短，时间陷阱假说验证 | 12-18月 | n=800-1000 | ⭐⭐⭐⭐ | 临床实践结合紧密，政策导向 |
| **第4** | **妊娠期IBD前瞻性注册** | TDM监测、母胎转运、新生儿预后 | 18-24月 | n=150-200 | ⭐⭐⭐ | 临床迫切，患者需求高 |
| **第5** | **肠脑轴三角互动** | 心理-免疫-微生物的协同干预 | 24-36月 | n=400-500 | ⭐⭐⭐ | 创新性高，机制复杂 |
| **第6** | 社会经济梯度研究 | 地区与社会决定因素对预后的影响 | 24-36月 | n=3000-5000 | ⭐⭐ | 公共卫生价值，队列管理复杂 |

#### 第1优先项目详解：TDM + 膳食干预协同 RCT

**核心假说**：蛋白丢失 → 低白蛋白 → FcRn失效 → 生物制剂清除加速；膳食纠正PLE可逆转此链条

**研究设计**：四臂RCT
- 臂1：生物制剂单药（对照）
- 臂2：生物制剂 + 完全肠内营养EEN(8周)
- 臂3：生物制剂 + CDED(8周)
- 臂4：生物制剂 + 部分肠内营养PEN(8周)

**主要终点**：12周内镜缓解率 + 生物制剂谷值维持率 + ADA形成率

**随访**：12-52周，评估膳食获益的持久性

**中国数据缺口**：虽国际报道膳食干预可提升内镜缓解率30-50%，但中国人群数据仍缺失

### Plan 2: 老年IBD年度综述完整框架
**老年人炎症性肠病年度综述的框架大纲** ([[synthesis-comprehensive-annual-review-elderly-ibd-framework]])

**11个核心维度**：

1. **流行病学变化**：≥65岁患者占比至2030年达1/3；诊断延迟普遍；寿命短6-7年
2. **病理生理学**：免疫衰老、细胞衰老、菌群失调、炎症衰老的多层级机制
3. **诊断困难**：非特异性症状、"模拟者"疾病(缺血性结肠炎、憩室炎、NSAID肠炎等)的鉴别
4. **临床表型**：老年起病IBD多见结肠仅累及、温和表型，但预后因衰弱而恶化
5. **医学治疗**：生物制剂决策树(Vedolizumab首选)、药物安全性平衡、减药策略
6. **手术管理**：围术期策略、术式选择(IPAA vs造口)、并发症风险
7. **特殊管理**：衰弱与肌肉减少症、骨质流失、VTE预防、多重用药管理
8. **多学科协作**：MDT模式、老年病学家整合、远程监控、患者中心共享决策
9. **健康维护**：疫苗接种、癌症筛查、心血管风险评估、生活方式干预
10. **研究空白**：免疫衰老生物标志物缺乏RCT数据、脆弱性评分在治疗指导中的应用不足
11. **核心建议**：虚弱而非年龄驱动的患者选择、综合老年评估三步框架、功能维持为治疗目标

**用途**：
- 指导Nature Reviews级别的国际综述撰写
- 支持多学科临床实践指南制定
- 识别优先研究课题

### Plan 3: 中国本土老年IBD指南制定背景
**中国本土老年IBD指南制定的完整背景与框架** ([[synthesis-elderly-ibd-guidelines-china-background]])

**现状分析**：
- ✗ 中国**缺少官方老年IBD管理指南**（国际有AGA 2020、British Society 2025）
- ✗ 本地**流行病学登记数据不足**（当前仅有零散医院队列）
- ✗ **多学科协作规范缺失**（老年病学、营养、心理等科室协作模式不标准化）
- ✗ **基层赋能不足**（欠发达地区患者可及性差）

**指南制定框架**（8个核心模块）：

1. **流行病学与筛查**：危险因素识别、早期诊断、延迟诊断的成因分析
2. **诊断与鉴别**：内镜、影像、生物标志物组合诊断、老年特异性"模拟者"排除
3. **风险分层**：综合老年评估框架(躯体-功能-心理)、虚弱指数、预后评估
4. **治疗原则**：药物选择(5-ASA、糖皮质激素、免疫调节剂、生物制剂)的安全性平衡、生物制剂个体化选择
5. **营养支持**：营养评估、蛋白质缺乏纠正、膳食干预、肠内营养应用
6. **围手术期管理**：风险评估、术前优化、术式选择、术后康复
7. **多学科协作**：胃肠科、老年病学、营养、心理、药学、护理的协作流程
8. **监测与随访**：生物标志物检测间隔、复发预防、健康维护清单、患者教育

**预期产出**：
- 中文版老年IBD临床实践指南（与中华医学会消化分会合作）
- 多学科协作规范
- 基层培训课程
- 患者教育资料

**协作机构建议**：
- **牵头**：北京协和医院、华中科技大学同济医学院附属协和医院
- **支持**：中华医学会消化病学分会、中国炎症性肠病研究小组(CCRC)

## VII. 成功关键因素

### 必要条件
✅ **多中心网络建立**：Tier 1(3级医院IBD专科)×3-4家/国 + Tier 2(地区医学中心)×2-3家/地区  
✅ **标准化操作手册(SOP)**：统一评估工具、数据采集、质量控制标准  
✅ **独立数据安全委员会(DSMB)**：中期分析、安全性监测、停止准则  
✅ **患者友好设计**：减轻负担、高频随访的合理化、远程随访选项  
✅ **学术团队**：有经验的主要研究者、生物统计、项目管理  

### 风险与缓解

| 风险 | 缓解策略 |
|------|---------|
| 患者招募困难 | 制定详细纳入标准、多中心合作、患者宣传 |
| 高脱落率 | 高频接触、远程随访、研究补偿 |
| 配方不耐受 | 前期可耐受性评估、灵活给药方案 |
| 多中心协调困难 | 中央协调员、定期启动会议、实时沟通平台 |
