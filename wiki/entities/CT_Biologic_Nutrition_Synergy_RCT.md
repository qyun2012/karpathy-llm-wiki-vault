# 生物制剂与营养配方协同优化的多臂分层 RCT
## (BioNutr-IBD Trial: Biologic-Nutrition Synergy in IBD)

---

## I. 试验总体概览

### 基本信息
| 项目 | 内容 |
|------|------|
| **试验名称** | Optimization of biologic therapeutic response through targeted nutritional interventions: a randomized controlled trial of enteral nutrition formulations with different lipid compositions (BioNutr-IBD) |
| **首字母缩写** | BioNutr-IBD |
| **试验阶段** | Phase III |
| **设计** | 多中心、前瞻性、双盲（营养干预）、平行对照、分层 RCT |
| **臂数** | 3 臂 |
| **预期样本量** | n=300-400（各臂 100-133） |
| **主要终点周期** | 12 周 |
| **延伸随访** | 26 周、52 周 |
| **预计总周期** | 18-24 个月（招募 10-12 月 + 随访 12 月 + 分析） |
| **资助方** | ECCO、营养品制造商（中立赞助）、公共卫生基金 |

### 试验假说

**主假说**：中链脂肪酸（MCFA）为主导的肠内营养配方与生物制剂联合使用，通过纠正蛋白丢失性肠病→恢复白蛋白→FcRn 受体复原化，能显著提升生物制剂谷值维持率和临床缓解率，相比生物制剂单药或传统混合脂肪酸营养配方。

**次假说**：
1. **配方-生物制剂交互作用**：不同脂肪酸组成（MCFA vs 混合 vs 对照）对 Anti-TNF vs 其他生物制剂的效果不同
2. **蛋白丢失性肠病（PLE）→生物制剂失效的因果链**：通过白蛋白纠正可逆转生物制剂失败
3. **菌群恢复的时间特异性**：MCFA 配方加速益菌（F. prausnitzii、Roseburia）恢复，为持续缓解奠基

## II. 研究设计

### 2.1 入组标准

#### 纳入标准
1. **年龄**：18-75 岁
2. **诊断**：确诊 IBD（UC 或 CD），符合标准诊断标准
3. **疾病状态**：
   - **新诊断活跃期**（未使用过生物制剂）OR
   - **已启动生物制剂但谷值浓度低下**（<目标浓度的 50% 或 ADA 阳性）
4. **生物标志物证实的活跃期**：
   - CRP ≥5 mg/L 或 FCP ≥200 μg/g（或两者）
5. **启动或维持生物制剂治疗**：
   - 即将启动或已启动 <4 周的生物制剂（Anti-TNF、Vedolizumab、Ustekinumab）
6. **营养不良或白蛋白低下的风险**：
   - 血清白蛋白 <3.5 g/dL 或 BMI <20（低营养状态指标之一）
   - 或 FCP >500 μg/g（提示高蛋白丢失风险）
7. **能进行肠内营养**：
   - 无吞咽困难或严重胃排空障碍
   - GI 症状允许口服摄入或鼻胃管给药
8. **知情同意**

#### 排除标准
1. **活动性感染**：活动性结核、HIV、重症感染
2. **肠管残留过短**：<100 cm 肠段（严重吸收不良）
3. **激素依赖**：定义为无法在过去 3 个月内逐渐减少类固醇至 <7.5 mg/日
4. **严重合并症**：
   - 肝硬化（Child-Pugh ≥B）
   - eGFR <30 mL/min/1.73m²
   - 心梗、脑卒中过去 3 个月内
5. **营养支持需求过高**：肠外营养依赖患者
6. **食物过敏或营养配方不耐受史**：对配方中关键成分（如某种油脂）已知过敏
7. **其他研究**：同时参与其他营养干预试验
8. **预期依从性差**：无法完成 12 周营养干预和随访

---

### 2.2 分层与随机化

#### 分层因子（比例随机化）

1. **生物制剂类别**：
   - Anti-TNF（Infliximab、Adalimumab、Golimumab 等）
   - 其他机制（Vedolizumab、Ustekinumab 等）

2. **基线白蛋白水平**：
   - 低：<3.5 g/dL（高 PLE 风险）
   - 正常：≥3.5 g/dL

3. **诊断**：
   - CD（Crohn's Disease）
   - UC（Ulcerative Colitis）

**总分层组数**：2 × 2 × 2 = 8 组

#### 随机化方法
- **方式**：电话中心化随机化 + 计算机生成的随机序列
- **配比**：1:1:1（三臂等比例）
- **盲法**：患者与营养师盲法，医生知晓营养干预分配（但不参与评估）

#### 配方的编码与盲法
- 所有三种营养配方采用相同的包装、外观、味道（三盲设计）
- 患者、营养师、医学评估者都不知道配方类型
- 仅数据管理员知晓编码

---

### 2.3 干预方案（3 臂）

#### 干预臂 A：MCFA 主导配方 + TDM 监测驱动的生物制剂调整（n=100-133）

**配方特性**：
- **产品例子**（符合以下标准即可）：
  - 产品 1：Peptamen AF（Nestlé，70% MCFA）
  - 产品 2：Vivonex Plus（Nestlé，中链甘油三酯）
  - 产品 3：MCT-rich 中国本地产品

- **营养成分标准**：
  | 指标 | 目标 |
  |-----|------|
  | 蛋白质来源 | 肽水解物或氨基酸混合 |
  | MCFA 占脂肪% | ≥60-70% |
  | 脂肪中长链脂肪酸 | <40% |
  | 纤维 | 无或极少（活跃期患者） |
  | 热量密度 | 1.0-1.5 kcal/mL |
  | 渗透压 | 低渗（<350 mOsm/kg） |

- **给药方案**：
  - **强化阶段（0-8 周）**：
    - 目标 1500-2000 kcal/日（相当于全日能量摄入的 50-75%）
    - 通过口服或鼻胃管给予
    - 分次给予（3-4 次/日）或持续泵入（若鼻胃管）
  
  - **维持阶段（8-12 周）**：
    - 根据患者耐受和临床反应调整
    - 可逐步与常规食物混合
    - 目标维持 1000-1500 kcal/日 MCFA 配方

- **TDM 驱动的生物制剂调整**：
  
  | 时间点 | 检测项目 | 决策标准 |
  |--------|--------|--------|
  | **0 周** | 基线生物制剂谷值 | 确立个体目标浓度 |
  | **4 周** | 生物制剂谷值、白蛋白、FCP | 若谷值下降>20% → 缩短间隔（如 8→6 周）|
  | **8 周** | 重复以上，加内镜评分 | 若缓解但谷值低 → 增加剂量或短间隔 |
  | **12 周** | 完整评估 | 评估是否继续 MCFA 或可转至常规饮食 |

---

#### 干预臂 B：混合脂肪酸配方 + 标准护理（n=100-133）

**配方特性**（对照配方）：
- **产品例子**：
  - Ensure Plus（Abbott，混合油脂）
  - Nutrison Advanced（Nutricia，均衡脂肪）
  - 中国本地混合油脂营养液

- **营养成分标准**：
  | 指标 | 目标 |
  |-----|------|
  | 蛋白质来源 | 整蛋白（酪蛋白、乳清蛋白） |
  | 脂肪组成 | 长链 + 中链均衡（LCFA 50-60%, MCFA 40-50%） |
  | 纤维 | 中等量（3-8 g/升，活跃期患者可选择无纤维版本） |
  | 热量密度 | 1.0-1.5 kcal/mL |
  | 渗透压 | 等渗（300-350 mOsm/kg） |

- **给药方案**：
  - 同干预臂 A（目标热量相同）
  - 但无 TDM 驱动的主动调整
  - 医生可按常规判断调整（但不鼓励主动 TDM）

---

#### 对照臂 C：生物制剂单药 + 常规饮食（n=100-133）

**营养管理**：
- **无配方干预**：患者自由进食常规饮食
- **营养建议**：由营养师提供基本的 IBD 饮食指导，但无营养补充
- **生物制剂治疗**：标准给药方案
  - 医生可按常规判断调整（如 TDM 指导），但非强制

---

### 2.4 共同的生物制剂与支持治疗（三组相同）

#### 生物制剂方案
**两种情景**：

**情景 1：新诊断患者（无既往生物制剂）**
- 选择：医生根据指南推荐 + 患者偏好
- **推荐**（不强制）：Anti-TNF（Infliximab 5 mg/kg）
  - 诱导：0、2、6 周
  - 维持：8 周间隔
- **替代方案**（同样接受）：Vedolizumab、Ustekinumab

**情景 2：既往患者（因谷值低或 ADA 形成）**
- 维持既有生物制剂或升级至不同机制
- 医生+患者共同决策

#### 支持性治疗
1. **激素**（0-8 周，如需）：
   - 泼尼松龙 30-40 mg/日，逐周递减至停用
   - 或 布地奈德 9 mg/日

2. **对症治疗**：
   - 止泻（洛哌丁胺）
   - 解痉（美贝维林）
   - 适应症为主

3. **营养补充（全组）**：
   - 微量营养素评估（铁、B12、叶酸、钙、Vit D）
   - 补充缺乏

4. **预防**：
   - TB 筛查
   - 疫苗评估

## III. 评估指标与时间表

### 3.1 主要终点

**主终点**（12 周）：

1. **内镜缓解率**
   - **定义**：无溃疡（SES-CD <5 for CD 或 Mayo 内镜评分 ≤1 for UC）
   - **比较**：MCFA 配方 vs 混合配方 vs 生物制剂单药
   - **预期**：MCFA 65% vs 混合 55% vs 单药 45%（p<0.05）

2. **生物制剂谷值维持率**
   - **定义**：达成并维持目标谷值浓度的患者比例
   - **目标谷值**（根据生物制剂类型）：
     - Anti-TNF：3-5 μg/mL（或生物类似药标准）
     - Vedolizumab：≥15 μg/mL
     - Ustekinumab：≥3.6 μg/mL
   - **预期**：MCFA 70% vs 混合 50% vs 单药 35%

### 3.2 次要终点（12 周）

1. **临床缓解与反应**：
   - 临床缓解率（症状评分正常）
   - 临床反应率（≥50% 改善）
   - HBI 或 Mayo 评分绝对变化

2. **蛋白丢失与营养恢复**：
   - **白蛋白变化**（g/dL）：
     - 预期 MCFA: +0.5-0.8 vs 混合: +0.3-0.5 vs 单药: -0.1-0.2
   - **粪便 α1-抗胰蛋白酶**（PLE 的金标准指标）：
     - 预期：MCFA 组下降 >50%
   - **前白蛋白、转铁蛋白**（营养指标）
   - **血红蛋白、淋巴细胞计数**恢复

3. **生物标志物与病情活动**：
   - CRP 变化（mg/L）
   - FCP 变化（μg/g）
   - **综合疾病控制（CDC）缓解率**（临床+内镜+炎症标志物）

4. **菌群与代谢恢复**：
   - **粪便菌群多样性**（Shannon 指数）
   - **益生菌恢复**：
     - F. prausnitzii（佛氏法夫菌）丰度恢复
     - Roseburia 丰度恢复
   - **短链脂肪酸代谢**：粪便丁酸、丙酸浓度
   - **代谢产物**：粪便代谢组学特征变化

5. **安全性**：
   - 任何不良事件率
   - 胃肠耐受性问题（恶心、腹泻、腹胀）
   - 过敏反应
   - 生物制剂相关感染
   - 严重不良事件

6. **生活质量与患者报告结局（PROs）**：
   - IBDQ 评分
   - EQ-5D-5L
   - 患者全球评估（VAS）
   - 配方满意度评分

## IV. 统计学设计

### 4.1 样本量计算

**主终点：12 周内镜缓解率的优越性**

**假设**：
- 生物制剂单药（对照）内镜缓解率：45%
- MCFA 配方缓解率：65%（改善 20 pp）
- 混合配方缓解率：55%（改善 10 pp）
- 两侧 α = 0.05（Bonferroni 校正为 α = 0.025）
- 1-β = 0.80（功效 80%）
- 脱落率：10%（12 周相对短期）

**计算**：
```
两臂比较样本量（MCFA vs 单药）：
n = 2 × [(Z_α/2 + Z_β)² × p(1-p)] / δ²
其中 p = (65% + 45%) / 2 = 55%
     δ = 20% = 0.20
     Z_0.025 = 1.96, Z_0.20 = 0.84

n = 2 × [(1.96 + 0.84)² × 0.55 × 0.45] / 0.20²
  ≈ 2 × [7.84 × 0.2475] / 0.04
  ≈ 96 × 1.10（脱落调整）
  ≈ 105 → 圆整为每臂 100-133（总 n=300-400）
```

**功效分析**：
- 主比较（MCFA vs 单药）：足够功效 80%
- 次级比较（混合 vs 单药）：通过样本量获得 ~65% 功效

### 4.2 分析计划

#### 主分析集合
1. **ITT 集合**：所有随机化患者，按原分配意向分析
2. **Per-Protocol (PP) 集合**：至少接受 ≥80% 的配方干预的患者
3. **安全性集合**：接受 ≥1 次研究干预的患者

#### 主分析

**内镜缓解率的优越性检验**（12 周，ITT）：

```
比较 1：MCFA vs 生物制剂单药
如果 MCFA 缓解率 - 单药缓解率 > 某个临床相关差异（如 10 pp）
且 95% CI 下限 > 0 且 p < 0.025
    → 宣布 MCFA 优越

比较 2：混合配方 vs 生物制剂单药
（类似分析，α = 0.025）

比较 3：MCFA vs 混合配方
（如上述两个都显著，则进行探索性比较）
```

#### 生物标志物驱动的分析

**白蛋白纠正→生物制剂谷值改善的因果链**：
- **中介分析（Mediation Analysis）**：
  - 总效应：配方类型 → 内镜缓解
  - 直接效应：配方 → 缓解（未通过白蛋白）
  - 间接效应：配方 → 白蛋白↑ → 生物制剂谷值↑ → 缓解
  - 使用 Sobel 检验或 bootstrap 法估计间接效应比例

**因果图**：
```
┌─────────────────┐
│   MCFA 配方     │
│   vs 混合配方    │
└────────┬────────┘
         ↓
    ┌────────────────┐
    │ 白蛋白 ↑      │ (中介变量)
    │ PLE ↓         │
    └────────┬───────┘
             ↓
    ┌────────────────┐
    │生物制剂谷值 ↑  │
    │ ADA 形成 ↓    │
    └────────┬───────┘
             ↓
    ┌────────────────┐
    │  内镜缓解 ↑    │ (终点)
    └────────────────┘
```

#### 亚组分析（预先规定）

| 亚组变量 | 分层值 | 比较 |
|---------|--------|------|
| **生物制剂类型** | Anti-TNF vs 其他 | 配方效果的交互作用 |
| **基线白蛋白** | <3.5 vs ≥3.5 g/dL | 低营养患者是否获益更大 |
| **诊断** | CD vs UC | 配方效果的疾病特异性 |
| **基线 FCP** | <500 vs ≥500 μg/g | PLE 程度的影响 |

**交互作用检验**：使用对数回归模型检验配方×亚组变量的交互项

#### 时间分析

- **内镜缓解达成时间**：Kaplan-Meier 曲线，log-rank 检验
- **复发时间**（26-52 周）：Cox 比例风险模型

#### 菌群与代谢特征分析

- **α-多样性**（Shannon、Simpson 指数）的组间比较：ANOVA
- **β-多样性**（Bray-Curtis 距离）：PERMANOVA 检验
- **关键菌属丰度变化**（F. prausnitzii, Roseburia）：广义线性模型
- **代谢组学特征**：主成分分析（PCA）分组，偏最小二乘判别分析（PLS-DA）模型

## V. 伦理与监管

### 5.1 伦理考量

1. **配方安全性**：
   - MCFA 配方已广泛用于 IBD，但本试验是首次前瞻性 RCT 验证与生物制剂的协同
   - 充分知情同意强调营养干预的研究性质

2. **患者保护**：
   - 独立 DSMB 进行中期分析（n=150 完成 12 周）
   - **停止准则**：
     - **有效性**：若中期分析显示 MCFA 组明显优越（p < 0.001），可考虑提前停止
     - **安全性**：若严重不良事件率 >5% 任何臂，暂停招募

3. **退出与保护**：
   - 患者随时可退出
   - 医生可因疾病恶化移出患者并给予救援治疗
   - 所有患者获得生物制剂和基本营养支持

### 5.2 监管申请

- **IND/CTA**：向 FDA、EMA 等申请
- **IRB/伦理委员会**：多中心审查
- **营养产品监管**：确认营养配方符合 CE 标志或 FDA 营养补充品规范
- **试验登记**：ClinicalTrials.gov、WHO ICTRP

## VII. 关键创新点与科学价值

1. **首个前瞻性 RCT 验证脂肪酸组成与生物制剂协同**
2. **量化 PLE-白蛋白-FcRn-生物制剂清除的因果链**
3. **MCFA 配方的标准化使用指南**（如成功证实）
4. **菌群恢复的时间特异性**（MCFA 诱导益菌恢复的速度量化）
5. **TDM 与膳食干预的整合框架**（未来临床实践的基础）

## VIII. 预期成果与临床转化

### 成果形式
1. **主论文**（内镜缓解率、白蛋白恢复、TDM）：Nature/Lancet/Gastroenterology 等
2. **机制论文**（菌群、代谢组学、因果分析）：Gut、GUT Microbes
3. **临床实践指南**（与 ECCO/AGA 协作）：MCFA 配方的最佳实践推荐
4. **患者教育资源**：营养干预与生物制剂协同的科普

### 临床转化
- ✓ 非专科医疗机构可实施（营养配方相对廉价、易获取）
- ✓ 改善生物制剂失效患者的预后
- ✓ 减少手术与住院需求
- ✓ 改善患者营养状态，尤其是老年患者
