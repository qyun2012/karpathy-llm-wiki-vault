# 老年IBD患者生物标志物驱动治疗与功能维持RCT
## (AGED-IBD-BioGuide Trial)

---

## I. 试验总体概览

### 基本信息
| 项目 | 内容 |
|------|------|
| **试验名称** | Biomarker-guided treatment versus standard care for older adults with IBD: a randomized controlled trial on functional preservation (AGED-IBD-BioGuide) |
| **首字母缩写** | AGED-IBD-BioGuide |
| **试验阶段** | Phase III |
| **设计** | 多中心、前瞻性、开放标签、非劣效性 RCT |
| **预期样本量** | n=250-300（两臂各125-150） |
| **主要终点周期** | 52 周 |
| **预计总周期** | 24-30 个月（包含招募12-15月 + 随访12月 + 分析） |
| **资助方** | ECCO、NIH、欧盟 Horizon 2025（待定） |

### 试验假说
**主假说**：基于免疫衰老生物标志物（CD4/CD8 比值、IL-6、端粒长度）和虚弱指数（Fried、VES-13）指导的个体化生物制剂选择，在保持非劣效临床缓解的前提下，能显著改善老年 IBD 患者的**功能维持**（日常生活活动能力、生活质量、安全性）

**次假说**：与年龄相比，"生物学衰老评分"（综合免疫衰老 + 虚弱指数）是更好的生物制剂选择的预测因子

## II. 研究设计

### 2.1 入组标准

#### 纳入标准
1. **年龄**：≥65 岁
2. **诊断**：确诊 IBD（UC 或 CD），符合标准诊断标准
3. **疾病状态**：
   - 新诊断或活跃期（CRP ≥5 mg/L 或 FCP ≥200 μg/g）
   - 或既往缓解但当前复发（活跃期重新定义为上述生物标志物）
4. **治疗状态**：
   - 新诊断患者：未使用过生物制剂
   - 既往患者：距离上一个生物制剂超过 12 个月，或因失效/不耐受停药
5. **综合老年评估**（三步框架，全部通过）：
   
   **躯体域（Somatic）**：
   - Mini Nutritional Assessment-SF (MNA-SF) ≥11（营养状态可接受）
   - Charlson Comorbidity Index ≤5（中等合并症）
   - 非 IBD 处方药 ≤7 种（多重用药控制）
   
   **功能域（Functional）**：
   - Katz ADL（日常生活活动）≥4/6（至少中度独立能力）
   - 握力（握力计）：男性 ≥27 kg，女性 ≥16 kg（排除严重肌肉减少）
   - 4 米步行速度 ≥0.8 m/s（基本步态功能）
   
   **精神域（Mental）**：
   - Six-Item Cognitive Impairment Test (6-CIT) ≤8 分（认知功能可接受）

6. **知情同意**：患者能理解研究目的和程序，自愿签署知情同意书

#### 排除标准
1. **活动性感染**：活动性结核、HIV、重症细菌感染
2. **恶性肿瘤**：过往 5 年内有恶性肿瘤史，或活动性恶性肿瘤
3. **肠切除**：过往手术切除 >50 cm 肠段（严重营养不良风险）
4. **预期寿命**：<2 年（医学判断）
5. **心血管不稳定**：过去 3 个月内心梗、脑卒中、不稳定心绞痛
6. **肝肾功能衰竭**：
   - 肝硬化（Child-Pugh ≥B）
   - eGFR <30 mL/min/1.73m²（肾衰竭）
7. **无法随访**：预期无法完成 52 周随访（如计划移居）
8. **其他研究**：同时参与其他干预性试验

---

### 2.2 分层与随机化

#### 分层因子（1:1 比例分层）
1. **诊断**：UC vs CD
2. **年龄段**：65-74 岁 vs ≥75 岁
3. **虚弱指数**：
   - 非虚弱（Fried <3 项）
   - 虚弱前期（3 项）
   - 虚弱（>3 项）

**总分层组数**：2 × 2 × 3 = 12 组

#### 随机化方法
- **方式**：电话中心化随机化 + 计算机生成的随机序列
- **配比**：1:1（生物标志物驱动组 : 标准治疗组）
- **盲法**：开放标签（患者和医护知晓分配）

---

### 2.3 干预方案

#### 干预组 A：生物标志物驱动个体化治疗（n=125-150）

**第一步：基线免疫衰老评估**（第 0 周）

评估指标（采血和检查）：
- **免疫衰老标志物**：
  - CD4 和 CD8 绝对计数 + 比值
  - IL-6（血清，pg/mL）
  - TNF-α（血清，pg/mL）
  - 白细胞端粒长度（Flow-FISH，三个患者分组）
- **虚弱指数**：
  - Fried Frailty Index（5 项）
  - VES-13（12 项）
- **营养与代谢**：
  - 血清白蛋白、前白蛋白
  - 肌酐、eGFR
  - 血红蛋白、淋巴细胞计数

**第二步：生物制剂选择算法**

```
┌─ 生物标志物评分 ─┐
│                   │
├─ CD4/CD8 比值     │  评分系统
│  ≤1.0: 高衰老(3分) │  
│  1.0-1.5: 中(2分) │  免疫衰老评分：0-10分
│  >1.5: 低衰老(1分) │  虚弱评分：0-5分
│                   │  → 合并评分：0-15分
├─ IL-6 水平       │     
│  >7 pg/mL: 高(3分)│  
│  3-7: 中(2分)     │  
│  <3: 低(1分)      │  
│                   │
├─ 虚弱指数(Fried)  │
│  ≥3项: 虚弱(3分)  │
│  1-2项: 前期(2分) │
│  0项: 非虚弱(1分) │
└─────────────────┘
         ↓
    合并评分计算
         ↓
   生物制剂推荐决策
```

**生物制剂选择决策树**：

| 免疫衰老评分 | 虚弱状态 | 推荐生物制剂 | 理由 |
|-----------|--------|-----------|------|
| 高 (≥5) | 虚弱或前期 | **Vedolizumab** | 肠道选择性，全身感染风险最低 |
| 高 | 非虚弱 | Vedolizumab 或 Anti-TNF | 医生+患者共同决策 |
| 中 (3-4) | 任何 | Anti-TNF（多优先）或其他 | 强效诱导，可控风险 |
| 低 (<3) | 任何 | Anti-TNF、IL-23抑制剂、JAK | 可选强效生物制剂 |

**生物制剂具体选项**：
1. **Vedolizumab 4 mg/kg IV**：0、2、6 周诱导，8 周间隔维持
2. **Infliximab 5 mg/kg IV**：0、2、6 周诱导，8 周间隔维持（或 TNF 拮抗剂生物类似药）
3. **Ustekinumab**（IL-12/23 抑制剂）：根据体重诱导，8 周间隔维持
4. **JAK 抑制剂**（Tofacitinib 等）：仅用于≥75 岁血栓风险低患者

**第三步：免疫调节剂的个体化选择**

在生物制剂基础上联合免疫调节剂（医生+患者共同决策）：
- **硫唑嘌呤 (AZA)**：1.0-1.5 mg/kg/日（如肾功能正常，eGFR >60）
- **甲氨蝶呤 (MTX)**：仅用于 ≥70 岁且认知功能完整者
- **单药生物制剂**：如患者拒绝或有禁忌

**维持期监测与调整**（TDM 驱动）：

| 时间点 | 评估项目 | 调整标准 |
|--------|--------|--------|
| 4 周 | 临床反应、生物标志物、谷值 | 若无反应，考虑增加剂量 |
| 8 周 | 内镜评分、FCP、生物标志物 | 若缓解，继续；若部分缓解，短间隔 |
| 12 周 | CDC 评估、功能评估 | 若缓解不佳，考虑换药 |
| 26、52 周 | 全面评估 | 根据反应维持或调整 |

---

#### 对照组 B：标准治疗（Standard Care）（n=125-150）

**治疗原则**：遵循 ECCO/AGA 指南标准，由临床医生根据经验和患者偏好选择生物制剂

**推荐流程**（非强制）：
1. 初始评估：临床、内镜、生物标志物
2. 生物制剂选择：医生根据传统判断（年龄、合并症、症状严重程度）
3. 维持期：8 周间隔给药，按需调整
4. 监测：FCP、CRP、症状

**对照组允许的灵活性**：
- 可选择任何 FDA/EMA 批准的生物制剂
- 可自由选择是否联合免疫调节剂
- 可按医生判断改变给药间隔或剂量

---

### 2.4 共同的支持性治疗（两组相同）

1. **急性诱导期**（0-8 周，如需要）：
   - 糖皮质激素短疗程（泼尼松龙或布地奈德 30-40 mg/日，逐周递减）
   - 症状控制药物（止泻、解痉）

2. **营养支持**：
   - 所有患者接受营养师评估和建议
   - 根据 MNA-SF 评分，推荐补充营养（蛋白质、微量营养素）

3. **预防性措施**：
   - 结核筛查 + 必要时预防性化疗
   - 疫苗接种评估（流感、肺炎球菌、COVID-19）
   - 骨质流失预防（钙、维D、必要时双膦酸盐）
   - VTE 预防（高危患者）

## III. 评估指标与时间表

### 3.1 主要终点

**主终点**（52 周）：

1. **非劣效性终点**：
   - **综合疾病控制 (CDC) 缓解率** ≥ 对照组缓解率 - 10%（非劣效性边界）
   - **定义**：同时满足以下 5 项
     - 临床：症状评分正常（HBI <5 for CD，Mayo <2 for UC）
     - 内镜：内镜缓解（SES-CD <5 或 Mayo 内镜评分 ≤1）
     - 生物标志物：CRP ≤5 mg/L 且 FCP <200 μg/g
     - 患者报告：症状缓解评分 ≥50% 改善
     - 无手术、无住院

2. **优越性终点**：
   - **功能维持**（Fried Frailty Index 或虚弱恶化比例）
     - 定义：不恶化或改善（虚弱项数不增加）的患者比例
     - 预期生物标志物驱动组 70% vs 对照组 55%

### 3.2 次要终点（52 周）

1. **临床结局**：
   - 临床缓解率（HBI <5 或 Mayo <2）
   - 内镜缓解率（无溃疡）
   - 黏膜愈合率（完全组织学缓解）
   - 复发次数和时间

2. **功能与生活质量**：
   - IBDQ 评分变化（IBD-specific quality of life）
   - EQ-5D-5L（通用生活质量）
   - Katz ADL 评分变化
   - Lawton IADL（工具性日常生活活动）评分
   - 握力、步行速度变化
   - SPPB（Short Physical Performance Battery）评分

3. **免疫衰老逆转**：
   - CD4/CD8 比值变化
   - IL-6 水平变化
   - 白细胞端粒长度变化
   - 衰老相关 T 细胞（CD8+CD28-CD27-）比例

4. **生物制剂药代动力学**：
   - 生物制剂谷值浓度（通过 LC-MS/MS）
   - 抗药物抗体（ADA）形成率
   - 治疗药物监测（TDM）指导的给药调整频率

5. **安全性**：
   - 任何不良事件发生率
   - **严重感染**发生率（需住院治疗的感染）
   - 机会性感染（CMV、TB 重新激活等）
   - 恶性肿瘤新发病例
   - 心血管事件（MI、脑卒中）
   - 治疗停止率（因不耐受）

6. **经济学**：
   - 医疗资源使用（住院日、门诊次数、影像学检查）
   - 生物制剂年均成本
   - 成本-效果比（每个 QALY 的成本）

### 3.3 探索性终点

1. **生物标志物的预测价值**：
   - 基线免疫衰老评分对 CDC 缓解的 AUROC（曲线下面积）
   - 虚弱指数对安全性结局的预测能力

2. **微生物组与营养代谢**：
   - 粪便菌群 α/β 多样性
   - 益生菌丰度（F. prausnitzii、Roseburia）
   - 短链脂肪酸（丁酸、丙酸）代谢能力

3. **患者报告结局 (PROs)**：
   - 工作/就学缺勤天数
   - 患者满意度（Visual Analog Scale）
   - 医学专业人士沟通质量评分

## IV. 统计学设计

### 4.1 样本量计算

**主终点：CDC 缓解率的非劣效性**

**假设**：
- 对照组 CDC 缓解率：65%
- 生物标志物驱动组非劣效性边界：-10%（绝对差异）
- 非劣效性单侧 α = 0.025
- 1-β = 0.80（功效 80%）
- 预期脱落率：15%

**计算**：
```
样本量 = 2 × [(Z_α + Z_β)² × p(1-p)] / δ²
其中 p = (65% + 55%) / 2 = 60%
     δ = 10% = 0.10
     Z_0.025 = 1.96, Z_0.20 = 0.84

n = 2 × [(1.96 + 0.84)² × 0.6 × 0.4] / 0.10²
  ≈ 2 × [7.84 × 0.24] / 0.01
  ≈ 375 × 1.15（脱落调整）
  ≈ 431 → 圆整为 n = 125-150/臂 (总 250-300)
```

**功效分析**：
- 主分析（intent-to-treat, ITT）：n = 250（两臂各 125）
- 敏感性分析（per-protocol）：预期 n = 210（15% 脱落）

### 4.2 分析计划

#### 主分析集合
1. **ITT 集合**（Intent-to-Treat）：所有随机化患者，按原分配意向分析
2. **Per-Protocol (PP) 集合**：至少接受 1 次研究药物且遵循方案 ≥80% 患者
3. **安全性集合**：接受 ≥1 剂研究药物患者

#### 非劣效性分析

**主分析**：
```
如果（生物标志物驱动组 CDC 缓解率 - 对照组 CDC 缓解率）的 95% CI 下限 > -10%
    → 宣布生物标志物驱动治疗非劣
否则
    → 未达到非劣效性
```

**ITT 分析中的缺失数据处理**：
- 方法：多重插补法（multiple imputation, n=20）
- 假设：缺失完全随机 (MCAR)
- 敏感性分析：采用"悲观" vs "乐观"情景（脱落者视为失败 vs 成功）

#### 次要终点分析
- **优越性检验**：若非劣效性达成，检验优越性（功能维持、安全性）
- **亚组分析**：按诊断、年龄段、虚弱状态进行预先规定的亚组分析
- **交互作用检验**：生物标志物驱动与年龄/诊断的交互

#### 时间生存分析
- **复发时间**（Time to first relapse）：Kaplan-Meier 曲线，log-rank 检验
- **缓解维持曲线**：按随访时间分层

#### 生物标志物的预测价值
- **ROC 曲线分析**：基线免疫衰老评分预测 CDC 缓解的 AUROC
- **受试者操作特性曲线**：确定最优截断值

## V. 伦理与监管

### 5.1 伦理考量

1. **知情同意**：
   - 特别说明老年患者可能的风险（感染、免疫反应）
   - 强调生物标志物驱动的个体化治疗仍在研究中
   - 对照组患者知道可获得标准最佳治疗

2. **患者保护**：
   - 独立数据安全监测委员会（DSMB）进行中期分析（n=100 患者完成 12 周）
   - 设定停止准则：
     - **有效性准则**：若中期分析显示压倒性优越性（p < 0.001），可考虑提前停止
     - **安全性准则**：若生物标志物驱动组严重感染率 > 对照组的 2 倍，暂停招募

3. **退出与保护**：
   - 患者随时可退出（无需说明理由）
   - 医生可因临床指示（疾病恶化）移出患者
   - 所有患者获得研究期间的治疗和必要的监测

### 5.2 监管申请

- **IND/CTA 申请**：向 FDA、EMA、 国家药监部门
- **伦理委员会审查**：多中心 IRB 批准
- **数据保护**：遵循 GDPR（欧洲）、HIPAA（美国）、相关国家法规
- **试验登记**：ClinicalTrials.gov、WHO ICTRP

## VI. 可行性与实施计划

### 6.1 多中心网络

**建议参与机构**（目标 15-20 个中心）：

**Tier 1（3 级医院，IBD 专科，各国 2-3 家）**：
- 欧洲：Kiel University（德国）、Mayo Clinic Rochester（美国）、St Mark's Hospital（英国）
- 亚太：National University Hospital Singapore、Seoul National University Hospital
- 其他地区：各大洲主要 IBD 中心

**Tier 2（地区性医学中心，各国 2-3 家）**：
- 具有老年医学科和胃肠科的三级医院
- 参与国家级临床试验网络

### 6.2 启动与进度

| 阶段 | 时间 | 关键里程 |
|------|------|--------|
| **开发** | 6 个月 | 方案定版、SOP 编制、CRF 开发 |
| **伦理与监管** | 3-6 个月 | IND 申请获批 |
| **启动会议** | 1 个月 | 研究人员培训、系统测试 |
| **患者招募** | 12-15 个月 | 目标 n=250-300 |
| **随访** | 12 个月 | 最后入组患者完成 52 周 |

## VII. 关键创新点
