## 研究设计

### 对照组与干预组

#### 干预组 (MIPD臂)
- 每4周使用**PK模型** + **PK-PD靶点**自动生成给药建议(剂量与间隔)
- 医生可接受或拒绝建议
- 血液和粪便样本每4周收集用于PK/PD估计

#### 对照组 (医生判断臂)
- 基于临床参数(CRP、FCP、疾病活动评分)的常规给药决策
- **盲至**试验结束(不知道药物浓度)
- 相同的采样计划便于汇总分析

### 采样与测量

| 时间点 | 采集内容 | 用途 |
|--------|---------|------|
| **基线** | 血液+粪便 | PK初始化 |
| **每4周** | 血液+粪便 | PK/PD更新 |
| **第48周** | 结肠镜检查 | 黏膜愈合评估 |

## 纳入与排除标准

### ✅ 纳入标准

1. **疾病诊断**：确诊UC或CD
2. **治疗稳定性**：当前在VDZ或UST治疗上稳定 ≥3个月
3. **缓解状态**：
   - Calprotectin ≤200 µg/g（粪便）
   - PRO2评分在缓解范围内
4. **年龄**：≥18岁

### ❌ 排除标准

- 不确定性结肠炎
- 有造口
- 瘘管病为主要适应证
- 其他标准IBD试验排除标准

## 主要与次要终点

### 🎯 主要终点 (Primary Outcome)

**48周时无激素的临床和内镜缓解**

```
CD患者：PRO2 ≤4 AND Calprotectin ≤200 µg/g
UC患者：PRO2 = 0 AND Calprotectin ≤200 µg/g

同时不使用系统性激素
```

### 📊 次要终点 (Secondary Outcomes)

| 类别 | 指标 |
|------|------|
| **内镜** | 黏膜愈合(SES-CD <3 for CD / UCEIS ≤1 for UC) |
| **经济** | 成本效益(QALYs via EQ-5D-5L) |
| **临床** | 药物持续性、疾病复发(激素使用、手术) |
| **生活质量** | 疲劳(VAS-F)、工作生产力(WPAI) |
| **炎症** | CRP、血清白蛋白、FCP(每4周测) |

## 子协议与特殊研究

### 子研究 1：生物样本库

**地点**：Odense 中心  
**样本量**：n=20患者

**时间点**：
- 基线：结肠镜+活检
- 完成(48周)：结肠镜+活检

**用途**：
- 探索性生物标志物分析
- 黏膜免疫激活与clearance的关系
- 菌群分析(TBD)

### 子研究 2：肠超声

**样本量**：n=20患者

**时间点**：
- 基线
- 48周完成

**指标**：
- 小肠厚度
- 肠血流(彩色多普勒)
- 肠脂肪增厚(脂肪衣)

## 与您的研究的战略定位

### 📍 MOVE-IT 的空白与您的机会

| 维度 | MOVE-IT | 您的队列 | 补充优势 |
|-----|---------|---------|---------|
| **患者群** | 稳定维持 (Cal≤200) | 全谱，包括活跃/复发 | 外部有效性 |
| **地理** | 北欧(同质人群) | 亚洲(中国,异质) | 人口统计学多样性 |
| **方法论** | 确定性模型 | 贝叶斯实时适应 | 框架互补 |
| **输出** | 自动MIPD建议 | 临床决策工具 | 实践化程度 |

### 🎯 推荐定位

> **"MOVE-IT在北欧受控条件下验证了MIPD原理；我们的中国真实世界队列提供了RCT无法提供的外部有效性、人口统计学多样性和高病活性患者的有效性证据。两项研究的发表时间差异凸显了从受控试验到临床实践转化的差距。"**
