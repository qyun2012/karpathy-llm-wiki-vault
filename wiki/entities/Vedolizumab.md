## 定义与机制

**Vedolizumab** 是一种**肠道选择性单克隆抗体**，靶向α4β7整合素，选择性阻断T淋巴细胞向肠道迁移，从而实现**局部肠道免疫抑制**而最小化全身免疫功能抑制。

### 作用机制
1. **特异性靶向**：阻断α4β7整合素与粘膜血管寻址素细胞粘附分子-1（MAdCAM-1）的相互作用
2. **肠道选择**：主要作用于肠道淋巴细胞，不进入中枢神经系统或肺部
3. **功能结果**：减少肠道浸润的T淋巴细胞，降低肠道炎症，同时保留全身免疫功能

## 临床药学特性

| 特性 | 参数 |
|-----|------|
| **给药途径** | 静脉输注 |
| **给药方案** | 0/2/6周诱导；之后4周或8周一次维持 |
| **给药时间** | 30分钟输注 |
| **起效时间** | **4-8周**（相对较慢） |
| **半衰期** | ~25天 |
| **肾肝代谢** | 不经P450；肾功能不全无需调整 |
| **FDA批准** | 2014年（UC和CD） |

## IBD中的临床应用

### 溃疡性结肠炎（UC）
- **诱导缓解**（6周）：显著改善症状和内镜愈合
- **维持治疗**：4周制度优于8周制度在预防复发中
- **有效率**：约50-60%的患者达到临床缓解

### 克罗恩病（CD）
- **诱导缓解**：临床和内镜缓解率类似UC
- **维持治疗**：长期缓解率良好
- **小肠CD**：相对于其他生物制剂有优势（肠道特异性机制）

## 老年IBD患者中的特殊优势

### 循证数据

根据老年IBD患者的3个临床试验汇总（n=244）：

| 指标 | 数据 |
|-----|------|
| **临床缓解率** | 38-54%（3-12月维持稳定） |
| **重症感染** | 17%（显著低于抗TNF的11.6-21.1%） |
| **恶性肿瘤** | 0-2%（极低风险） |
| **安全性评分** | ✅ 最优（肠道选择性） |

### 老年患者的临床优势

1. **感染风险最低**
   - 肠道选择性 → 全身免疫抑制最小
   - 不增加机会性感染（TB、真菌）
   - 特别适合免疫衰老患者

2. **多病共存兼容性强**
   - 不进入中枢神经系统 → 认知安全
   - 不与其他药物产生重大相互作用
   - 肾肝功能不全无需调整

3. **恶性肿瘤风险最低**
   - 长期数据显示肿瘤风险不增加
   - 与硫唑嘌呤联合时恶性肿瘤风险不升高

4. **与脆弱性/肌肉减少症的相容性**
   - 不导致额外的肌肉分解
   - 不加重骨质流失（与皮质激素联合时安全）

## 与其他生物制剂的对比

### vs 抗TNF（Infliximab/Adalimumab）

| 维度 | Vedolizumab | 抗TNF |
|-----|-----------|------|
| **缓解率** | 38-54% | 50-79% |
| **起效速度** | 4-8周 | 2-4周 |
| **感染风险** | 17% (低) | 11.6-21.1% (高) |
| **全身免疫抑制** | 最小 | 显著 |
| **老年首选** | ✅ **是** | ⚠️ 谨慎 |

### vs Ustekinumab

| 维度 | Vedolizumab | Ustekinumab |
|-----|-----------|-----------|
| **缓解率** | 38-54% | 中-高 |
| **感染** | 17% | 中等 |
| **给药** | IV输注 | SC注射 |
| **老年推荐** | 首选 | 优秀替代 |

## 合并症与安全性（新证据2020）

### Charlson Comorbidity Index (CCI)与Vedolizumab安全性

根据荷兰ICC登记库前瞻性多中心队列研究（n=203例vedolizumab患者，Asscher et al., Aliment Pharmacol Ther 2020）：

**关键发现**：
- **合并症是感染风险的独立预测因子**
  - CCI与**任何感染**独立相关：OR 1.387 (95% CI 1.022-1.883, P=0.036)
  - CCI与**住院**独立相关：OR 1.586 (95% CI 1.127-2.231, P=0.008)
  
- **年龄非独立风险因素**：患者年龄本身与感染或住院风险**无独立关联**
  - 推翻了"年龄是风险因素"的常见假设
  - CCI是更准确的安全性预测工具

**临床意义**：
1. 应进行系统合并症评估（CCI评分）而不仅依赖年龄判断
2. 高CCI患者（≥2分）需要加强感染监测
3. 合并症管理与IBD治疗同等重要

### 治疗前评估

**强制评估**（新标准）：
- **Charlson Comorbidity Index (CCI)评分**（替代年龄为风险评估核心）
- 结核病筛查（标准）
- 肝功能检查（无需调整）
- 肾功能检查（无需调整）
- 完全血象（基线）
- 免疫衰老评分（推荐）

### 监测计划
- **诱导期（0-6周）**：每周临床评估，6周时评估反应
- **维持期**：4周至8周一次临床评估；每6-12个月粪便钙卫蛋白监测
- **年度**：完全血象、肝肾功能、血清白蛋白（营养监测）
- **感染监测**：教育患者识别感染症状；TB/真菌感染风险相对较低

### 联合治疗
- **可安全联合**：美沙拉明、糖皮质激素（短期）、硫唑嘌呤（老年患者通常避免）
- **避免联合**：其他生物制剂、JAK抑制剂
- **营养支持**：强烈建议维生素D、钙、蛋白质补充

## 老年患者中的治疗决策框架

### Vedolizumab是首选的场景
1. **高感染风险**：既往严重感染、免疫衰老、T细胞功能↓
2. **多病共存**：心肺疾病、神经系统疾病、肾功能不全
3. **脆弱性患者**：Fried脆弱指数中等-高度
4. **多药物治疗**：药物相互作用风险高

### 考虑抗TNF的场景（仅在Vedolizumab失效或禁忌时）
- 需要快速起效（<2周症状改善）
- 严格监测条件充分（住院或门诊密集随访）
- 感染风险分层后确定为低风险

### 转换策略
- 如果Vedolizumab部分应答 → 增加维持频率（4周制度）或添加5-ASA
- 如果Vedolizumab失效 → 转换到Ustekinumab或（谨慎）抗TNF+密切监测

## 安全性与不良反应

### 常见不良反应
- **输注反应**：1-3%，通常轻度；预处理可减少
- **头痛**：10%
- **关节痛**：5%
- **感染**：鼻窦炎、上呼吸道感染（老年患者风险不增加）

### 罕见但严重的不良反应
- **进行性多焦点脑白质病(PML)**：极罕见，与JC病毒相关（0.01%）；不使用其他免疫抑制剂时风险极低
- **脱髓鞘疾病**：罕见
- **心肌炎**：罕见

### 老年患者的特殊安全考虑
✅ **无需肾肝调整**  
✅ **不增加TB/真菌感染**（相比抗TNF）  
✅ **不增加恶性肿瘤风险**  
✅ **认知安全**（不进入CNS）  
⚠️ **输注反应管理**：老年患者可能对输注时间敏感，建议适当延长输注时间

---

## 推荐的一线使用策略

**结论**：Vedolizumab是老年IBD患者（特别是≥65岁、多病共存、脆弱性高）的**首选一线生物制剂**，理由：
1. 感染风险最低（肠道选择性）
2. 恶性肿瘤风险最低
3. 与多病共存和多药物治疗的兼容性最好
4. 长期安全数据充分

**给药方案推荐**：
- **诱导**：0/2/6周，IV 300mg
- **维持**：**4周间隔**（优于8周，特别是对老年患者有更好的缓解维持）
- **监测**：粪便钙卫蛋白、临床症状、营养指标
