## 核心摘要

省级自然科学基金资助的两项互补性生物医学研究项目，共同聚焦于 **肠道屏障修复的双向调控机制**——一个从"病理学破坏"与"生理学修复"两个角度解析IBD发病本质的创新研究体系：

1. **项目一** — [[粪杆菌LacCer干扰Notch屏障修复]]：揭示 **病理性微生物代谢产物** 如何阻滞肠上皮屏障恢复（破坏维度）
2. **项目二** — [[raw/02-papers/烟酸调控IBD治疗新机制研究]]：阐明 **营养分子** 如何激活多靶点信号通路促进屏障修复（修复维度）

## 项目对比框架

| 维度 | 项目一（LacCer-Notch） | 项目二（烟酸-多靶点） |
|------|----------------------|----------------------|
| **干扰因素** | 病理菌群代谢产物（LacCer）↑ | 营养缺乏（烟酸）↓ |
| **靶标信号** | Notch单一通路（上皮细胞分化） | GPR109A → AhR → Sirt1 多靶点 |
| **生物学效应** | 屏障破坏↑（紧密连接蛋白↓、分化阻滞） | 屏障修复↑（GLP-1↑、ILC3激活、菌群恢复） |
| **调控层级** | **微生物→上皮细胞** | **营养→免疫细胞+菌群+上皮细胞** |
| **时间动力学** | 急性干扰（可逆） | 慢性修复（长期维持需要） |
| **临床价值** | 诊断标志物（菌群失调识别） | 治疗策略（可及性强、成本效益高） |

## 协同与互补机制

```
IBD发病的二元平衡破坏模型：

正常稳态
├─ 健康菌群（含低LacCer、高SCFA产生菌）
├─ 充足烟酸供应（来自饮食+菌群）
└─ Notch信号 active → 屏障完整

病理状态（项目一的问题）
├─ 病理菌群过度增殖（LacCer高产粪杆菌株）
├─ LacCer堆积 → Notch信号 ↓
└─ 屏障破坏 ↑

修复策略（项目二的解决方案）
├─ 补充烟酸 → GPR109A激活
├─ 烟酸衍生物 → AhR激活 → 健康菌群优先化
├─ NAD+补充 → Sirt1激活 → 上皮细胞应激耐受 ↑
└─ 综合效果：LacCer↓（菌群改组）+ Notch信号恢复 + 屏障修复加速
```

## 预期研究协同效应

### 整合前景（相互支持的研究方向）

1. **基础研究的相互验证**
   - 项目一证实 LacCer 对 Notch 的干扰 → 项目二证实烟酸衍生物（喹喏酸）的解救效应
   - 建立"LacCer诱导屏障破坏，烟酸衍生物驱动修复"的完整信号传导链

2. **菌群工程的双向目标**
   - 减少 LacCer 生成菌（项目一目标）+ 增加烟酸代谢菌（项目二目标）= 益生菌菌株设计的黄金组合
   - 例：敲除 *F. prausnitzii* 中的 LacCer 生物合成基因 + 导入烟酸衍生物生成基因 → "万能益生菌"

3. **临床转化的阶段性应用**
   - 项目一的早期诊断（粪便 LacCer 浓度识别高危患者）
   - 项目二的分层治疗（识别后立即烟酸补充，加速屏障修复）
   - 联合指标：LacCer↓ + 烟酸↑ + Notch活性↑ = 治疗成功的三参数评估

4. **多组学轨迹追踪**（与[[OmicTrajectory]]联合）
   - **纵向采样**：同一患者的粪便微生物组、血清代谢产物、肠组织转录组
   - **关键时点**：基线 → 烟酸补充开始 → 2周、4周、8周、12周
   - **追踪指标**：
     - LacCer浓度（脂质质谱）
     - 烟酸与代谢物（NAD+, 喹喏酸）
     - Notch通路基因表达（肠活检原位混合荧光+单细胞测序）
     - 菌群动态（16S/宏基因组）
     - 屏障完整性（血清zonula occludens-1, LPS通透性)
   - **数据整合**：识别"快速响应者"与"缓慢响应者"的分子特征
