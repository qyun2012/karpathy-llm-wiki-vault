## 论文基本信息

**标题**：Is it time to include older adults in inflammatory bowel disease trials? A call for action

**发表**：The Lancet Healthy Longevity, 2022

**来源URL**：https://www.thelancet.com/journals/lanhl/article/PIIS2666-7568(22)00060-5/fulltext

**通讯作者**：Laurent Peyrin-Biroulet（法国南洛林大学）

**关键作者**：Sophie Vieujean, Bénédicte Caron, Vipul Jairath, Athanase Benetos, Silvio Danese, Edouard Louis

---

## 核心问题与研究方法

### 问题陈述
- IBD患者中，≥65岁人群现占总人口的1/3以上（20%成人发病型老龄化，15%老龄发病型）
- 老年IBD患者的治疗管理面临挑战：合并症、多重用药、营养不良、功能受限、长期治疗风险
- **但关键是**：缺乏老年患者证据，因其从临床试验中被系统性排除

### 研究设计
- **范围**：系统综述，纳入PubMed和ClinicalTrials.gov注册的第3期IBD试验（1995年6月-2021年8月）
- **纳入标准**：222项第3期试验
  - 其中127项已发表，95项仅注册
- **评估项目**：
  - 年龄上限排除标准
  - 65岁及以上患者的纳入百分比
  - 非年龄相关的排除标准（合并症、既往恶性肿瘤、既往吻合口）

---

## 主要发现

### 1. 年龄排除的普遍性
| 指标 | 数据 |
|------|------|
| 使用年龄排除标准的试验 | 129/222 (58%) |
| 报告≥65岁患者比例的试验 | 32/222 |
| 65岁及以上患者占比 | 763/14,124 (5.4%) |

**含义**：老年患者在IBD临床试验中的代表性严重不足，远低于其在真实IBD人群中的比例。

### 2. 非年龄相关的排除标准（复合因素）
| 排除原因 | 使用频率 | 涉及试验数 |
|---------|---------|----------|
| 合并症（肾/肝/心血管） | 76% | 169/222 |
| 既往吻合口或肠切除 | - | 多数试验 |
| 既往恶性肿瘤或dysplasia | 45% | 100/222 |
| 既往IBD治疗 | 19% | 42/222 |
| 认知障碍 | 5% | 12/222 |
| 功能受限 | <1% | 2/222 |

**含义**：年龄并非唯一排除老年患者的因素，多重排除标准的叠加进一步限制了老年患者的纳入。

### 3. "虚弱"（Frailty）作为更好的风险评估工具
论文强调，**虚弱比年龄更能预测不良结局**：
- 虚弱独立于年龄和合并症，与感染风险增加相关（aOR 2.05-1.81）
- 虚弱与死亡率关联（OR 2.90 [95% CI 2.29-3.68]，n=11,001例）
- 虚弱与住院风险增加相关（aHR 1.21-1.57）
- 虚弱是手术后并发症的更强预测因子（优于年龄）

## 论文提出的三步包纳框架

### 第一步：患者选择标准（基于综合老年评估）

#### 躯体域（Somatic Domain）
- **营养状态**：Mini Nutritional Assessment-SF (MNA-SF) >11
- **合并症负担**：Charlson Comorbidity Index 低值
- **多重用药**：非IBD处方药≤5种

#### 功能域（Functional Domain）
- **日常生活活动**：Katz Index ADL <1 或 Lawton IADL（性别调整）<1
- **物理容量**：握力强度（Jamar手动力计）、4米步行速度

#### 精神域（Mental Domain）
- **认知功能**：Six-Item Cognitive Impairment Test <8分

#### 其他标准
- 确诊IBD
- 无既往恶性肿瘤/dysplasia史
- 最新dysplasia筛查结肠镜正常
- 正常预期寿命

### 第二步：试验设计考虑
- 使用**患者报告结局（PRO）**和**生物标志物**验证
- 定义**适当的临床终点**
- 明确**临床反应评估的时间点**

### 第三步：试验执行特殊需求
- **学科构成**：与老年医学专家合作
- **护理人员**：聘请老年患者护理沟通专家
- **资源**：扩展研究团队或增加工作时间
- **便利性**：在患者家中进行部分咨询

## 核心观点与临床启示

### 关键论点
1. **老年患者具有代表性**：≥65岁IBD患者占全球IBD人群的1/3，应被纳入
2. **虚弱优于年龄**：应用综合老年评估识别"适合"的老年患者，而非根据年龄排除
3. **合理的包纳标准可行**：通过结构化评估（MNA, Charlson, Katz, Lawton），可以识别能够参与试验的老年患者
4. **改革对所有人有利**：包纳老年患者的试验设计改进对所有人都有益（简化操作、改善患者报告结局等）

### 对中国的启示
1. **流行病学空白**：建立中国老年IBD患者的自然史和治疗模式数据库
2. **临床实践指南**：开发中国老年IBD管理指南，基于证据而非任意年龄限制
3. **研究设计**：在未来中国IBD临床试验中，应采纳综合老年评估而非年龄排除标准
4. **队列建设**：建立老年IBD患者队列（生物库），支持年龄相关的亚组分析
