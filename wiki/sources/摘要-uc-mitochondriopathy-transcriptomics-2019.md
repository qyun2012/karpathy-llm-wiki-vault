## 核心摘要

PROTECT研究是迄今最大的未治疗儿科溃疡性结肠炎(UC)患者入院队列研究(n=206),通过高通量RNA-seq转录组学分析发现了UC的关键分子病理机制。**关键发现：**(1) 全部13个线粒体编码基因及线粒体生物发生调控因子(PGC-1α)显著下调，导致复杂体Ⅰ活性减低和线粒体膜电位下降，揭示UC患者存在"线粒体病变(mitochondriopathy)"；(2) 疾病严重程度与712个基因富集相关，特别是腺瘤/腺癌和先天免疫基因上调；(3) 发现115个基因的"糖皮质激素反应基因签名"，可预测周4临床缓解并与成人抗TNFα和抗α4β7整合素治疗反应相关；(4) 严重程度和治疗反应基因签名与肠道菌群失调相关,特别是有益菌群(乳酸菌科、双歧杆菌)的耗竭和病原菌(柏氏菌、韦荣球菌)的升高。多变量分析表明基因表达数据显著改进了仅基于临床因素的周4缓解预测模型(AUC从0.67升至0.77)。

## 关键数据与发现

### 1. 线粒体功能障碍(Mitochondriopathy)
- **转录表现**：全部13个线粒体编码基因(复杂体Ⅰ、Ⅲ、Ⅳ、V)在UC患者中均显著下调(FDR < 2.76E-35)
- **关键调控因子**：PGC-1α(线粒体生物发生的主调控因子)在PROTECT/RISK和成人UC队列中均显著下调
- **功能验证**：高分辨呼吸测量显示活跃期UC患者的复杂体Ⅰ活性显著降低；线粒体膜电位(MMP)在上皮细胞中特异性降低
- **特异性**：此线粒体基因下调在结肠型IBD患者中最为显著(UC > 仅结肠CD > 回肠-结肠CD),在感染性结肠炎或憩室炎中未观察到
- **临床意义**：线粒体功能缺陷可加重屏障功能障碍和炎症,是UC病理生理学的新靶点

### 2. 疾病严重程度基因签名
- **规模**：712个基因(292下调、420上调)与UC严重程度相关(FDR < 0.001, FC ≥ 1.5)
- **富集通路**：
  - 上调：腺瘤/腺癌相关基因(P < 7.62E-51)、先天免疫(P < 7.07E-19)、中性粒细胞脱颗粒(P < 1.51E-16)
  - 下调：脂质代谢、腺瘤/腺癌相关基因
- **免疫细胞推断**：重症患者中活化树突状细胞、浆细胞和单核细胞增加
- **历史血管变化关联**：187个基因与直肠黏膜表面绒毛样改变(Mayo ≥ 3)相关,预示更高的治疗升级需求

### 3. 糖皮质激素反应基因签名
- **规模**：115个基因(FDR < 0.05, FC ≥ 1.5)区分周4缓解者与非缓解者
- **预测能力**：
  - PC1值与周4临床缓解显著相关
  - 与粪便钙卫蛋白 < 250 mcg/gm(黏膜愈合)相关
  - 在独立验证队列(Lexogen平台)中重现
- **功能富集**：
  - 趋化因子(CXCR1/2、CXCL6/8/10/11/17) - P < 7.12E-12
  - 先天性髓样免疫 - P < 1.62E-15
  - 细菌反应 - P < 2.16E-13
- **与抗生物制剂反应的关联**：
  - Oncostatin M(OSM)和TREM1两个关键基因与抗TNFα缓解相关,在糖皮质激素签名中高度相关(r = 0.79-0.89, P < 0.0001)
  - 在成人抗TNFα(GSE16879)和抗α4β7整合素(GSE73661)队列中观察到类似的响应/非响应者区分
- **临床模型性能**：
  - 模型1(仅临床):AUC = 0.68
  - 模型2(临床+基因签名):AUC = 0.774(LR P = 0.002)
  - 模型3(用ALOX15基因替代组织学嗜酸性粒细胞计数):AUC = 0.777、敏感性62.7%、特异性76.6%

### 4. ALOX15基因的特殊角色
- **生物学功能**：表达在组织嗜酸性粒细胞和巨噬细胞中,催化花生四烯酸代谢产生脂质介质
- **临床关联**：高ALOX15表达与>32 个直肠嗜酸性粒细胞/HPF(高水平浸润)相关,预示更好的周12缓解预后
- **替代价值**：在多变量模型中用ALOX15基因表达替代组织学嗜酸性粒细胞计数不损害模型准确性,可能因基因更准确反映整体的抗炎调控状态

### 5. 微生物群落关联
- **样本量**：156名UC患者的16S rRNA测序数据
- **关键发现**：
  - **病原菌升高**：柏氏菌、韦荣球菌、肠球菌与严重程度和治疗失败基因相关
  - **有益菌耗竭**：梭菌纲(Clostridiales)的减少,与严重程度和不良治疗反应呈负相关
- **多维关联**：使用分层所有对所有关联(HAllA)方法,在所有样本中进行菌群OTU与基因/通路的关联分析(FDR < 0.2)
- **注意**：RISK和Lexogen验证队列中菌群数据不足,微生物关联结果需在独立队列中验证

## 研究设计亮点

1. **入院队列**：PROTECT研究是首次系统地在未治疗、新诊断的儿科UC患者中收集治疗前样本
2. **大规模转录组**：206名UC患者 + 20名对照的Illumina RNAseq(迄今最大规模UC转录组)
3. **多队列验证**：
   - 内部验证：PROTECT队列的50名独立患者
   - RISK队列验证(儿科IBD入院队列)
   - 成人UC微阵列数据(GSE59071)
   - 孤立上皮细胞队列
4. **功能验证**：高分辨呼吸测量、流式细胞术(JC-1线粒体膜电位)、免疫组织化学

## 临床启示

1. **线粒体功能障碍作为新靶点**：超出当前生物制剂(抗TNFα、α4β7整合素拮抗剂)的作用机制,提示通过医学、营养或环保手段补充线粒体电子转运链的潜力
2. **个性化预测与分层治疗**：基因签名可改进早期缓解预测,支持生物标志物驱动的治疗选择(特别对非缓解高风险患者)
3. **微生物-基因互作**：强调了菌群恢复作为辅助治疗的重要性,特别是乳酸菌科和梭菌纲的重建
4. **代偿性免疫激活与上皮转化**：即使在诊断时,严重程度基因已富集腺癌相关基因,提示长期UC病程中的癌变风险需要更长的随访

## 数据可用性

- **主要数据库**：GEO(Gene Expression Omnibus)
  - PROTECT:GSE109142(Illumina RNAseq, n=226样本)
  - RISK:GSE117993(Illumina RNAseq, n=150样本)
- **微生物数据**：NCBI SRA Bioproject PRJNA436359(16S rRNA)
