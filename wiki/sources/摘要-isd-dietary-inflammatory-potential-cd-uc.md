## 核心摘要

基于欧洲癌症与营养前瞻性调查队列（EPIC）的大规模研究（394,255人，中位随访13.6年），发现高炎症膳食评分（ISD）与克罗恩病（CD）风险显著增加相关（第4四分位 vs 第1四分位的调整风险比aHR: 1.88, 95% CI: 1.14–3.10），但与溃疡性结肠炎（UC）无显著关联（aHR: 0.85）。在女性中CD的关联更强（aHR: 2.14）。低纤维、低单不饱和脂肪酸（MUFA）、低维生素C、低镁和低洋葱摄入是主要驱动因素。

## 关键研究设计

- **队列**：EPIC-IBD (394,255人，8个欧洲国家)
- **随访期**：13.6年（4,889,910人年）
- **CD发病率**：184例（3.8/10万人年），UC 459例（9.4/10万人年）
- **方法**：膳食频率问卷基线评估，Cox比例风险模型分析
- **调整变量**：吸烟、BMI、身体活动、能量摄入、教育水平、酒精摄入

## 主要发现

### ISD与IBD风险的关联

| 疾病 | 四分位 | aHR (95% CI) | p-trend |
|------|--------|-------------|---------|
| **CD** | Q2 | 1.55 (0.97-2.49) | <0.01 |
| | Q3 | 1.86 (1.16-2.98) | |
| | Q4 | 1.88 (1.14-3.10) | |
| **UC** | Q2 | 1.15 (0.89-1.49) | 0.21 |
| | Q3 | 0.96 (0.73-1.27) | |
| | Q4 | 0.85 (0.63-1.15) | |

### 性别差异与CD风险
- 女性（n=67,150）：Q4 vs Q1, aHR 2.14 (1.17-3.91)，p-trend <0.01
- 男性（n=31,414）：Q4 vs Q1, aHR 1.41 (0.56-3.56)，p-trend 0.55
- **注**：女性占样本68.1%，交互作用检验不显著

### ISD分项元素与CD风险
单位增加时的显著关联（Q4 vs Q1）：
- **低纤维摄入**：aHR 1.85 (1.12-3.04)
- **低单不饱和脂肪酸（MUFA）**：aHR 1.95 (1.08-3.52)
- **低维生素C**：aHR 1.76 (1.09-2.85)
- **低镁**：aHR 1.67 (1.01-2.74)
- **低洋葱摄入**：aHR 1.72 (1.01-2.95)
- **低酒精摄入**（≤40g/day）：aHR 2.74 (1.05-7.14)

**与UC无显著关联**：所有ISD分项均与UC风险无统计学关联

## 炎症饮食评分（ISD）方法学

### ISD计算框架
- **基础**：修订版膳食炎症指数（DII）
- **食物参数**：45个食物参数中的28个（排除总脂肪以避免冗余）
- **标准化**：使用EPIC队列的均值和标准差（而非DII的全球数据集）
- **炎症效应权重**：基于与炎症生物标志物（IL-1β, IL-4, IL-6, IL-10, TNF-α, CRP）关联确定
- **百分位转换**：中心化百分位值（P转换为2P-1）
- **解释**：正值表示促炎潜能，负值表示抗炎潜能

### ISD评分范围
- 总体：−6.36 至 4.94
- Q1（最低炎症）：−1.92 ± 0.91
- Q4（最高炎症）：2.55 ± 0.58
- 一单位ISD增加 → CD风险增加12%（aHR 1.12, 95% CI 1.01-1.23）

## 敏感性分析与稳健性

✓ 排除前1、2、5年随访以排除反向因果关系：结果一致  
✓ 排除纤维从ISD计算：ISD效应大小不变  
✓ 使用三次样条曲线建模：结果一致  
✓ 性别特异性十分位数分析：Q10 vs Q1的CD aHR 2.14 (0.96-4.78)

## 与既往研究对比

### EDIP评分 vs ISD
- **EDIP研究1**（Lo et al., 美国3队列，166,903女性+41,931男性）：CD高EDIP四分位增加51%风险，UC无关联
- **EDIP研究2**（Narula et al., PURE队列7国28,428人）：无显著关联（数值趋势存在）
- **本研究（ISD）**：与EDIP一致，更突出营养元素（而非食物组）的作用

### 机制解释

#### 纤维的保护机制
- 短链脂肪酸（SCFA）生产与免疫调节
- 多种多糖、低聚糖、木质素、抗性淀粉的不同作用
- **关键发现**：纤维来源（蔬菜/水果/谷物）的差异化影响需进一步研究

#### MUFA的新发现
- 橄榄油和夏威夷坚果等植物油中丰富
- 克罗恩病排除饮食（CDED）和地中海饮食中包含无限制橄榄油
- MUFA与不饱和脂肪酸比例（n-3:n-6）的关键作用

#### 维生素C和镁的关联
- **新发现**：与更高水果和蔬菜摄入关联
- **对比**：Mendelian随机化研究显示遗传预测的镁（而非维生素C）与CD相关

#### 洋葱的特殊作用
- 含黄酮醇等生物活性化合物
- 可能通过微生物组调节和屏障功能改善

## CD vs UC的不同机制

**CD的饮食敏感性**：营养质量和超加工食品摄入相关
**UC的驱动因素**：油酸（linoleic acid）高摄入、红肉、低DHA（EPA/DHA不足）

→ **结论**：CD和UC的致病机制中饮食因素的作用途径存在根本差异

## 临床意义与展望

### 防控策略（高风险人群）
✓ 增加膳食纤维摄入（多来源：蔬菜、水果、全谷物）  
✓ 优化脂肪酸组成（增加MUFA和n-3 PUFA）  
✓ 强化维生素C和矿物质（镁）补充  
✓ 增加洋葱/蒜等含黄酮醇食物  
✓ 适度饮酒（仅在适度范围内）

### 未来研究方向
- 纤维类型与来源的个体化影响
- MUFA在特定碳水化合物饮食（CDED）和地中海饮食中的协同效应
- ISD评分在不同地理和文化背景的外部验证
- 微生物组-营养-免疫的纵向整合分析

## 研究局限

1. 饮食仅在基线测量一次（可能随时间变化）
2. 食物频率问卷 vs 24小时饮食回顾（可能低估变异性）
3. EPIC人口特征（68%女性、平均年龄52岁）可能不代表全欧洲或全球年轻人口
4. 观察性研究无法排除剩余混杂（年龄本身不作为混杂因子）
