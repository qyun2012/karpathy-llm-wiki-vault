# IBD术后营养-生物制剂协同优化的分阶段多臂自适应平台试验

## 英文标题
Multi-Arm Adaptive Platform Trial of Perioperative Nutritional Intervention Synergistic with Biologic Therapy in IBD Surgery: Two-Stage Assessment of Postoperative Complications and Long-Term Remission Maintenance

---

## 1. 研究概况

| 项目 | 内容 |
|------|------|
| **研究类型** | 前瞻性、随机对照、多中心、贝叶斯自适应设计 |
| **研究阶段** | 二阶段框架：I期(术后30天并发症)；II期(长期缓解维持，12个月) |
| **设计架构** | 多臂多阶段(MAMS) + 共享对照 + 动态臂分配 |

| **入组规模** | 130-140例(n=40对照固定; n=35-45免疫营养臂; n=35-45 C-EEN臂) |
| **研究中心** | 2家三甲医院(中国医科大学附属第一医院、中日友好医院) |
| **预期患者/年/中心** | 50-70例IBD腹部手术患者 |
| **伦理编号** | 待申报 |

## 2. 研究背景与流行病学证据

### 2.1 IBD围术期营养不良的临床负担

**患病人数与经济成本**：
- 全球IBD患者数：300-600万例(发达国家患病率0.5-1%)
- 中国IBD患者：50-100万例(仍处上升期)
- CD患者手术率：60-80% (平均3-5次手术/患者生涯)
- UC患者手术率：20-40% (全结肠切除或J-pouch)
- 围术期营养不良患病率：**45-72%**(取决于疾病活动度与手术类型)

**术后并发症与营养状态的关联**：
- 蛋白营养不良患者(白蛋白<3.0 g/dL)的术后感染率：**35-45%** vs 正常营养患者的8-15%
- 低白蛋白与愈合延迟、脓肿形成、肠瘘、再手术的独立相关(OR 2.1-4.5)
- 术后30天Clavien-Dindo ≥II级并发症率：**36-42%**(基线数据) vs 目标**≤22%**(干预后)

### 2.2 生物制剂清除机制与围术期失效风险

**术后生物制剂清除加速的三大驱动因素**：

#### 2.2.1 靶点介导的药物处置(TMDD)
- **高效炎症负荷**：手术创伤导致TNF-α、IL-23等炎症细胞因子快速升高(术后1-3天峰值)
- **机制**：抗TNF制剂(IFX、ADA)被靶点消耗，导致非线性清除
- **临床后果**：IFX谷值<3 μg/mL患者的清除率为0.288 L/day，而谷值>7 μg/mL患者仅为0.15 L/day(~2倍差异)
- **手术特异影响**：大型腹部手术后48-72小时，IFX清除率可上升30-50%

#### 2.2.2 免疫原性与抗药物抗体(ADAs)
- **低谷浓度的悖论**：谷值<3.5 μg/mL → ADA快速产生(1-3个月内) → 进一步加速清除(形成正反馈)
- **定量证据**：IFX阳性ADA患者的清除率从0.288 L/day升至0.768 L/day(~2.7倍)
- **围术期特殊性**：手术应激与免疫激活加速ADA形成，围术期ADA阳转率高于非手术患者

#### 2.2.3 蛋白丢失性肠病(PLE)与低白蛋白血症
- **粘膜破损的直接后果**：肠道炎症与手术创伤导致全身蛋白(含IgG类生物制剂)从肠道直接丧失
- **FcRn受体饱和**：血清白蛋白<3.5 g/dL时，FcRn受体饱和度上升 → IgG类生物制剂(IFX、ADA)失去回收保护 → 清除加速
- **临床数据**：白蛋白<2.5 g/dL患者的IFX半衰期仅为4-5天(vs 正常10-14天)

### 2.3 膳食干预与生物制剂协同效应的证据

#### 2.3.1 膳食干预的四大协同机制

**机制1：粘膜屏障快速修复**
- **EEN/CDED在6-8周内的效果**：粪便钙卫蛋白(FCal)下降50-76%
- **机制**：排除性饮食减少微生物群致病体，促进上皮细胞紧密连接恢复
- **临床转化**：FCal下降 → 肠道上皮通透性↓ → PLE减少 → 血清白蛋白自然恢复

**机制2：血清白蛋白快速恢复**
- **集中蛋白补充效果**：免疫营养(ω-3 + VitD + 高蛋白肠内营养)可将严重低白蛋白血症从1.3 g/dL快速升至2.5 g/dL(2-3周内)
- **FcRn恢复**：白蛋白正常化 → FcRn受体饱和缓解 → IgG类生物制剂回收效率恢复 → 半衰期延长

**机制3：炎症消耗减少**
- **排除性饮食的靶点降低**：CDED/EEN快速降低TNF-α、IL-23等靶点分子负荷(1-2周内)
- **TMDD的逆转**：靶点浓度↓ → IFX消耗减少 → 谷值维持更容易

**机制4：抗药物抗体(ADAs)抑制**
- **谷值维持策略**：通过及时营养-生物制剂协同优化维持IFX谷值>6.9 μg/mL
- **证据**：谷值>6.9 μg/mL患者的ADA阳转率仅为8-12%，而谷值<3.5 μg/mL患者的ADA阳转率达45-65%

#### 2.3.2 临床试验证据汇总

**IFX + 部分肠内营养(PEN, Sazuka等2019)**：
- 第14周临床反应率：87.01% (联合PEN) vs 74.75% (单IFX) 
- 第54周内镜缓解率：84.42% (联合) vs 65.66% (单药)，**OR=3.33** (95% CI 1.52-7.29)

**ADA + 膳食干预(多中心分析)**：
- 第26周谷值：7.6 ± 3.6 μg/mL (联合) vs 6.5 ± 3.9 (单药) 
- **关键发现**：第26周谷值高水平(≥8 μg/mL)的患者，第52周缓解率达**100%** vs 谷值低(<5 μg/mL)的患者仅39%

**维多珠单抗(VDZ) + 完全肠内营养(EEN)**：
- 第16周临床缓解：81.6% (EEN联合) vs 30.0% (VDZ单药)，**RR=2.72**
- 第16周粘膜愈合(Mayo 0或SES-CD 0)：85.7% (EEN联合) vs 26.9% (单药)，**RR=3.18**
- 获益持续至52周(缓解维持率85% vs 42%)

**乌司司单抗(UST) + EEN vs PEN(Liao等2024)**：
- EEN组在8周诱导期获得显著更高的临床缓解率(78% vs PEN 54%)与内镜缓解率(71% vs PEN 38%)

## 3. 研究目标

### 3.1 主要研究目标

#### **I期目标(术后1个月)：验证营养-生物制剂协同机制**

**主要假设**：围术期免疫营养或C-EEN干预能通过以下机制显著降低术后并发症：
- 快速恢复血清白蛋白(≥3.5 g/dL)
- 提升IFX谷值(≥6.9 μg/mL)
- 降低粘膜炎症负荷(FCal < 200 μg/g)

**主要研究目标**：
- **主要终点**：比较三组间术后30天Clavien-Dindo ≥II级并发症率
- **预期结果**：免疫营养组与C-EEN组的并发症率分别降至20-22%，相比对照组(38%)的**相对风险降低42-46%**

#### **II期目标(术后12个月)：验证长期缓解维持与生物制剂优化**

**主要假设**：通过TDM引导的生物制剂剂量优化，结合围术期营养干预的持续获益，显著提升长期缓解维持率

**主要研究目标**：
- **主要终点**：比较三组间12个月临床缓解维持率与内镜缓解维持率
- **预期结果**：免疫营养+TDM组与C-EEN+TDM组的12月缓解维持率达**75-85%**，相比对照组(55%)的**相对风险提升36-55%**

### 3.2 次要研究目标

- 验证白蛋白-IFX-FCal的相关性路径(通过中介分析)
- 评估不同营养干预方案对IFX清除率与ADAs形成的影响
- 比较TDM引导策略(反应性 vs 主动性)的效能
- 评估营养干预对患者生活质量、工作能力恢复的影响
- 进行成本-效益分析(干预成本 vs 并发症节省)

## 4. 研究设计

### 4.1 设计架构：多臂多阶段贝叶斯自适应设计

```
时间轴与阶段划分

I期(术后30天)：验证短期机制
├─ M0-M3：伦理审批与试点招募 (n=15-20 pilot)
├─ M4-M9：主体招募与随访 (累计n=40-50例)
├─ M9：**第一次中期分析**
│  ├─ Bayesian posterior probability计算
│  ├─ 决策规则：P(臂优越性)>0.85 → 扩大分配; <0.50 → 考虑停止
│  └─ 适应性调整：样本量重新估算、臂分配比例调整
└─ M9-M18：继续招募至全部样本，完成I期随访锁定

II期(术后12个月)：验证长期缓解维持与生物制剂协同
├─ M1-M18：所有患者完成12月随访
├─ M18：**I期数据锁定** → 第一批论文(并发症、机制论文)
├─ M18：**第二次中期分析**(可选，若样本充足)
└─ M24：II期数据锁定 → 第二批论文(长期缓解、TDM论文)
```

### 4.2 试验架构：三臂共享对照设计

```
┌─────────────────────────────────────────────────────────────┐
│          对照组(n=40, 固定)                                  │
│  标准营养支持 + 标准TDM(每3个月)                            │
│  └─ 按术式分层：开放手术/腹腔镜 1:1随机                    │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────┐  ┌──────────────────────┐
│  免疫营养臂              │  │  C-EEN循环臂         │
│  n=35-45(自适应)        │  │  n=35-45(自适应)     │
├─────────────────────────┤  ├──────────────────────┤
│ ω-3脂肪酸10g/日         │  │ 2周EEN循环           │
│ VitD 5000IU/日          │  │ + 6周正常饮食        │
│ 高蛋白肠内营养          │  │ (持续12个月)         │
│                         │  │                      │
│ TDM-引导:               │  │ TDM-引导-增强:       │
│ • M0-M1: 双周检测       │  │ • EEN周: 周检测      │
│ • M1-M12: 月检测        │  │ • 恢复周: 月检测     │
│ • 阈值: 若谷值<3        │  │ • 阈值: 若谷值<4     │
│   μg/mL → 剂量增加      │  │   μg/mL → 立即调整   │
└─────────────────────────┘  └──────────────────────┘
```

### 4.3 贝叶斯适应性框架

#### **先验设定(Prior Specification)**

基于已发表的膳食-生物制剂协同研究(Sazuka 2019, Liao 2024)：

| 参数 | 对照组 | 免疫营养组 | C-EEN组 |
|------|--------|---------|--------|
| **术后30天并发症率** | Beta(15, 25) → 平均37.5% | Beta(8, 32) → 平均20% | Beta(7, 30) → 平均19% |
| **12月缓解维持率** | Beta(22, 18) → 平均55% | Beta(30, 13) → 平均70% | Beta(32, 12) → 平均73% |

#### **后验更新与决策规则**

**M9中期分析(n=40-50)**：
```
后验概率计算：
P(免疫营养优于对照) = ∫P(并发症率_IM < 对照) × f(后验数据) dθ

决策规则：
- 若 P(臂优越性) > 0.85 → 扩大臂分配比例至 2:1 (对照:干预)
- 若 P(臂优越性) ∈ [0.50, 0.85] → 维持等比例分配
- 若 P(臂优越性) < 0.50 → 考虑停止该臂招募
```

**M18阶段锁定**：
```
I期最终分析采用ITT原则：
- 主要结局：logistic回归调整年龄、BMI、手术类型
- 95% 可信区间(Bayesian CrI)替代传统95% CI
- 效应量转化为临床相关性概率P(effect > 最小临床意义差)
```

## 5. 研究人群

### 5.1 纳入标准

1. **年龄与诊断**：
   - 年龄18-65岁
   - 确诊IBD(CD或UC)，病程≥6个月
   - 需行择期腹部手术(符合术前营养不良标准)

2. **营养与生物制剂使用状态**：
   - **营养不良指标至少符合以下之一**：
     - NRS-2002评分≥3分
     - 血清白蛋白<3.5 g/dL
     - 体重减少≥5% (过去3个月内)
   - **必须使用抗TNF制剂**：
     - IFX(英夫利昔单抗)：维持谷值已建立
     - ADA(阿达木单抗)：已进入维持治疗阶段(≥2次给药)
   - **生物制剂最后一次给药与手术的间隔≤6周** (确保围术期内有检测点)

3. **手术类型** (所有符合以下任意一项的IBD腹部手术)：
   - 回肠造口术、全结肠切除、直肠保留手术(J-pouch)
   - 部分结肠切除、小肠切除(短肠综合征除外)
   - 肛周脓肿引流、肠瘘修补
   - 肠粘连分解
   - **排除标准**：紧急/急诊手术、ECMO支持

### 5.2 排除标准

1. **医学禁忌**：
   - 肾功能不全(eGFR<30 mL/min/1.73m²)
   - 严重肝功能异常(Child-Pugh C级)
   - 活动性感染或败血症
   - 近3个月内恶性肿瘤诊断或治疗

2. **营养干预禁忌**：
   - 短肠综合征(残留小肠<100 cm)
   - 完全肠梗阻
   - 活动性肠外瘘
   - 无法进行肠内营养的生理条件

3. **研究相关**：
   - 无法进行TDM血液检测(如检测前7天使用影响IFX浓度的其他单克隆抗体)
   - 妊娠或哺乳期
   - 无法提供知情同意或依从性差

## 6. 干预措施

### 6.1 对照组 (n=40, 固定臂)

#### **营养支持策略**
- **目标**：满足每日蛋白40-50 g、热量1500-2000 kcal
- **方法**：标准肠内营养制剂(不含免疫营养成分) + 膳食指导
- **监测**：白蛋白、体重、NRS评分(M0, M1, M3, M12)

#### **生物制剂管理(标准TDM)**
- **监测频率**：每3个月检测一次IFX/ADA谷值
- **目标谷值**：IFX 3-8 μg/mL, ADA 5-12 μg/mL (根据ECCO共识)
- **调整策略**(反应性)：
  - 若临床失效(CRP↑、FCal>250) → 给药间隔↓ (如从8周改6周)
  - 若 ADA阳性 → 考虑更换至另一生物制剂类别
- **IFX检测方法**：HPLC或ELISA(中国符合标准的检测机构)

### 6.2 免疫营养 + TDM引导臂 (n=35-45, 自适应)

#### **营养干预详细方案**

**基础营养成分**：
- **ω-3多不饱和脂肪酸**：10 g/日(EPA+DHA)
  - 来源：医用鱼油胶囊或肠内营养乳剂
  - 机制：减轻炎症、促进抗炎细胞因子(IL-10)生成
  - 证据：CD患者使用ω-3后TNF-α下降30-40%
- **维生素D**：5000 IU/日
  - 理由：IBD患者VitD缺乏率>60%，与肠屏障功能↓、复发率↑相关
  - 目标：血清25(OH)VitD ≥30 ng/mL
- **高蛋白肠内营养**：
  - 蛋白含量1.5 g/kg/日(目标60-80 g/日)
  - 亮氨酸、精氨酸强化配方(促进伤口愈合与免疫功能)
  - 具有易消化的MCT脂肪链

**应用时间**：
- 术前2-4周开始，术后持续≥12个月(或至生物制剂响应稳定)

**个体化调整**：
- 若患者无法耐受肠内营养 → 改用口服免疫营养粉(混合正常饮食)
- 若出现腹泻 → 添加益生元(低聚果糖)或调整脂肪比例

#### **TDM引导策略(主动优化)**

**监测频率与目标**：
- **M0-M1(术后关键期)**: 双周检测IFX/ADA谷值
  - 目标：IFX ≥6.9 μg/mL(第14周预测谷值，基于PANTS研究)
  - 原理：此期是决定早期应答的关键窗口
- **M1-M12(维持期)**：月检测
  - 目标：维持IFX 6-8 μg/mL或ADA 16-20 μg/mL(深度愈合阈值)

**主动调整决策树**：
```
若IFX谷值<3 μg/mL(术后1-2周检测)
├─ 原因诊断：
│  ├─ 白蛋白<3.5 g/dL? → 强化蛋白补充、增加热量
│  ├─ FCal>250 μg/g? → 炎症未控制，考虑增加IFX给药频次
│  └─ ADA阳性? → 添加硫唑嘌呤或咪唑硫嘌呤(降低ADA)
└─ 干预：给药间隔从8周改为6周或5周

若白蛋白无法自然恢复(M1白蛋白仍<3.0 g/dL)
├─ 添加肠外营养支持(部分静脉营养，PPN)
│  └─ 蛋白500 mL/日(10% 氨基酸液)，持续1-2周
└─ 重新评估是否存在未诊断的PLE(24h粪便蛋白)

若白蛋白恢复正常(≥3.5 g/dL) + IFX谷值≥6.9 μg/mL
└─ 继续维持，月检测一次，若连续2次达标可改为3月检测
```

**血液监测指标(伴随检查)**：
- C反应蛋白(CRP)、降钙素原(PCT)
- 全血细胞计数(CBC)、肝功能(ALT/AST)、肾功能(Cr)
- 粪便钙卫蛋白(FCal)

### 6.3 周期性EEN + TDM引导-增强臂 (n=35-45, 自适应)

#### **C-EEN方案(循环式完全肠内营养)**

**原理**：基于CD-HOPE试验(Zhang等2021)与RETOS试验框架，采用"间断-恢复"策略降低长期EEN的依从性负担

**8周为一个循环周期(持续12个月，共3个完整循环 + 部分4th周期)**：

| 阶段 | 时间 | 饮食形式 | 具体操作 |
|------|------|--------|--------|
| **EEN诱导期** | 第1-2周 | 100% 肠内营养 | 1800-2000 mL/日标准EEN(无脂肪、无纤维、易消化) |
| **食物再引入** | 第3-8周 | 渐进式混合 | 周3→30%EEN+70%食物; 周4→20%EEN+80%食物; 周5-8→正常饮食+EEN停止 |
| **维持与监测** | 周期间隔 | 正常IBD饮食 | 实施CDED(排除性饮食)指导，避免高脂肪、乳糖、添加剂 |

**EEN制剂选择**：
- 配方特征：蛋白40-50 g/L、脂肪<3 g/L(MCT油)、碳水化合物60-70 g/L
- 推荐产品：益力健(Modulen IBD)或类似配方
- 给药方式：鼻饲或口服(患者可选)，持续或间歇给药

**关键支持手段**：
- **患者教育**：强调"间断性EEN是自我修复，不是永久限制"
- **心理支持**：引入营养师月度咨询，解决依从性问题
- **食物替代方案**：提供可耐受的食物清单(低脂鸡肉、白米、香蕉等)

#### **TDM引导-增强策略**

**强化监测(相比免疫营养臂)**：

| 阶段 | 监测频率 | 关键指标 | 决策阈值 |
|------|----------|--------|---------|
| **EEN期(1-2周)** | **周检测** | IFX谷值 | <4 μg/mL → 立即增加IFX给药量或缩短间隔 |
| **食物再引入期(3-8周)** | 双周检测 | IFX谷值 + FCal | 同时监测粘膜愈合(FCal<100μg/g目标) |
| **维持期(第9-8周内)** | 月检测 | IFX谷值 + FCal | 若FCal升高>150 μg/g → 提前启动下一周期EEN |

**适应性决策树(特殊于C-EEN组)**：

```
周期1(M1-M2.5)结束时评估
├─ 若IFX谷值充分(≥6.9) + FCal<100 + 临床缓解 
│  └─ 继续周期2-3，维持"2周EEN + 6周正常"
├─ 若IFX仍低(<3) 尽管EEN + 白蛋白充分
│  ├─ 原因：可能基因型TPMT异常或隐性ADA
│  └─ 干预：考虑增加IFX剂量(如从5mg/kg改7.5 mg/kg) + 添加免疫抑制剂
└─ 若FCal高(>250) 尽管EEN
   └─ 建议：延长EEN周期至3周，或改用更严格的CDED

若患者无法耐受2周NPO(严重心理负担或社会因素)
└─ 修改方案：4周EEN + 4周正常(保持50% EEN来自肠内营养) → 称为"部分EEN"
```

**C-EEN的预期效果(证据基础)**：
- 第8周粘膜钙卫蛋白下降：**70%患者FCal↓>50%**(vs PEN仅35%)
- 粘膜愈合率(Mayo 0或SES-CD 0)：**71%**(vs PEN 38%)
- **长期缓解维持率**：EEN后12月缓解率79% vs PEN 52%

## 8. 样本量计算

### 8.1 I期样本量(主要终点：术后30天并发症率)

**参数设定**：
- **对照组基线并发症率**：38%(基于过往单中心队列数据)
- **干预臂目标并发症率**：22%(相对风险降低42%)
- **统计功效**：80% (β=0.20)
- **双侧显著性水平**：α=0.05
- **设计**：两样本比例比较(卡方检验)

**传统固定设计**：
```
n = 2 × [(Z_α + Z_β)² × p(1-p)] / (p₁ - p₂)²
  = 2 × [(1.96 + 0.84)² × 0.30 × 0.70] / (0.38 - 0.22)²
  = 2 × [7.84 × 0.21] / 0.0256
  ≈ 64 per arm
总样本量(三臂) = 64 × 3 = 192例
```

**贝叶斯自适应设计的优化**：
- **共享对照**：对照臂n=40(固定) + 两干预臂共n=90-100(自适应)
- **早期淘汰**：若M9中期分析显示某臂P(优越性)<0.50，停止该臂招募，样本节省15-20例
- **优化分配**：M9后根据posterior probability调整分配比例(如从1:1:1改为1:1.5:1.5)
- **样本量重估**：根据M9的效应量重新计算所需样本完成II期，通常↓15-25%

**最终样本量**：
- **总体**：n=130-140例(相比传统192例的32-35%降低)
- **分配**：对照n=40(固定), 免疫营养n=35-45, C-EEN n=35-45
- **考虑脱落**：假设15%脱落率，目标招募n=155-165例

### 8.2 II期样本量(主要终点：12月缓解维持率)

**参数设定**(基于Sazuka等2019证据)：
- **对照组缓解维持率**：55%
- **干预臂缓解维持率目标**：75-82%(相对风险提升36-49%)
- **统计功效**：85%(β=0.15，考虑更严格的II期证据标准)
- **显著性水平**：α=0.05

**需要样本**：
```
按相同的I期队列(n=110-120完成)，有足够统计功效检测上述缓解率差异
(样本量≈85-110/臂，已涵盖在I期设计内)
```

**实际意义**：II期并不需要额外样本，而是对I期完成者的长期随访扩展

## 11. 统计分析方法

### 11.1 分析原则

**主体分析**：基于意向治疗(Intention-to-Treat, ITT)原则
- 所有随机化患者纳入，按原分组分析(不管依从性)
- 理由：最真实的临床效果反映，降低选择偏倚

**敏感性分析**：
- **按协议(Per-Protocol, PP)分析**：至少完成I期干预的患者(排除早期脱落)
- **完全病例(Complete Case)分析**：无缺失数据的患者

### 11.2 I期并发症率分析

#### **主要模型：逻辑回归**

```
logit(P(Clavien-Dindo ≥II)) = β₀ + β₁×臂分配 + β₂×中心 + β₃×年龄 
                                + β₄×BMI + β₅×手术类型 + β₆×基线白蛋白

结果呈现：
- 对照组：基准(参考)
- 免疫营养 vs 对照：调整后OR与95% Bayesian CrI
- C-EEN vs 对照：调整后OR与95% Bayesian CrI

优越性判定：
- 若CrI下限<1.0 → 支持非劣效
- 若CrI完全<1.0 → 支持优越性
- P(OR<1.0) > 0.95 → 强证据支持优越性
```

#### **多重比较矫正**

传统方法的缺点：
- 两个干预臂 vs 对照 = 两次比较 → Bonferroni矫正(α=0.025)过于保守

贝叶斯方法的优势：
- 自动在后验概率计算中校正多重比较
- 无需额外矫正，保持α=0.05

### 11.3 中期分析与适应性调整

#### **M9中期分析(n=40-50患者完成M1随访)**

**步骤1：后验概率计算**

```
利用Beta-Binomial共轭模型：

对照组 Clavien≥II 事件: x_c ~ Binomial(n_c, p_c)
后验: p_c | data ~ Beta(α_c + x_c, β_c + n_c - x_c)
其中(α_c, β_c) = (15, 25) [先验，基于历史数据]

类似地计算免疫营养组与C-EEN组的后验

后验概率：
P(p_IM < p_control | data) = ∫∫ I(p_IM < p_control) 
                               × f(p_IM | data) × f(p_control | data) dp_IM dp_control
                             [数值积分或MCMC]
```

**步骤2：决策规则**

```
决策表：

┌──────────────────────┬─────────────────┬─────────────────────┐
│ P(臂优越性)          │ 免疫营养臂      │ C-EEN臂              │
├──────────────────────┼─────────────────┼─────────────────────┤
│ >0.85 (强证据)       │ 扩大招募(2:1)   │ 扩大招募(2:1)        │
│ 0.50-0.85 (中等)     │ 维持1:1:1       │ 维持1:1:1            │
│ <0.50 (弱证据)       │ 停止招募或缓招   │ 停止招募或缓招        │
└──────────────────────┴─────────────────┴─────────────────────┘

实例：
- 若P(免疫营养优越) = 0.88 + P(C-EEN优越) = 0.79
- → 两臂均>0.85阈值 → 分配比从1:1:1改为1:1.5:1.5(对照:免疫:EEN)
- → 继续招募，目标总n≈130-140(从原120-130增加10例)
```

**步骤3：样本量重估**

```
基于M9的效应量(effect size)，更新所需样本完成II期
- 若M9显示效应量>预期 → 总样本可减少至110-120
- 若效应量<预期 → 可增加至140-150

重估后的样本量结合脱落率，更新最终招募目标
```

#### **M18 I期数据锁定**

```
包括：
- 所有患者的M30(术后30天)数据
- 所有患者的M0-M1医学检查数据
- 安全监测报告

结果分析：
- 主要分析：I期并发症率、白蛋白改善、IFX谷值提升(如上述逻辑回归)
- 次要分析：机制验证(白蛋白-IFX-FCal的中介分析)
- 亚组分析：按手术类型、生物制剂类型、基线营养状态

输出：
- 第一篇论文：并发症与机制(J Crohns Colitis或Gastroenterology level)
- 第二篇论文：TDM引导策略的成本-效益(Journal of Parenteral and Enteral Nutrition)
```

### 11.4 II期缓解维持率分析

#### **主要模型：Cox比例风险回归**

```
观察变量：患者在12个月内首次丧失缓解的时间

Cox模型：
h(t|臂, 协变量) = h₀(t) × exp(β₁×臂 + β₂×年龄 + β₃×基线CRP + ...)

结果呈现：
- 缓解维持曲线(Kaplan-Meier, K-M)
- 调整后风险比(aHR) 及95% Bayesian CrI
- P(缓解维持率₁₂ₘ > 对照) [后验概率]

优越性判定：
- K-M曲线分离(在M6-M12明显分组间差异)
- aHR 95% CrI 下限 < 1.0 → 非劣效支持
- P(aHR<1.0) > 0.95 → 强证据支持
```

#### **患者报告结局(PRO)分析**

```
测量工具：
- IBDQ评分(问卷)：正常≥170，每次增加分数表示改善
- 工作能力恢复：以"返回全职工作"为标准，用逻辑回归计算风险比

模型：
logit(P(工作恢复)) = β₀ + β₁×臂 + β₂×基线IBDQ + ...

结果：
- 表3：各臂M3、M6、M12的IBDQ均值(SD)
- 表4：12月工作恢复率及调整后RR
```

### 11.5 群体药代动力学(PK)分析

**目标**：评估IFX清除率(CL)与营养干预的关系

```
模型：CL = CLpop × e^(η_CL)
其中：
CLpop = θ_CL × (白蛋白 / 参考值)^θ_白蛋白 × (FCal / 参考值)^θ_FCal × ...

参数：
- θ_CL：群体基础清除率
- θ_白蛋白：白蛋白对清除率的影响强度
- θ_FCal：粪便钙卫蛋白的影响

拟合软件：NONMEM 或 Monolix

结果：
- 评估营养干预(白蛋白恢复)对IFX CL的独立效应
- 量化每增加1 g/dL白蛋白时，IFX CL的相对降低幅度
- 支持"蛋白丢失→FcRn饱和→IFX清除加速"的病理生理假设
```

### 11.6 中介分析(Mediation Analysis)

**目的**：验证"白蛋白恢复"是否为"营养干预→IFX谷值提升"的主要机制

```
路径模型：
营养干预 ──直接效应→ IFX谷值
       ──→ 白蛋白↑ ──(β)→ IFX谷值 (间接/中介效应)

Baron-Kenny 方法：
1. 总效应(Total Effect) = c
2. 营养→白蛋白 = a (路径1)
3. 白蛋白→IFX谷值 = b (路径2)
4. 直接效应(Direct Effect) = c'
5. 间接效应(Indirect Effect) = a × b

比例计算：
% mediated = (a × b) / c × 100%

结果解释：
- 若%mediated > 50% → 白蛋白是主要机制
- 若%mediated 30-50% → 白蛋白是部分机制，还有其他路径
- 若%mediated < 30% → 白蛋白不是主要驱动力，需探索其他因素

学位论文质量：应发表在营养-药代学专业期刊
```

### 11.7 缺失数据处理

**方法**：多重插补(Multiple Imputation, MI)

```
假设：MCAR(Missing Completely at Random) 或 MAR(Missing at Random)
       - 若因脱组导致缺失(MNAR) → 进行敏感性分析

步骤：
1. 使用 MICE (Multivariate Imputation by Chained Equations) 算法
2. 生成 m=10 个完整数据集
3. 对每个数据集分别进行分析
4. 使用Rubin规则合并结果(获得overall β与SE)

输出：
- 主体结果 = 10个分析结果的平均
- 标准误 = Rubin合并误差
- 与Complete Case分析的对比(作为敏感性分析)
```

## 12. 数据管理与质量保证

### 12.1 数据采集与管理系统

- **模块3**：营养干预记录(每周输入，追踪依从性)

- **模块5**：临床结局(并发症、缓解状态、IBDQ评分)
- **模块6**：安全监测(不良事件、实验室异常)

**数据录入**：
- 时点：每周一次(营养与TDM)，或每月汇总(临床评估)
- 权限：按岗位设定(营养师可编辑营养模块，医生可编辑临床模块)
- 验证：自动范围检查(如年龄18-65、白蛋白0-10 g/dL)

### 12.2 质量保证

**审核流程**(周、月、季度)：

| 频率 | 审核项目 | 执行人 | 标准 |
|------|---------|--------|------|
| **周** | 数据完整性、逻辑错误 | 研究协调员 | 缺失率<5% |
| **月** | 20%随机抽样验证(对比源文件) | 临床监督员 | 数据准确率>99% |
| **季** | 安全信号检测、不良事件趋势 | 数据安全委员会 | SAE上报率100% |

**源文件验证**：
- 对象：病历、实验室报告单、营养记录表
- 范围：20%随机样本(26-28例患者)
- 核查项：人口学数据、关键实验室数据、并发症记录
- 不符合率>2% → 启动全体数据审计

### 12.3 数据安全与隐私

- **加密**：所有个人身份信息(PII)离线存储，通过患者ID号关联eCRF

## 14. 数据安全监查

### 14.1 数据安全委员会(DSC)结构

| 职位 | 背景 | 职责 |
|------|------|------|
| **主席** | 独立临床医学专家(非项目参与者) | 监督整体安全性与试验完整性 |
| **临床医学委员** | IBD领域专家 | 评估临床安全信号 |
| **生物统计委员** | 独立统计学家 | 分析安全性数据、进行中期分析 |
| **患者代表** (可选) | IBD患者或患者组织代表 | 代表患者权益与体验反馈 |

**与主研究团队的独立性**：DSC成员不参与临床护理或数据分析，保证独立评估

### 14.2 安全性监测指标

**常规监测**(每次随访)：
- 严重不良事件(SAE)发生
- 相关不良事件(AE)：胃肠道症状恶化、导管相关感染、过敏反应
- 实验室异常：肝功能(ALT>3×ULN)、肾功能(Cr>2×基线)、血细胞减少(WBC<2500)

**主要安全终点**(累积监测，每季度DSC审查)：
- SAE发生率与关联性
- 提前停止试验的规则：
  - 若某臂SAE发生率>20% → 提议停止该臂
  - 若总体死亡数>3例 → 启动特别调查
  - 若因安全问题脱组>30% → 评估试验继续的必要性

### 14.3 不良事件报告程序

**分级**：
- **轻度(Grade 1)**：症状(如腹泻)，不影响日常活动，无需干预
- **中度(Grade 2)**：症状，轻度影响日常活动，需简单干预(如止泻药)
- **重度(Grade 3)**：症状，显著影响日常活动，需医学干预
- **危重(Grade 4)**：生命威胁
- **致命(Grade 5)**：死亡

**报告时限**：
- Grade 3-5(SAE)：**24小时内** 通知DSC、伦理委员会、监管部门
- Grade 1-2(AE)：累积至每月汇总报告

**报告内容**：
- 患者ID、事件日期、事件描述、严重程度
- 与研究药物/干预的关联评估(肯定、可能、无关)
- 患者预后、采取的措施

### 17.1 主要风险与应对

| **伦理与监管变化** | 低 | 中 | 与伦理委员会保持沟通、提前了解政策动向 |

### 17.2 应急启动清单

**若I期中期分析显示某臂无效(P<0.50)**：
- 决策点：是否停止该臂？
  - 若对照臂对比显示统计可靠的差异(P>0.50) → 停止无效臂，加速有效臂招募
  - 若无明确差异但趋势一致 → 保持招募，延迟停止规则至M12

**若脱组率>20%**：
- 启动患者保留计划(retention strategy)：
  - 增加与患者的电话联系(月2次改为周1次)
  - 简化随访程序(部分可远程完成)
  - 增加患者激励(每次随访补助从无改为50元)

**若发生预期外的SAE集群**：
- DSC召开紧急会议(48小时内)
  - 评估事件与干预的因果关系
  - 决策：继续、修改或停止试验
  - 向伦理委员会与监管部门报告
